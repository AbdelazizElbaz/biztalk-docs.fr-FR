 <span data-ttu-id="ed352-108">Le code suivant utilise un client WCF pour appeler le **Execute** méthode du service métier TimeStamp.</span><span class="sxs-lookup"><span data-stu-id="ed352-108">The following code uses a WCF client to invoke the **Execute** method of the TimeStamp business service.</span></span> <span data-ttu-id="ed352-109">L’horodatage, qui est retourné dans l’heure locale du serveur Siebel, est ensuite écrites dans la console.</span><span class="sxs-lookup"><span data-stu-id="ed352-109">The time stamp, which is returned in the local time of the Siebel server, is then written to the console.</span></span> <span data-ttu-id="ed352-110">Le client WCF dans cet exemple est initialisé à partir d’un fichier de configuration généré par le [!INCLUDE[addadapterservrefshort](../../includes/addadapterservrefshort-md.md)].</span><span class="sxs-lookup"><span data-stu-id="ed352-110">The WCF client in this example is initialized from a configuration file generated by the [!INCLUDE[addadapterservrefshort](../../includes/addadapterservrefshort-md.md)].</span></span> <span data-ttu-id="ed352-111">Pour obtenir un exemple de fichier de configuration généré, consultez [configurer un Client WCF pour un système Siebel](../../adapters-and-accelerators/adapter-siebel/configure-a-wcf-client-for-a-siebel-system.md).</span><span class="sxs-lookup"><span data-stu-id="ed352-111">For an example of a generated configuration file, see [Configure a WCF Client for a Siebel System](../../adapters-and-accelerators/adapter-siebel/configure-a-wcf-client-for-a-siebel-system.md).</span></span>  
  
```  
using System;  
using System.Collections.Generic;  
using System.Text;  
  
using System.ServiceModel;  
using microsoft.lobservices.siebel._2007._03.BusinessServices.TimeStamp;  
  
namespace Business_Service  
{  
    class Program  
    {  
        static void Main(string[] args)  
        {  
            BusinessServices_TimeStamp_OperationClient client = null;  
  
            try  
            {  
                client =  
                     new BusinessServices_TimeStamp_OperationClient("SiebelBinding_BusinessServices_TimeStamp_Operation");  
  
                client.ClientCredentials.UserName.UserName = "YourUserName";  
                client.ClientCredentials.UserName.Password = "YourPassword";  
                client.Open();  
  
                ExecuteResponseRecord er = client.Execute();  
                Console.WriteLine(er.Time);  
  
                Console.WriteLine("\nHit <RETURN> to finish");  
                Console.ReadLine();  
            }  
            catch (Exception e)  
            {  
                Console.WriteLine(e.Message);  
            }  
            finally  
            {  
                // Close the client.  
                if (client != null)  
                {  
                    if (client.State == CommunicationState.Opened)  
                        client.Close();  
                    else  
                        client.Abort();  
                }  
            }  
        }  
    }  
}  
```
