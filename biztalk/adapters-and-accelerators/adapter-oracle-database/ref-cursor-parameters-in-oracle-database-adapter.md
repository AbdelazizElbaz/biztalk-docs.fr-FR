<span data-ttu-id="1e8a4-103">Un REF CURSOR est un type de données PL/SQL qui représente un pointeur vers un jeu de résultats côté serveur généré en exécutant une requête.</span><span class="sxs-lookup"><span data-stu-id="1e8a4-103">A REF CURSOR is a PL/SQL data type that represents a pointer to a server-side result set generated by executing a query.</span></span> <span data-ttu-id="1e8a4-104">Un type REF CURSOR permet d’entrée et de sortie de diffusion en continu de données et est idéal pour le transfert de grandes quantités de données vers et à partir d’un code de PL/SQL.</span><span class="sxs-lookup"><span data-stu-id="1e8a4-104">A REF CURSOR type enables input and output streaming of data and is ideal for transferring large amounts of data to and from a PL/SQL code.</span></span> <span data-ttu-id="1e8a4-105">Le [!INCLUDE[adapteroracle_short](../../includes/adapteroracle-short-md.md)] fortement typée et faiblement typé (SYS_REFCURSOR) REF CURSOR qui peuvent être passés aux procédures PL/SQL et aux fonctions, OUT ou IN des paramètres est prise en charge.</span><span class="sxs-lookup"><span data-stu-id="1e8a4-105">The [!INCLUDE[adapteroracle_short](../../includes/adapteroracle-short-md.md)] provides support for strongly-typed and weakly-typed (SYS_REFCURSOR) REF CURSORs that can be passed to PL/SQL procedures and functions as IN, OUT, or IN OUT parameters.</span></span>
