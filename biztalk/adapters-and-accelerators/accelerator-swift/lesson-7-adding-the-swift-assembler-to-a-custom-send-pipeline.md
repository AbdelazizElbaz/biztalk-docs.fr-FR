# <a name="lesson-7-adding-the-swift-assembler-to-a-custom-send-pipeline"></a>Leçon 7 : Ajout de l’assembleur SWIFT à un Pipeline d’envoi personnalisé
Dans cette leçon, vous ajoutez l’assembleur personnalisé SWIFT (ASM) à votre pipeline.  
  
### <a name="to-add-the-swift-assembler-pipeline-component-to-the-pipeline"></a>Pour ajouter le composant de pipeline assembleur SWIFT au pipeline  
  
1.  Dans le **boîte à outils des composants de Pipeline BizTalk**, faites glisser le **SWIFT assembleur** à la **Déposer ici** zone ci-dessous le **assemblage** forme  **Le Concepteur de Pipeline BizTalk**.  
  
2.  Sur le **fichier** menu, sélectionnez **Enregistrer tout** pour enregistrer vos modifications.  
  
 Passez à [leçon 9 : création et déploiement de l’Assembly](../../adapters-and-accelerators/accelerator-swift/lesson-8-building-and-deploying-the-assembly.md).
