# <a name="module-4-adding-an-xml-receive-location-and-flat-file-send-port"></a>Module 4 : Ajout d’un document XML emplacement de réception et Port d’envoi de fichier plat
Dans ce module, vous ajoutez un port et l’emplacement de réception et un port d’envoi. Vous utilisez le port de réception et l’emplacement pour recevoir des messages de MT103 d’au format XML à partir d’un dossier de fichiers entrant. Vous utilisez le port d’envoi pour envoyer des messages de fichier plat MT103 validés à un dossier sortant.
