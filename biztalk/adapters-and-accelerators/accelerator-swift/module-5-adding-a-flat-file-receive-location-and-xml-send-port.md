# <a name="module-5-adding-a-flat-file-receive-location-and-xml-send-port"></a>Module 5 : Ajout d’un fichier plat emplacement de réception et le Port d’envoi XML
Dans ce module, vous ajoutez des fichiers plats un port de réception et emplacement, et XML d’un port d’envoi. Vous utilisez le port de réception et l’emplacement pour recevoir un fichier plat MT103 à partir d’un dossier de fichiers. Vous utilisez le port d’envoi pour envoyer le message de MT103 à un dossier sortant au format XML.
