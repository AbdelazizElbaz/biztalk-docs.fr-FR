# <a name="prequisities-for-the-wcf-lob-adapter-sdk-tutorials"></a>Prequisities pour les didacticiels de WCF LOB Adapter SDK
Cette section fournit des détails sur les concepts que vous devez être familiarisé avec à tirer le meilleur parti les didacticiels, ainsi que les logiciels requis à effectuer pour ceux-ci.  
  
## <a name="what-to-install"></a>Que se passe-t-il pour l’installation  
 Pour commencer les didacticiels, vous devez disposer des logiciels suivants sur votre ordinateur.  
  
-   [!INCLUDE[btsVStudioNoVersion_md](../../includes/btsvstudionoversion-md.md)]
  
-   Internet Information Services (IIS)  
  
-   [!INCLUDE[btsDotNetFramework_md](../../includes/btsdotnetframework-md.md)]
  
-   [!INCLUDE[afproductnameshort](../../includes/afproductnameshort-md.md)](disponible avec vos fichiers d’installation de BizTalk Server, nommés ASDK_x64 et ASDK_x86)  
  
 Pour terminer le didacticiel 3, vous devez disposer des logiciels suivants sur votre ordinateur.  
  
-   Microsoft SharePoint  
  
-   Kit de développement logiciel de Microsoft SharePoint  
  
> [!CAUTION]
>  N’effectuez pas ces didacticiels dans votre environnement de production.  
  
L’adaptateur Echo terminé est inclus dans les fichiers d’installation de BizTalk à `\BizTalk Server\ASDK_x86\Program Files\WCF LOB Adapter SDK\Documents\Samples` ou `\BizTalk Server\ASDK_x64\Program Files\WCF LOB Adapter SDK\Documents\Samples`.
  
## <a name="prerequisites"></a>Conditions préalables  
 Les didacticiels supposent que vous êtes familiarisé avec les éléments suivants :  
  
-   Développement et conception de logiciels orientés objet  
  
-   Windows Communication Foundation (WCF), plus précisément, la programmation du modèle de canal et programmation de modèle de service  
  
-   XSD, XML, WSDL et la relation entre Web Services Description Language (WSDL) et une API  
  
-   Langage de programmation c#  
  
-   .NET Framework  
  
 Avant de commencer les didacticiels, il peut être utile si vous avez consulté les rubriques conceptuelles de ce Kit de développement et que vous avez une idée générale des canaux dans [!INCLUDE[nextref_btsWinCommFoundation](../../includes/nextref-btswincommfoundation-md.md)], y compris les types dans les **T:Microsoft.ServiceModel.Channels**et **T:Microsoft.ServiceModel.Channels.Common** espaces de noms.  Pour plus d’informations sur les canaux, consultez le [vue d’ensemble du modèle de canal](https://msdn.microsoft.com/library/ms729840.aspx).
