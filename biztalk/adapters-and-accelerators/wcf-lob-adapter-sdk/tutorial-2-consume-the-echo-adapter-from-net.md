# <a name="tutorial-2-consume-the-echo-adapter-from-net"></a>Didacticiel 2 : Utiliser l’adaptateur d’écho à partir de .NET
Cette section fournit les étapes pour consommer les opérations entrantes et sortantes exposées par l’adaptateur d’écho à partir de .NET. L’adaptateur Echo a été développé dans [didacticiel 1 : développer l’adaptateur Echo](../../adapters-and-accelerators/wcf-lob-adapter-sdk/tutorial-1-develop-the-echo-adapter.md).  
  
> [!NOTE]
>  Le code d’adaptateur d’écho et de test est inclus dans les fichiers d’installation de BizTalk à `\BizTalk Server\ASDK_x86\Program Files\WCF LOB Adapter SDK\Documents\Samples` ou `\BizTalk Server\ASDK_x64\Program Files\WCF LOB Adapter SDK\Documents\Samples`.
