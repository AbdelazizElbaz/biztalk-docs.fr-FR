# <a name="pipeline-samples"></a>Exemples de pipeline
Le [!INCLUDE[btsCoName](../../includes/btsconame-md.md)]® [!INCLUDE[BTARN_CurrentVersion_FirstRef](../../includes/btarn-currentversion-firstref-md.md)] Kit de développement logiciel (SDK) inclut un exemple de pipeline d’envoi et un exemple de pipeline de réception. Cette section fournit des informations sur les fonctionnalités que chaque exemple montre comment, instructions pour générer et exécuter les exemples et les résultats que vous pouvez attendre.  
  
 Pour créer un pipeline personnalisé, vous ajoutez des références au projet de pipeline pour le RNPIPs, RNIFv11 et RNIFv201 des assemblys de schéma qui se trouvent dans le [!INCLUDE[btsCoName](../../includes/btsconame-md.md)]. Espace de noms Solutions.BTARN.Schemas. Pour plus d’informations sur la création de pipelines personnalisés, consultez « Pipelines ([!INCLUDE[btsBizTalkServerNoVersion](../../includes/btsbiztalkservernoversion-md.md)] dossier d’exemples) » dans l’aide de BizTalk Server.
