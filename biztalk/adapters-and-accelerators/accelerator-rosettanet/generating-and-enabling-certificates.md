# <a name="generating-and-enabling-certificates"></a>Génération et l’activation de certificats
Ce didacticiel utilise des certificats pour signer et chiffrer les messages envoyés par les organisations de Contoso et Fabrikam. Si vous avez déjà effectué le didacticiel Double Action, vous pouvez ignorer cette étape et passez à [création de la Contoso Solution](../../adapters-and-accelerators/accelerator-rosettanet/creating-the-contoso-solution.md). Pour générer et utiliser des certificats pour ce didacticiel, les procédures décrites dans les rubriques suivantes :  
  
-   [Étape 1 : Création d’une autorité de Certification](../../adapters-and-accelerators/accelerator-rosettanet/step-1-creating-a-certification-authority.md)  
  
-   [Étape 2 : Création publiques et privées des certificats](../../adapters-and-accelerators/accelerator-rosettanet/step-2-creating-public-and-private-certificates.md)  
  
-   [Étape 3 : Importation publiques et privées des certificats](../../adapters-and-accelerators/accelerator-rosettanet/step-3-importing-public-and-private-certificates.md)  
  
-   [Étape 4 : Sécurisé Sockets Layer dans IIS](../../adapters-and-accelerators/accelerator-rosettanet/step-4-enabling-secure-sockets-layer-in-iis.md)  
  
> [!IMPORTANT]
>  Ces étapes sont décrites dans le didacticiel Double Action. À l’issue de chaque étape, revenez sur cette rubrique pour continuer le didacticiel sur les processus privés.
