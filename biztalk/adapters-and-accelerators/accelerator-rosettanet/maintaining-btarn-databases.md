# <a name="maintaining-btarn-databases"></a>Maintenance des bases de données BTARN
[!INCLUDE[btsCoName](../../includes/btsconame-md.md)]® [!INCLUDE[BTARN_CurrentVersion_FirstRef](../../includes/btarn-currentversion-firstref-md.md)] bases de données peuvent augmenter ainsi que leur taille peut affecter les performances du système. Cela peut être dû à des circonstances particulières qui laissent les entrées non utilisées dans les tables, telles que les pièces jointes orphelins ou résumés inutilisés. Il peut également se produire de ne pas supprimer les anciennes entrées dans les tables. Suivez les procédures décrites dans cette section pour mettre à jour votre [!INCLUDE[btaBTARN3.3abbrevnonumber](../../includes/btabtarn3-3abbrevnonumber-md.md)] donc aucun effet sur les performances des bases de données.
