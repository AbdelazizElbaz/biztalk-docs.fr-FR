# <a name="creating-and-configuring-the-contoso-solution"></a>Création et configuration de la Solution de Contoso
Cette section du didacticiel détaille les étapes à suivre pour l’organisation Contoso. L’organisation Contoso représente le fabricant et par conséquent agit comme le répondeur pour toutes les demandes entrantes de processus PIP (Partner Interface). Dans ce didacticiel, vous créez des accords de partenariat pour quatre PIP et générerez et déployez une orchestration pour envoyer une réponse appropriée en fonction du type d’adresse PIP.
