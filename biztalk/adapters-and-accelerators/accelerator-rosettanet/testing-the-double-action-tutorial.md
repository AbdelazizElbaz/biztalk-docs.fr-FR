# <a name="testing-the-double-action-tutorial"></a>Test du didacticiel double action
Dans cette section, vous utilisez les solutions Contoso et Fabrikam que vous avez créées dans les sections précédentes pour tester quatre processus PIP (Partner Interface Processes) : 0C2, 0C4, 3A2 et 3A4. Vous utilisez LOBWebApplication sur l'ordinateur Fabrikam pour effectuer la requête. Le système envoie la requête à l'aide d'un canal de communication amélioré à l'ordinateur Contoso. L'orchestration double action génère une réponse appropriée en fonction du processus PIP que vous utilisez.
