# <a name="administration-and-run-time-tools"></a>Outils d’administration et exécution
[!INCLUDE[btsCoName](../../includes/btsconame-md.md)]BizTalk Server et [!INCLUDE[btsCoName](../../includes/btsconame-md.md)] [!INCLUDE[BTARN_CurrentVersion_FirstRef](../../includes/btarn-currentversion-firstref-md.md)] fournissent un ensemble d’outils pour exécuter des tâches d’administration et d’exécution, telles que le déploiement, de gestion de la configuration, d’administration, de gestion des règles d’entreprise et l’analyse. Pour plus d’informations sur [!INCLUDE[btsBizTalkServerNoVersion](../../includes/btsbiztalkservernoversion-md.md)] outils, consultez [!INCLUDE[btsCoName](../../includes/btsconame-md.md)]à l’aide de BizTalk Server.  
  
## <a name="biztalk-explorer"></a>Explorateur BizTalk  
 Vous utilisez l’Explorateur BizTalk pour configurer et gérer des projets, les tiers et les orchestrations. L’Explorateur BizTalk affiche le contenu de la base de données de Configuration, des éléments tels que les assemblys, les ports et les parties.  
  
## <a name="biztalk-accelerator-for-rosettanet-management-console"></a>BizTalk Accelerator for RosettaNet Management Console  
 Vous utilisez la [!INCLUDE[btaBTARN3.3abbrevnonumber](../../includes/btabtarn3-3abbrevnonumber-md.md)] Console de gestion pour gérer les [!INCLUDE[btaBTARN3.3abbrevnonumber](../../includes/btabtarn3-3abbrevnonumber-md.md)] implémentation, y compris les éléments suivants :  
  
-   Paramètres de configuration de processus, des organisations de base, des partenaires et des accords  
  
-   Certificats  
  
-   BizTalk et [!INCLUDE[btaBTARN3.3abbrevnonumber](../../includes/btabtarn3-3abbrevnonumber-md.md)] [!INCLUDE[btsSQLServerNoVersion](../../includes/btssqlservernoversion-md.md)] bases de données  
  
-   Événements  
  
-   Journaux et alertes de performance ;  
  
## <a name="business-rule-engine"></a>moteur de règles d'entreprise  
 Le moteur de règles d’entreprise BizTalk (BRE) est un moteur d’inférence d’exécution que vous pouvez utiliser pour évaluer les règles par rapport à des faits et d’initier des actions basées sur les résultats de l’évaluation.  
  
## <a name="biztalk-administration-console"></a>Console Administration de BizTalk  
 Vous pouvez utiliser la Console Administration de BizTalk pour la création de rapports, d’analyse et de débogage des données d’entreprise actives et optimiser vos processus d’entreprise. Vous pouvez également utiliser la Console Administration de BizTalk pour surveiller l’intégrité d’un [!INCLUDE[btsBizTalkServerNoVersion](../../includes/btsbiztalkservernoversion-md.md)] batterie de serveurs et pour effectuer le suivi et afficher des événements spécifiques et les messages.  
  
## <a name="windows-performance-monitors"></a>Analyses de performances Windows  
 [!INCLUDE[btaBTARN3.3abbrevnonumber](../../includes/btabtarn3-3abbrevnonumber-md.md)]prend en charge de [!INCLUDE[btsWinNoVersion](../../includes/btswinnoversion-md.md)] des analyses de performances pour l’état de la connexion et le débit.
