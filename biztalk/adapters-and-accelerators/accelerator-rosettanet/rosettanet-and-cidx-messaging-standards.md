# <a name="rosettanet-and-cidx-messaging-standards"></a>RosettaNet et CIDX normes de messagerie
L’objectif principal de [!INCLUDE[btsCoName](../../includes/btsconame-md.md)]® [!INCLUDE[BTARN_CurrentVersion_FirstRef](../../includes/btarn-currentversion-firstref-md.md)] consiste à traiter les messages qui sont conformes aux normes de messagerie RosettaNet ou Chemical Industry données Exchange (CHEMICAL). Cette section fournit des informations sur les deux principales normes RosettaNet : le Framework RNIF (RosettaNet Implementation) et le processus d’Interface partenaires (PIP). Il fournit également des informations sur les normes de le CIDX Chem.  
  
 Les normes RosettaNet sont un ensemble de nouvelle, largement reconnue de normes pour l’implémentation de l’échange de messages automatique dans les scénarios de haute technologie. L’objectif de l’organisation RosettaNet consiste à créer, mettre en œuvre et promouvoir des normes de processus de commerce électronique. Ces normes sont couramment utilisées dans les applications de la chaîne d’approvisionnement et de manufacturière haute technologie. Les six conseils d’industrie qui régissent l’organisation RosettaNet montrent l’étendue de l’application :  
  
-   Composants électroniques  
  
-   Fabricants de semi-conducteurs  
  
-   Technologies de l’information  
  
-   Télécommunications  
  
-   Logistique  
  
-   Fournisseurs de solutions  
  
 Les normes de le CIDX chem sont un ensemble répandue des normes pour l’implémentation de l’échange de messages automatique dans les scénarios de secteur de produits chimiques.  
  
 Pour plus d’informations sur les normes, consultez le site Web de RosettaNet [http://go.microsoft.com/FWLink/?LinkID=33859](http://go.microsoft.com/FWLink/?LinkID=33859) et le site Web de CIDX [http://go.microsoft.com/FWLink/?LinkID=34540](http://go.microsoft.com/FWLink/?LinkID=34540).
