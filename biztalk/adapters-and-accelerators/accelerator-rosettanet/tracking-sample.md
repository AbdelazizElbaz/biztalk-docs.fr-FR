# <a name="tracking-sample"></a>Exemple de suivi
L’exemple de l’Interface utilisateur de suivi montre comment créer un [!INCLUDE[btsDotNet](../../includes/btsdotnet-md.md)]-application Web pour afficher l’état actuel de messages et de processus basée sur [!INCLUDE[btsCoName](../../includes/btsconame-md.md)] [!INCLUDE[BTARN_CurrentVersion_FirstRef](../../includes/btarn-currentversion-firstref-md.md)]. L’application présente à l’utilisateur avec un formulaire qui contient plusieurs critères de recherche. Lorsque l’utilisateur effectue une recherche, l’exemple suivi génère une liste détaillée montrant les messages et traite dans [!INCLUDE[btaBTARN3.3abbrevnonumber](../../includes/btabtarn3-3abbrevnonumber-md.md)] qui correspondent à ces critères.  
  
 Cet exemple est disponible en téléchargement à partir de la [!INCLUDE[btsCoName](../../includes/btsconame-md.md)] [!INCLUDE[btsBizTalkServerNoVersion](../../includes/btsbiztalkservernoversion-md.md)] site Web à l’adresse [http://go.microsoft.com/fwlink/?LinkId=37467](http://go.microsoft.com/fwlink/?LinkId=37467).
