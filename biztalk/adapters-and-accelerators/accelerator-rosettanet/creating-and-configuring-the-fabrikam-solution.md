# <a name="creating-and-configuring-the-fabrikam-solution"></a>Création et configuration de la Solution Fabrikam
Cette section du didacticiel détaille les étapes à suivre pour l’organisation de Fabrikam. L’organisation Fabrikam représente l’acheteur et par conséquent agit en tant que l’initiateur pour toutes les demandes de processus PIP (Partner Interface). Dans ce didacticiel, créer des accords pour quatre PIP, partenaire et générer et déployer un ASP line-of-business (LOB)[!INCLUDE[btsDotNet](../../includes/btsdotnet-md.md)] application d’envoyer une demande à l’ordinateur Contoso basée sur un certain type PIP.
