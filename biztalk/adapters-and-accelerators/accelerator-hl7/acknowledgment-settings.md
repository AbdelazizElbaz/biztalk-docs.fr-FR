# <a name="acknowledgment-settings"></a>Paramètres de l’accusé de réception
Vous utilisez la **accusé de réception** onglet de [!INCLUDE[btaBTAHL71.3abbrevnonumber](../../includes/btabtahl71-3abbrevnonumber-md.md)] l’Explorateur de Configuration (sous le niveau supérieur **Parties** onglet) pour configurer les paramètres de l’accusé de réception (ACK).  
  
 Les types d’accusé de réception suivants sont disponibles :  
  
-   **None**. Sélectionnez cette option si vous ne souhaitez pas configurer les accusés de réception.  
  
-   **D’origine**. Sélectionnez cette option pour configurer **MSH1-séparateur de champs**, **où MSH2 a été – l’encodage de caractères**et le **MSH8 – sécurité** options uniquement.  
  
-   **Amélioré**. Sélectionnez cette option pour configurer toutes les options disponibles d’accusé de réception.  
  
-   **Différée**. Sélectionnez cette option pour configurer **MSH1-séparateur de champs**, **où MSH2 a été – l’encodage de caractères**, et **MSH8 – sécurité** options uniquement.  
  
-   **Statique**. Sélectionnez cette option pour configurer le **en cas de réussite** et **en cas d’échec** options d’accusé de réception.  
  
 Une fois que le type d’accusé de réception est défini, vous pouvez définir les valeurs pour les champs d’en-tête et les accusés de réception, en fonction du type d’accusé de réception.
