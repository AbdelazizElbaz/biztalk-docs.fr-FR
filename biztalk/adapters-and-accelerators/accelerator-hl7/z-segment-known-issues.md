# <a name="z-segment-known-issues"></a>Problèmes connus du Segment Z
Cette section contient des informations utiles qui peuvent vous aider à éviter les erreurs de segment Z.  
  
## <a name="characters-not-recommended-to-use-for-naming-z-segments"></a>Caractères ne pas recommandés d’utiliser pour nommer les segments de Z  
 Il est recommandé de ne pas utiliser les caractères suivants pour nommer vos segments Z tels qu’ils sont les noms de segments HL7 peuvent provoquer des erreurs :  
  
-   FHS  
  
-   BHS  
  
-   BTS  
  
-   FTS
