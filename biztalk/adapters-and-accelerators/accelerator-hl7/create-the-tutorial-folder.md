# <a name="create-the-tutorial-folder"></a>Créer le dossier du didacticiel
Créez le dossier suivant à l’aide [!INCLUDE[btsWinNoVersion](../../includes/btswinnoversion-md.md)] Explorer :  
  
-   \<*lecteur*: > \Program Files\Microsoft BizTalk \<version > Accelerator for HL7\SDK\Interrogative didacticiel  
  
     Ce dossier vous permet de stocker la clé de requête, réponse et les fichiers d’accusé de réception que vous utilisez dans ce scénario.  
  
 Passez à [créer le fichier QRY^Q01.txt](../../adapters-and-accelerators/accelerator-hl7/create-the-qry-q01-txt-file.md).
