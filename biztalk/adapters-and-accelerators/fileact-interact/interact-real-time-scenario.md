# <a name="interact-real-time-scenario"></a>Interagir scénario en temps réel

Pour plus d’informations, consultez le site Web de SWIFT [http://go.microsoft.com/fwlink/?LinkId=100479](http://go.microsoft.com/fwlink/?LinkId=100479).
