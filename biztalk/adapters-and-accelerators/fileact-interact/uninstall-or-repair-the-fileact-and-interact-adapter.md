# <a name="uninstall-or-repair-the-fileact-and-interact-adapter"></a>Désinstaller ou réparer l’adaptateur FileAct et InterAct
Étapes pour désinstaller ou réparer l’adaptateur BizTalk FileAct et InterAct.  
  
## <a name="uninstall-the-fileact-and-interact-adapter"></a>Désinstaller l’adaptateur FileAct et InterAct  
  
1.  Dans le **le panneau de configuration**, sélectionnez **désinstaller un programme**.  
  
2.  Sélectionnez le **BizTalk FileAct et interagir de carte**, puis sélectionnez **désinstallation** ou **modification**.  
  
3.  Sélectionnez le **supprimer** option et sélectionnez **suivant**.  
  
4.  Dans la section Résumé, sélectionnez **désinstallation**.  
  
5.  Sélectionnez **Terminer** lorsque terminée.

## <a name="repair-the-fileact-and-interact-adapter-installation"></a>Réparer le FileAct et interagir l’installation de l’adaptateur  
  
1.  Dans le **le panneau de configuration**, sélectionnez **désinstaller un programme**.  
  
2.  Sélectionnez le **BizTalk FileAct et interagir de carte**, puis sélectionnez **modification**.  
3.  Sur le **Maintenance du programme** page, sélectionnez le **réparation** option et sélectionnez **suivant**.  
  
4.  Dans la section Résumé, sélectionnez **désinstallation**.  
  
5.  Sélectionnez **Terminer** lorsque terminée.
