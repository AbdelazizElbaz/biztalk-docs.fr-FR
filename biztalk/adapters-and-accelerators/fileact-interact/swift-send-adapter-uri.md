# <a name="swift-send-adapter-uri"></a>Adaptateur d’envoi SWIFT URI
L’URI de l’adaptateur d’envoi est le nom de répondeur en fonction construit (SWIFT : / /\<répondeur nom\>). Nom du répondeur est un nom unique défini par SWIFT. Il indique à qui le message doit être envoyé. SWIFT utilise cette valeur pour acheminer le message au destinataire.
