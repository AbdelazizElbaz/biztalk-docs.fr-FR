# <a name="accessibility-for-people-with-disabilities"></a>Accessibilité pour personnes handicapées
Microsoft s’engage à rendre ses produits plus faciles à utiliser. Pour plus d’informations sur les options d’accessibilité, dans l’aide en ligne Windows 2000 Server, dans la prise en main d’annuaire de Windows 2000, consultez « Accessibilité pour les personnes atteintes de handicaps ». Dans l’aide en ligne de Windows 2000 Professionnel, consultez « Accessibilité pour des besoins particuliers. »
