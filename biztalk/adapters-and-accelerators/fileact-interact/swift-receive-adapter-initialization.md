# <a name="swift-receive-adapter-initialization"></a>Initialisation de l’adaptateur de réception SWIFT
Le processus d’initialisation de l’application serveur SWIFT est similaire à celle de l’application cliente SWIFT. Les arguments d’initialisation sont passés en tant que paramètres de ligne de commande pour le récepteur exécutables. Nom de l’application configuré dans l’emplacement de réception au moment du design, est passé comme argument de ligne de commande. Nom de cette application est utilisé pour construire l’URI pour récupérer les propriétés de configuration de l’emplacement de réception.  
  
 Pour plus d’informations sur l’utilisation de l’argument de ligne de commande, consultez la rubrique « Comment pour configurer la réception adaptateur environnement » dans Microsoft [!INCLUDE[swift_adapter](../../includes/swift-adapter-md.md)] Guide d’Installation et Configuration.
