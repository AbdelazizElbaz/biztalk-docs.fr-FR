# <a name="interact-adapter-server-application"></a>Application de serveur de l’adaptateur interAct
Cette section présente une description générale de la composition des messages de demande d’application serveur. Voici les documents XML provenant du côté serveur SNL à l’application serveur. Par conséquent, ces messages sont caractéristique de la première partie des primitives SWIFTNet testés côté serveur.
