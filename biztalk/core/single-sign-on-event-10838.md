## <a name="explanation"></a>Explication  
 L'authentification unique peut créer un nom de base de données SQL incluant jusqu'à 123 caractères, pour tenir compte du nom de fichier journal par défaut.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Utilisez un nom conforme à ces instructions.
