# <a name="single-sign-on-event-10732"></a>Single Sign-On : Événement 10732
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10732|  
|Source de l'événement|ENTSSO|  
|Composant|N\A|  
|Nom symbolique|SSO_INFO_PS_SET_WINDOWS_PASSWORD|  
|Texte du message|Le mot de passe Windows a été mis à jour dans la base de données SSO.%r<br /><br /> ID de suivi : %1 %r<br /><br /> Le compte Windows : %2\\%3|  
  
## <a name="explanation"></a>Explication  
 Cette information indique que le mot de passe Windows a été mis à jour dans la base de données SSO.  
  
## <a name="user-action"></a>Action de l'utilisateur  
  
-   Aucune action utilisateur n’est nécessaire
