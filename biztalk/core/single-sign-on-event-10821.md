# <a name="single-sign-on-event-10821"></a>Single Sign-On : Événement 10821
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10821|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|ENTSSO_E_ADAPTER_ASSIGNED_TO_GROUP_ADAPTER|  
|Texte du message|L'adaptateur ne peut pas être supprimé car il est actuellement affecté à un adaptateur de groupe.|  
  
## <a name="explanation"></a>Explication  
 L'adaptateur ne peut pas être supprimé car il est actuellement affecté à un adaptateur de groupe.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Supprimez l'affectation de l'adaptateur, puis supprimez-le.
