# <a name="single-sign-on-event-10815"></a>Single Sign-On : Événement 10815
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10815|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|ENTSSO_E_WRONG_SECRET|  
|Texte du message|Secret principal incorrect.|  
  
## <a name="explanation"></a>Explication  
 Lors de la tentative de restauration du secret principal, il se peut que vous ayez spécifié un fichier incorrect.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Vérifiez de nouveau le fichier et assurez-vous qu'il est correct.
