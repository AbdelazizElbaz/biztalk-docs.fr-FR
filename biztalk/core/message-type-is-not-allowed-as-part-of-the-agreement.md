## <a name="explanation"></a>Explication  
 Cet événement d'erreur/d'avertissement/d'informations indique que BizTalk Server a pu traiter le document car le type de message du document n'est pas autorisé dans le cadre de l'accord.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Pour résoudre cette erreur, consultez les types de messages autorisés et non autorisés dans le cadre de l'accord.
