# <a name="using-design-time-xml-tools"></a>Utilisation des outils XML au moment de la conception
Les outils XML EDI permettent d'effectuer des opérations de conception sur les schémas et mappages EDI. Ils sont exécutés à partir du menu qui est affiché lorsque vous cliquez avec le bouton droit sur un schéma ou un mappage dans le volet Explorateur de solutions de Visual Studio.
