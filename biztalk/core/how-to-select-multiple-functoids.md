# <a name="how-to-select-multiple-functoids"></a>Sélection de plusieurs fonctoids
Lorsque vous avez affaire à de grands mappages, vous avez la possibilité d'effectuer des opérations sur plusieurs fonctoids à la fois. Pour ce faire, vous pouvez sélectionner tous les fonctoids qui vous intéressent puis effectuer l'opération voulue. C'est utile par exemple lorsque vous souhaitez déplacer un ensemble de fonctoids connectés ainsi que leurs liens d'une page de grille à une autre.  
  
### <a name="to-select-multiple-functoids-at-the-same-time"></a>Pour sélectionner plusieurs fonctoids simultanément  
  
1.  Dans une page de grille du Mappeur BizTalk, cliquez sur le premier fonctoid afin de le sélectionner. Le fonctoid est mis en surbrillance.  
  
2.  Dans la même page de grille du Mappeur BizTalk, maintenez la touche CTRL ou la touche MAJ enfoncée, puis cliquez sur un autre fonctoid pour le sélectionner également. Ce fonctoid est également mis en surbrillance.  
  
3.  Répétez l'étape 2 autant que nécessaire pour sélectionner tous les fonctoids voulus.  
  
    > [!NOTE]
    >  La sélection multiple dans une page de grille fonctionne pour les fonctoids et les liens simultanément et est requise pour le déplacement d'un ensemble de fonctoids connectés et de leurs liens de connexion d'une page de grille à une autre.
