# <a name="single-sign-on-event-10721"></a>Single Sign-On : Événement 10721
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10721|  
|Source de l'événement|ENTSSO|  
|Composant|N\A|  
|Nom symbolique|SSO_INFO_REPLAY_FILE_CLOSED_FULL|  
|Texte du message|Le fichier de relecture a été fermé car il est plein.%r<br /><br /> Nom du fichier : %1|  
  
## <a name="explanation"></a>Explication  
 Cet événement d'informations indique que le fichier de relecture a été fermé car il est plein. Le cas échéant, un autre fichier de relecture sera créé.  
  
## <a name="user-action"></a>Action de l'utilisateur  
  
-   Aucune action utilisateur n’est nécessaire
