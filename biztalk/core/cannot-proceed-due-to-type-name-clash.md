# <a name="cannot-proceed-due-to-type-name-clash"></a>Impossible de continuer en raison d'un conflit de nom de type
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]|  
|Version du produit|[!INCLUDE[btsWCFVersion](../includes/btswcfversion-md.md)]|  
|ID d'événement|0|  
|Source de l'événement|0|  
|Composant|0|  
|Nom symbolique|0|  
|Texte du message|Impossible de continuer en raison d'un conflit de nom de type. Le nom « {0} » existe déjà dans l'espace de noms|  
  
## <a name="explanation"></a>Explication  
 Cette erreur indique que plusieurs artefacts d'un même espace de noms défini portent le même nom.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Renommez les artefacts appartenant au même espace de noms ou placez-les dans un autre espace de noms.
