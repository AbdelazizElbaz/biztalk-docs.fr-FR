# <a name="single-sign-on-event-10502"></a>Single Sign-On : Événement 10502
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10502|  
|Source de l'événement|ENTSSO|  
|Composant|N\A|  
|Nom symbolique|SSO_INFO_SERVICE_STARTED|  
|Texte du message|Le service SSO a démarré.|  
  
## <a name="explanation"></a>Explication  
 Cet événement d'informations indique que le service d'authentification unique de l'entreprise a démarré.
