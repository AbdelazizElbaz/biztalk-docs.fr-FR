# <a name="undeploying-biztalk-applications"></a>Annulation du déploiement des applications BizTalk
Vous devez annuler le déploiement d'une application BizTalk pour la supprimer de votre groupe BizTalk et des ordinateurs hôtes. Cette section décrit les tâches qu'implique cette annulation.  
  
-   **Supprimer l’application BizTalk.** cette tâche supprime les données de l'application des bases de données BizTalk du groupe BizTalk. Cette tâche une fois effectuée, l'application n'apparaît plus dans la console Administration de BizTalk.  
  
-   **Désinstallez l’application BizTalk.**  cette tâche supprime les fichiers et les paramètres associés à l'application et à ses artefacts du système de fichiers local.  
  
-   **Suppression d’autres fichiers et paramètres.** vous pouvez également être amené à supprimer un certain nombre d'autres fichiers et paramètres associés à l'application.
