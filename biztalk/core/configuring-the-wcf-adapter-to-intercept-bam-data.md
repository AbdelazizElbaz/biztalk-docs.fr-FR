# <a name="configuring-the-wcf-adapter-to-intercept-bam-data"></a>Configuration de l'adaptateur WCF pour intercepter les données BAM
Cette section contient des informations sur les étapes à suivre pour configurer l'adaptateur WCF de BizTalk, afin de l'utiliser avec un intercepteur WCF BAM.  
  
 Les étapes de base sont les suivantes :  
  
-   ajout du comportement BAM au fichier machine.config ;  
  
-   création d'un adaptateur WCF BizTalk ;  
  
-   configuration de l'interception WCF BAM.
