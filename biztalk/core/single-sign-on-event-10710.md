# <a name="single-sign-on-event-10710"></a>Single Sign-On : Événement 10710
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10710|  
|Source de l'événement|ENTSSO|  
|Composant|N\A|  
|Nom symbolique|SSO_INFO_PS_WIN_CHANGE_DAMPED|  
|Texte du message|La modification d'un mot de passe Windows a été amortie (détectée comme doublon et ignorée).%r<br /><br /> ID de suivi : %1 %r<br /><br /> Compte Windows : %2 %r<br /><br /> Utilisateur client : %3|  
  
## <a name="explanation"></a>Explication  
 Cet événement d'informations indique que la modification d'un mot de passe Windows, détectée comme doublon, a été ignorée.  
  
## <a name="user-action"></a>Action de l'utilisateur  
  
-   Aucune action utilisateur n’est nécessaire.
