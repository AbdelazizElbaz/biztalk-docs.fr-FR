# <a name="management"></a>Gestion
TIBCO Rendezvous n'inclut pas de référentiels de métadonnées. Par ailleurs, l'assembly de gestion de l'adaptateur Microsoft BizTalk pour TIBCO Rendezvous n'inclut pas de fonctionnalités de navigation ou de génération de schémas. Vous devez donc fournir un schéma à BizTalk Server. Pour plus d’informations, consultez [génération de schéma dans l’adaptateur](../core/schema-generation-in-the-adapter.md).  
  
 L'adaptateur BizTalk pour TIBCO Rendezvous inclut un schéma avec des types prédéfinis. L'adaptateur utilise ces types lors de la génération de messages pour certains types de données spécifiques (tableaux). Pour plus d’informations, consultez [propriétés de contexte du Message BizTalk Server (gestionnaires d’envoi)](../core/biztalk-server-message-context-properties-send-handlers.md).
