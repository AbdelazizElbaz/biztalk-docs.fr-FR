# <a name="single-sign-on-event-10716"></a>Single Sign-On : Événement 10716
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10716|  
|Source de l'événement|ENTSSO|  
|Composant|N\A|  
|Nom symbolique|SSO_INFO_NEW_REPLAY_DIR|  
|Texte du message|Nouveau répertoire des fichiers de relecture créé.%r<br /><br /> L’utilisateur client : %1 %r<br /><br /> Répertoire des fichiers de relecture : %2|  
  
## <a name="explanation"></a>Explication  
 Cet événement d'informations indique que la synchronisation de mot de passe de l'authentification unique a créé un répertoire des fichiers de relecture.  
  
## <a name="user-action"></a>Action de l'utilisateur  
  
-   Aucune action utilisateur n’est nécessaire.
