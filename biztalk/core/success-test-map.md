# <a name="success---test-map"></a>Réussite : Test Map
**Code d’erreur**  
  
 btm1053  
  
 **Explication**  
  
 L'opération Test Map a été correctement effectuée sur le mappage indiqué. Le fichier de message d'instance de sortie a été généré.  
  
 **Action de l’utilisateur**  
  
 Si nécessaire, consultez le fichier de message de l'instance de sortie créé par l'opération Test Map.
