# <a name="moving-biztalk-server-databases"></a>Déplacement de bases de données BizTalk Server
Dans tout centre de données actif, les bases de données doivent parfois être transférées sur un autre matériel. Par exemple, il peut être nécessaire de déplacer une base de données lorsque celle-ci prend trop de place sur son support ou lorsque le support est mis à niveau ou remplacé par un matériel plus récent. Les procédures décrites dans cette section expliquent comment déplacer les bases de données BizTalk Server les plus susceptibles de faire l'objet d'un transfert.
