# <a name="receivelocations-receiveport-node"></a>ReceiveLocations (nœud ReceivePort)
Le nœud ReceiveLocations du nœud ReceivePort d'un fichier de liaison est le nœud parent de tous les nœuds ReceiveLocation qui contiennent des informations spécifiques sur les emplacements de réception exportés avec ce fichier de liaison.  
  
## <a name="nodes-in-the-receivelocations-node"></a>Nœuds du nœud ReceiveLocations  
 Le tableau suivant répertorie les propriétés que vous pouvez définir pour ce nœud d'un fichier de liaison :  
  
|**Nom**|**Type de nœud**|**Type de données**|**Description**|**Restrictions**|**Commentaires**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|[Emplacement de réception](../core/receivelocation-receivelocations-node.md)|Record|ReceiveLocation (ComplexType)|Spécifie les informations relatives à un emplacement de réception exporté avec le fichier de liaison.|Facultatif|Valeur par défaut : Aucun|
