 <span data-ttu-id="94049-105">Les analystes d'entreprise peuvent utiliser l'analyse BAM pour contrôler les processus qui couvrent plusieurs applications.</span><span class="sxs-lookup"><span data-stu-id="94049-105">Business analysts can use BAM to monitor processes that span multiple applications.</span></span> <span data-ttu-id="94049-106">Par exemple, un processus de bon de commande peut comporter plusieurs parties qui sont traitées par des applications différentes, notamment, BizTalk Server, l'activité de base de données et des messages échangés entre un site Web interne, un site Web de fournisseur et le site Web de l'expéditeur.</span><span class="sxs-lookup"><span data-stu-id="94049-106">For example, a purchase order process may have parts that are completed by different applications, including BizTalk Server, database activity, and messages sent from and to an internal Web site, a supplier Web site, and the shipper Web site.</span></span>  
  
 <span data-ttu-id="94049-107">Les utilisateurs d'entreprise peuvent utiliser l'analyse BAM pour afficher les données en temps réel et archivées (historiques) relatives à des processus d'entreprise.</span><span class="sxs-lookup"><span data-stu-id="94049-107">Business users can use BAM to view both real-time and archived (historical) data about business processes.</span></span> <span data-ttu-id="94049-108">Par exemple, un utilisateur d'entreprise peut souhaiter être informé de l'arrivée d'un bon de commande supérieur à 100 euros (données en temps réel) ou connaître le nombre de bons de commande supérieurs à 100 euros enregistrés dans le mois (données archivées).</span><span class="sxs-lookup"><span data-stu-id="94049-108">For example, a business user may want to know when a purchase order arrives for more than $100 (real-time data) or, how many purchase orders for more than $100 arrived this month (archived).</span></span> <span data-ttu-id="94049-109">L'analyse BAM regroupe les données en temps réel et les données archivées pour accroître la vitesse de présentation des données.</span><span class="sxs-lookup"><span data-stu-id="94049-109">BAM aggregates real-time and archived data to improve the speed at which it presents data.</span></span>
