# <a name="single-sign-on-event-10831"></a>Single Sign-On : Événement 10831
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10831|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|ENTSSO_E_SSO_DATABASE_EXISTS|  
|Texte du message|La base de données SSO spécifiée existe déjà.|  
  
## <a name="explanation"></a>Explication  
 Lors de la configuration, vous avez tenté de créer une base de données qui existe déjà.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Vérifiez votre système ou donnez un autre nom à votre nouvelle base de données.
