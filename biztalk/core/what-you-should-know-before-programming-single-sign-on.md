# <a name="what-you-should-know-before-programming-single-sign-on"></a>Éléments à connaître avant de programmer l’authentification unique
Pour que cette documentation vous soit utile, vous devez être familiarisé avec les éléments suivants :  
  
-   Les systèmes d'exploitation [!INCLUDE[btsPlatformsComApis](../includes/btsplatformscomapis-md.md)] Microsoft. Vous devez connaître en particulier les fonctionnalités Windows liées à la sécurité.  
  
-   Les fonctionnalités administratives de l'authentification unique de l'entreprise, en particulier, celles permettant d’effectuer des actions administratives à l’aide de l’interface utilisateur.  
  
 En fonction de l'API et du développement utilisés, les interfaces suivantes doivent également vous être familières :  
  
-   COM  
  
-   .NET Framework et le Common Language Runtime  
  
-   Windows Networking, et en particulier la manière d'envoyer et de recevoir des tickets
