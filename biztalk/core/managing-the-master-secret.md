# <a name="managing-the-master-secret"></a>Gestion du Secret
Le secret principal est la clé utilisée pour chiffrer les informations stockées dans la base de données SSO. Si le serveur de secret principal échoue et que vous perdez le secret, vous ne pourrez pas récupérer les informations stockées dans la base de données SSO. Aussi, est-il très important de sauvegarder le secret principal dès que celui-ci est généré.
