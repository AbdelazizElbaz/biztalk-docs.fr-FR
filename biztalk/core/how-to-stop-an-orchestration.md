# <a name="how-to-stop-an-orchestration"></a>Arrêt d'une orchestration
Cette rubrique explique comment arrêter une orchestration à l'aide de la console Administration de BizTalk Server. L'arrêt d'une orchestration désactive et suspend tous les messages d'activation entrants.  
  
> [!NOTE]
>  Le développeur peut arrêter une orchestration au cours du processus de développement à l'aide de la procédure décrite dans cette rubrique.  
  
## <a name="prerequisites"></a>Conditions préalables  
 Pour exécuter la procédure dans cette rubrique, vous devez être connecté avec un compte qui est membre du groupe opérateurs BizTalk Server ou du groupe Administrateurs de BizTalk Server. Pour plus d’informations sur les autorisations, consultez [autorisations requises pour déployer et gérer une Application BizTalk](../core/permissions-required-for-deploying-and-managing-a-biztalk-application.md).  
  
### <a name="to-stop-an-orchestration"></a>Pour arrêter une orchestration  
  
1.  Cliquez sur **Démarrer**, cliquez sur **tous les programmes**, cliquez sur [!INCLUDE[btsBizTalkServerStartMenuItemui](../includes/btsbiztalkserverstartmenuitemui-md.md)], puis cliquez sur **Administration de BizTalk Server**.  
  
2.  Dans l’arborescence de la console, développez **Administration de BizTalk Server**, développez le groupe BizTalk, **Applications**, puis développez l’application contenant l’orchestration que vous souhaitez arrêter.  
  
3.  Cliquez sur **Orchestrations**, avec le bouton droit de l’orchestration, puis cliquez sur **arrêter**.
