# <a name="testing-published-web-services"></a>Test des services web publiés
Lorsque vous développez un service Web, vous pouvez le tester en le republiant. Pour ce faire, vous devez redémarrer IIS (Internet Information Services).  
  
 Pour tester le service Web publié, créez une application cliente de service Web ou utilisez un navigateur (Internet Explorer, par exemple).
