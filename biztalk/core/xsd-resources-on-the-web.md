# <a name="xsd-resources-on-the-web"></a>Ressources XSD sur le Web
Ce document ne propose pas d'informations détaillées ni même résumées sur le langage XSD (XML Schema definition). Pour obtenir des informations plus détaillées, consultez les spécifications et éléments de base XSD proposés sur le site Web du consortium W3C (World Wide Web Consortium). Il est recommandé de consulter plus spécifiquement les didacticiels suivants, le premier permettant d’assimiler rapidement les éléments de base, et les deux derniers constituant la source d’informations par définition concernant XSD. Pour consulter les didacticiels sur XSD, accédez aux pages W3C suivantes :  
  
-   XML Schema Part 0 : Primer [http://go.microsoft.com/fwlink/?LinkId=16787](http://go.microsoft.com/fwlink/?LinkId=16787).  
  
-   XML Schema Part 1 : Structures, [http://go.microsoft.com/fwlink/?LinkId=16788](http://go.microsoft.com/fwlink/?LinkId=16788).  
  
-   XML Schema Part 2 : Datatypes, [http://go.microsoft.com/fwlink/?LinkId=16789](http://go.microsoft.com/fwlink/?LinkId=16789).  
  
 Il est conseillé de posséder une solide compréhension des principes XSD avant d'utiliser l’Éditeur BizTalk pour construire des schémas complexes.
