# <a name="how-to-construct-a-web-message-with-no-message-parts"></a>Développement d'un message web sans partie de message
Un message Web n'ayant aucune partie de message est une méthode n'ayant aucun paramètre d'entrée.  
  
### <a name="to-construct-a-web-message-that-does-not-have-message-parts"></a>Pour construire un message Web n'ayant aucune partie de message  
  
1.  Ouvrez une orchestration, ouvrez le **boîte à outils** et cliquez sur le **Orchestrations BizTalk** onglet.  
  
2.  Ajouter un **construire un Message** forme.  
  
3.  Sélectionnez le message pour le **construire un Message** forme.  
  
> [!NOTE]
>  Vous ne sont pas obligés d’avoir le **assignation du Message** mettre en forme à l’intérieur de la **construire un Message** forme.
