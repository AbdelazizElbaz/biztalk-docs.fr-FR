# <a name="using-the-microsoft-biztalk-loadgen-2007-tool"></a>Utilisation de l'outil Microsoft BizTalk LoadGen 2007
L'outil Microsoft BizTalk LoadGen 2007 permet aux développeurs et professionnels de l'informatique de simuler la présence d'une charge sur un serveur BizTalk Server. Avec cet outil, vous pouvez simuler une charge pour instrumenter les performances et le stress par rapport à un déploiement BizTalk. Cet outil peut en outre être étendu par les développeurs pour simuler une charge pour les transports personnalisés.  
  
> [!NOTE]
>  L’outil LoadGen 2007 est disponible pour téléchargement à l’adresse [http://go.microsoft.com/fwlink/?LinkId=59841](http://go.microsoft.com/fwlink/?LinkId=59841). La version précédente de cet outil, l’outil de génération de charge de BizTalk Server 2004 est disponible pour téléchargement à l’adresse [http://go.microsoft.com/fwlink/?LinkId=108999](http://go.microsoft.com/fwlink/?LinkId=108999).  
  
 Cet outil doit être utilisé exclusivement dans un environnement de test et non de production. L'outil est fourni « en l'état » et n'est pas pris en charge.  
  
> [!NOTE]
>  Vous pouvez associer l'outil BizTalk Server 2004 Load Generation Tool à l'adaptateur MSMQ, mais pour cela, quelques étapes supplémentaires sont nécessaires. Pour obtenir des instructions, consultez [à l’aide de LoadGen 2007 avec MSMQ](../core/using-loadgen-2007-with-msmq.md).  
  
> [!NOTE]
>  L'outil LoadGen n'est pas pris en charge par les ordinateurs 64 bits. Vous pouvez l'utiliser à distance sur un ordinateur 32 bits de façon à générer une charge sur un serveur 64 bits.
