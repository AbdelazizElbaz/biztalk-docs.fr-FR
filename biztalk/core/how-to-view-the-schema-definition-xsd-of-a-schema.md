# <a name="how-to-view-the-schema-definition-xsd-of-a-schema"></a>Affichage du langage XSD (XML Schema Definition) d'un schéma
Cette rubrique décrit comment afficher le langage XSD d'un schéma à l'aide de la console Administration de BizTalk Server. Cela vous permet de dépanner les erreurs de validation du schéma ou de valider le déploiement du schéma correct.  
  
## <a name="prerequisites"></a>Conditions préalables  
 Pour exécuter cette procédure, vous devez ouvrir une session en tant que membre du groupe Administrateurs BizTalk Server ou Opérateurs BizTalk Server. Pour plus d’informations sur les autorisations, consultez [autorisations requises pour déployer et gérer une Application BizTalk](../core/permissions-required-for-deploying-and-managing-a-biztalk-application.md).  
  
### <a name="to-view-the-xsd-of-a-schema"></a>Pour afficher le langage XSD d'un schéma  
  
1.  Cliquez sur **Démarrer**, cliquez sur **tous les programmes**, cliquez sur [!INCLUDE[btsBizTalkServerStartMenuItemui](../includes/btsbiztalkserverstartmenuitemui-md.md)], puis cliquez sur **Administration de BizTalk Server**.  
  
2.  Dans l’arborescence de la console, développez **Administration de BizTalk Server**, développez le groupe BizTalk contenant le schéma pour lequel vous souhaitez afficher le langage XSD, puis développez l’application contenant le schéma.  
  
3.  Cliquez sur **schémas**, cliquez sur le schéma, puis cliquez sur **propriétés**.  
  
4.  Dans le volet gauche, cliquez sur **vue schéma**.  
  
     Le langage XSD s'affiche dans le volet de droite.
