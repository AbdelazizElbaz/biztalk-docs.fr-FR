# <a name="creating-web-ports"></a>Création de ports web
Les ports Web sont des ports spécialement configurés pour l'utilisation (appel) des services Web. Un port Web peut contenir plusieurs opérations qui représentent diverses méthodes Web unidirectionnelles (demande uniquement) et bidirectionnelles (demande-réponse). Chaque opération dans un port Web représente une méthode d'un service Web.  
  
 Vous devez ajouter une référence Web avant de créer un port Web.  
  
> [!NOTE]
>  Vous ne pouvez pas supprimer, couper, copier, coller ou modifier la configuration pour les types de ports Web et les types de messages Web.
