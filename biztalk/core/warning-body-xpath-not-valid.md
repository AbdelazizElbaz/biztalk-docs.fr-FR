# <a name="warning---body-xpath-not-valid"></a>Avertissement : XPath de corps non valide
**Code d’erreur**  
  
 BEC1009  
  
 **Explication**  
  
 Le **XPath de corps** propriété du nœud racine indiqué dans le schéma d’enveloppe fait référence à un nœud qui n’est pas un **enregistrement** nœud, en fonction des besoins.  
  
 **Action de l’utilisateur**  
  
 Sélectionnez le nœud racine indiqué et sélectionnez la valeur de la **XPath de corps** propriété qui identifie correctement le niveau supérieur **enregistrement** nœud du corps du message associé.
