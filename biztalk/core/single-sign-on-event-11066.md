# <a name="single-sign-on-event-11066"></a>Single Sign-On : Événement 11066
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|11066|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|SSO_INFO_PASSWORD_SYNC_MIIS_STARTED|  
|Texte du message|La synchronisation des mots de passe pour Windows (à partir de MIIS) a démarré.%r|  
  
## <a name="explanation"></a>Explication  
 Ce message d'information indique que la synchronisation des mots de passe pour Windows (à partir de MIIS) a démarré.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Aucune action de l'utilisateur n'est requise.
