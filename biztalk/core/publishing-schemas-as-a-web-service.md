# <a name="publishing-schemas-as-a-web-service"></a>Publication de schémas en tant que services web
L'Assistant Publication de services Web BizTalk permet de créer des services Web qui utilisent des schémas existants. Vous devez déclarer les services Web, les méthodes Web et les schémas de demande et de réponse que vous souhaitez publier. L'Assistant vous permet de définir l'espace de noms cible, les configurations d'en-tête SOAP et l'emplacement du projet de service Web généré.  
  
> [!IMPORTANT]
>  Vous devez compiler les assemblys BizTalk contenant les schémas.  
  
> [!IMPORTANT]
>  Chaque méthode Web peut inclure un schéma de demande et un schéma de réponse.  
  
> [!NOTE]
>  Pour publier les schémas en tant que service Web, [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)] et [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] doivent être installés.  
  
> [!IMPORTANT]
>  Avant d’exécuter l’Assistant de publication des Services Web BizTalk, vous devez activer les services Web. Pour plus d’informations sur l’activation des services Web, consultez [activation des Services Web](../core/enabling-web-services.md).
