# <a name="single-sign-on-event-10839"></a>Single Sign-On : Événement 10839
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10839|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|ENTSSO_E_MAX_PASSWORD|  
|Texte du message|La longueur maximale du mot de passe ne doit pas dépasser 256 caractères.|  
  
## <a name="explanation"></a>Explication  
 La longueur maximale du mot de passe ne doit pas dépasser 256 caractères.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Utilisez un mot de passe respectant ces restrictions.
