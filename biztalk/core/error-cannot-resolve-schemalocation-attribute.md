# <a name="error---cannot-resolve-schemalocation-attribute"></a>Erreur : Impossible de résoudre l’attribut schemaLocation
**Code d’erreur**  
  
 BEC8001  
  
 **Explication**  
  
 L'Éditeur BizTalk ne parvient pas à trouver un schéma importé par le schéma que vous ouvrez.  
  
 **Action de l’utilisateur**  
  
 Cliquez sur le **schéma** nœud. Dans la fenêtre Propriétés, cliquez sur le **importations** propriété. Les points de suspension (**...** ) bouton devient actif. Cliquez sur le bouton . Le **importations** boîte de dialogue s’affiche. Dans celle-ci, modifiez le chemin d'accès au schéma importé.
