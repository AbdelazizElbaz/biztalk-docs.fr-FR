# <a name="artifacts"></a>Artefacts
Cette rubrique présente les divers artefacts de BizTalk Server. Les artefacts incluent notamment les assemblys, les certificats de sécurité, les stratégies de règles d'entreprise, les fichiers de configuration BAM et les liaisons nécessaires au fonctionnement d'une application BizTalk. Vous pouvez afficher et gérer les artefacts d'une application à partir de la console Administration de BizTalk.
