# <a name="implementing-message-security"></a>Implémentation de la sécurité des messages
Vous pouvez configurer votre environnement Microsoft [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] pour qu'il utilise des certificats permettant de recevoir et d'envoyer des messages signés et chiffrés. L'utilisation de certificats pour le chiffrement et les signatures numériques permet à BizTalk Server d'effectuer les opérations suivantes :  
  
-   Envoyer et recevoir des données fiables.  
  
-   S'assurer que les données traitées sont sécurisées.  
  
-   S'assurer que les tiers autorisés reçoivent ses messages.  
  
-   S'assurer qu'il reçoit les messages envoyés par les tiers autorisés.  
  
 Cette section fournit des informations sur la configuration des pipelines BizTalk Server, des emplacements de réception, des ports et de l'environnement BizTalk Server afin d'implémenter la sécurité des messages.  
  
 Pour plus d’informations sur la sécurité de message, consultez [planification de la sécurité de Message](../core/planning-message-security.md).
