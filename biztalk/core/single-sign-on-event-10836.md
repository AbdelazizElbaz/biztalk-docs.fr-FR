# <a name="single-sign-on-event-10836"></a>Single Sign-On : Événement 10836
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10836|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|ENTSSO_E_FILE_PASSWORD_TOO_SHORT|  
|Texte du message|Le mot de passe spécifié est trop court. Il doit inclure au moins 7 caractères.|  
  
## <a name="explanation"></a>Explication  
 Le mot de passe d'un fichier de sauvegarde doit inclure au moins sept caractères.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Entrez un mot de passe incluant au moins sept caractères.
