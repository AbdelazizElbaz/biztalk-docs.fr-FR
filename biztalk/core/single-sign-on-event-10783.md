# <a name="single-sign-on-event-10783"></a>Single Sign-On : Événement 10783
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10783|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|ENTSSO_E_NO_SECRET|  
|Texte du message|Impossible d'effectuer le chiffrement ou le déchiffrement car le secret n'est pas disponible depuis le serveur de secret principal. Recherchez les erreurs associées dans le journal des événements.|  
  
## <a name="explanation"></a>Explication  
 Il se peut qu'il y ait eu un problème lors de la configuration.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Pour plus d'informations, consultez le journal des événements et recherchez les erreurs associées.
