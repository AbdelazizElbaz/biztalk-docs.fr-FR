# <a name="communication-pattern"></a>Modèle de communication
Chaque *type de port* a un modèle de communication. Le modèle de communication détermine si des transmissions unidirectionnelles ou bidirectionnelles (requête-réponse) peuvent avoir lieu sur les ports du type donné. Pour plus d’informations, consultez la **Assistant Configuration du Port** [!INCLUDE[ui-guidance-developers-reference](../includes/ui-guidance-developers-reference.md)].
  
> [!NOTE]
>  Les différents ports d'un même type auront toujours le même modèle de communication, mais pas nécessairement la même direction de communication.
