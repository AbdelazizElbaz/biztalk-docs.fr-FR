# <a name="before-installing-the-business-process-management-solution"></a>Avant l'installation de la solution de gestion des processus d'entreprise

## <a name="requirements"></a>Spécifications
Pour déployer la solution de gestion des processus d'entreprise sur un seul ordinateur, vous devez installer les logiciels requis suivants :  
  
-   Microsoft Windows
  
-   Microsoft [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]  
  
-   Microsoft Message Queuing (MSMQ) avec prise en charge de HTTP MSMQ
  
-   Microsoft SQL Server
  
-   Microsoft Internet Information Services WWW et FTP  
  
-   ASP.NET activé  
  
-   Microsoft Visual Studio  
  
-   Microsoft Excel
