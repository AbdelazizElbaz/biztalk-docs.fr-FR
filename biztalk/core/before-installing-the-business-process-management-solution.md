# <a name="before-installing-the-business-process-management-solution"></a>Avant l'installation de la solution de gestion des processus d'entreprise
Pour déployer la solution de gestion des processus d'entreprise sur un seul ordinateur, vous devez installer les logiciels requis suivants :  
  
-   Microsoft [!INCLUDE[btsWinSvr2k8](../includes/btswinsvr2k8-md.md)] ou [!INCLUDE[btsWinSvr2k8R2](../includes/btswinsvr2k8r2-md.md)]  
  
-   Microsoft [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]  
  
-   Microsoft Message Queuing (MSMQ) 3.0 (prise en charge HTTP MSMQ)  
  
-   Microsoft [!INCLUDE[SQLServer2008or2005](../includes/sqlserver2008or2005-md.md)]  
  
-   Microsoft Internet Information Services 7.0 ou Internet Information Services 7.5 (WWW et FTP)  
  
-   ASP.NET 4 activé  
  
-   Microsoft [!INCLUDE[vs2010](../includes/vs2010-md.md)]  
  
-   Microsoft Excel
