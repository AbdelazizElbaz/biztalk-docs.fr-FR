# <a name="before-you-begin-the-tutorial"></a>Avant de commencer le didacticiel
Avant de commencer les didacticiels, vous devez installer BizTalk Server sur un seul ordinateur et également les fichiers des didacticiels. Les didacticiels ne fournissent pas d'instructions pour les environnements multiserveurs. Si vous utilisez un environnement [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] distribué, vous devez en tenir compte lors de l'exécution des étapes des didacticiels.
