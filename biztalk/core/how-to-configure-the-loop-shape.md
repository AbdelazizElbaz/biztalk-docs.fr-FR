# <a name="how-to-configure-the-loop-shape"></a>Configuration de la forme Boucle
![](../core/media/ebiz-orch-loop.gif "ebiz_orch_loop")  
Forme Boucle  
  
### <a name="to-configure-a-loop-shape"></a>Pour configurer une forme Boucle  
  
1.  Si l’éditeur d’Expression BizTalk n’est pas visible, cliquez sur le **boucle** mettre en forme et cliquez sur **modifier l’Expression booléenne**, ou dans la fenêtre Propriétés, cliquez sur le **points de suspension** (**...** ) bouton pour le **Expression** propriété.  
  
2.  Dans l'Éditeur d'expression BizTalk, créez une expression booléenne pour la condition d'itération. Pour plus d’informations, consultez [spécifications et Limitations pour les Expressions](../core/requirements-and-limitations-for-expressions.md).
