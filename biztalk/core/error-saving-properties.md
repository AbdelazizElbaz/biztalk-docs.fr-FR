|Texte du message|Erreur lors de la sauvegarde des propriétés|  
  
## <a name="explanation"></a>Explication  
 Cette erreur se produit lorsque l'adaptateur enregistre les propriétés de l'emplacement de réception ou du port d'envoi à partir de la base de données.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Le message d'erreur doit contenir les informations nécessaires à la résolution du problème.
