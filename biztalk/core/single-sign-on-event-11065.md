# <a name="single-sign-on-event-11065"></a>Single Sign-On : Événement 11065
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|11065|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|SSO_INFO_PASSWORD_SYNC_MIIS_STARTING|  
|Texte du message|La synchronisation des mots de passe pour Windows (à partir de MIIS) est en cours de démarrage.%r|  
  
## <a name="explanation"></a>Explication  
 Ce message d'information indique que la synchronisation des mots de passe pour Windows (à partir de MIIS) est en cours de démarrage.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Aucune action de l'utilisateur n'est requise.
