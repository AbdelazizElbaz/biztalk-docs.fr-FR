# <a name="how-to-open-the-biztalk-server-administration-console"></a>Ouverture de la console Administration de BizTalk Server
Vous pouvez utiliser les procédures suivantes pour ouvrir la console Administration de BizTalk Server et pour trier ou actualiser les données affichées dans le volet des détails.  
  
## <a name="prerequisites"></a>Conditions préalables  
 Pour exécuter cette procédure, vous devez avoir ouvert une session en tant que membre du groupe des opérateurs de BizTalk Server ou du groupe d'administrateurs de BizTalk Server.  
  
### <a name="to-open-the-biztalk-server-administration-console"></a>Pour ouvrir la console Administration de BizTalk Server  
  
-   Cliquez sur **Démarrer**, cliquez sur **programmes**, cliquez sur [!INCLUDE[btsBizTalkServerStartMenuItemui](../includes/btsbiztalkserverstartmenuitemui-md.md)], puis cliquez sur **Administration de BizTalk Server**.  
  
     Ou  
  
-   Cliquez sur **Démarrer**, cliquez sur **exécuter**, type **BTSmmcLauncher.exe**, puis cliquez sur **OK**.  
  
### <a name="to-refresh-the-biztalk-server-administration-console"></a>Pour actualiser la console Administration de BizTalk Server  
  
-   Cliquez sur le nœud dans l’arborescence de la console vous souhaitez actualiser, puis cliquez sur **Actualiser**.  
  
     Tous les objets de ce nœud sont actualisés.  
  
### <a name="to-sort-columns-in-the-details-pane-of-the-biztalk-server-administration-console"></a>Pour trier les colonnes du volet des détails de la console Administration de BizTalk Server  
  
-   Dans l'arborescence de la console, cliquez sur un élément pour afficher les informations qui le concernent dans le volet des détails.  
  
-   Dans le volet des détails, cliquez sur l'en-tête de la colonne en fonction duquel vous voulez procéder au tri. Cela trie les éléments sous l'en-tête dans l'ordre croissant.  
  
-   Cliquez à nouveau sur l'en-tête de colonne pour trier dans l'ordre décroissant (inverse).  
  
### <a name="to-reorder-the-columns-in-the-details-pane-of-the-biztalk-server-administration-console"></a>Pour changer l'ordre des colonnes du volet des détails de la console Administration de BizTalk Server  
  
-   Dans le volet des détails, cliquez sur une colonne et faites-la glisser à un nouvel emplacement pour modifier l'ordre des colonnes dans le volet des détails.
