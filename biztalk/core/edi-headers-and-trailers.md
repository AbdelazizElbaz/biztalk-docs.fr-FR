 <span data-ttu-id="aa329-121">Si un en-tête ISA est immédiatement suivi d'un code de fin IEA, l'échange est vide.</span><span class="sxs-lookup"><span data-stu-id="aa329-121">If an ISA header is followed immediately by an IEA trailer, the interchange is empty.</span></span> <span data-ttu-id="aa329-122">Si un document informatisé existe, l'en-tête GS et le code de fin GE doivent être présents ; sinon, ils sont conditionnels.</span><span class="sxs-lookup"><span data-stu-id="aa329-122">If a transaction set exists, the GS header and GE trailer must be present; otherwise, they are conditional.</span></span>  
  
 <span data-ttu-id="aa329-123">Les champs d'en-tête de contrôle d'échange ISA dans un message X12 sont de longueur fixe.</span><span class="sxs-lookup"><span data-stu-id="aa329-123">The ISA interchange control header fields in an X12-encoded message are of fixed length.</span></span> <span data-ttu-id="aa329-124">Pour certains champs, vous pouvez entrer une valeur inférieure à celle du champ.</span><span class="sxs-lookup"><span data-stu-id="aa329-124">For some of the fields, you can enter a value less than the fixed length of the field.</span></span> <span data-ttu-id="aa329-125">Dans ce cas, l'échange doit contenir des espaces de fin (pour un champ de chaîne) ou des zéros de début (pour un champ numérique) afin que chaque champ soit conforme à la longueur requise.</span><span class="sxs-lookup"><span data-stu-id="aa329-125">If you do so, the interchange must contain trailing spaces (for a string field) or leading zeroes (for a numerical field) so that each field is of the required length.</span></span> <span data-ttu-id="aa329-126">Lors de la création d'un échange sortant, [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] entre des espaces de fin ou des zéros de début pour définir un en-tête de contrôle de l'échange de longueur appropriée.</span><span class="sxs-lookup"><span data-stu-id="aa329-126">When creating an outbound interchange, [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] will enter trailing spaces or leading zeroes to create an interchange control header of the right length.</span></span> <span data-ttu-id="aa329-127">Les champs d'en-tête de groupe GS et les champs d'en-tête de document informatisé ST ne sont pas de longueur fixe.</span><span class="sxs-lookup"><span data-stu-id="aa329-127">The GS group header fields and the ST transaction set header fields are not of fixed length.</span></span>
