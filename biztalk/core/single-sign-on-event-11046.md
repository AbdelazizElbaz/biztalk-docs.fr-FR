## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|11046|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|SSO_WARN_EXISTING_MAPPING_EXTERNAL|  
|Texte du message|Impossible de créer un mappage car il existe déjà un mappage pour le compte externe spécifié.%r<br /><br /> ID de suivi : %1 %r<br /><br /> Nom de l’application : %2 %r<br /><br /> Compte externe : %3|  
  
## <a name="explanation"></a>Explication  
 Un mappage existe déjà pour le compte externe spécifié et les doublons ne sont pas autorisés.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Aucune action utilisateur n’est nécessaire.
