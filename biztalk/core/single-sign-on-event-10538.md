# <a name="single-sign-on-event-10538"></a>Single Sign-On : Événement 10538
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10538|  
|Source de l'événement|ENTSSO|  
|Composant|CO|  
|Nom symbolique|SSO_WARN_APP_DISABLED|  
|Texte du message|L'application est actuellement désactivée.%r<br /><br /> Nom de l’application : %1|  
  
## <a name="explanation"></a>Explication  
 Cet événement d'avertissement indique que l'application associée à authentification unique a été désactivée par un administrateur d'application.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Pour résoudre cet avertissement, effectuez l'une ou plusieurs des opérations suivantes :  
  
-   Contactez votre administrateur d'application pour activer l'application associée à authentification unique. Pour ce faire, utilisez les outils d'administration de l'authentification unique (interface graphique ou ligne de commande).  
  
 Pour plus d'informations, consultez les ressources suivantes dans l'aide de [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] :  
  
-   [Comment activer une Application associée](../core/how-to-enable-an-affiliate-application.md)  
  
-   [Comment désactiver une Application associée](../core/how-to-disable-an-affiliate-application.md)
