# <a name="certificate-stores-that-biztalk-server-uses"></a><span data-ttu-id="f67fb-102">Magasins de certificats utilisés par BizTalk Server</span><span class="sxs-lookup"><span data-stu-id="f67fb-102">Certificate Stores that BizTalk Server Uses</span></span>
<span data-ttu-id="f67fb-103">BizTalk Server utilise deux types de magasins de certificats : le magasin de certificats Autres personnes pour les clés publiques et le magasin de certificats Personnel pour chaque compte de service d'instance d'hôte pour la clé privée.</span><span class="sxs-lookup"><span data-stu-id="f67fb-103">BizTalk Server uses two types of certificate stores, the Other People certificate store for public keys, and the Personal certificate store for each host instance service account for the private key.</span></span>  
  
 <span data-ttu-id="f67fb-104">**Magasin de certificats autres personnes.**</span><span class="sxs-lookup"><span data-stu-id="f67fb-104">**Other People certificate store.**</span></span> <span data-ttu-id="f67fb-105">Comme leur nom l'indique, les certificats de clé publique sont publics et accessibles par tout individu ayant accès à l'ordinateur sur lequel ils sont stockés.</span><span class="sxs-lookup"><span data-stu-id="f67fb-105">Public key certificates, as their name implies, are public and accessible by anyone with access to the computer on which they are stored.</span></span> <span data-ttu-id="f67fb-106">BizTalk Server utilise des certificats de clé publique pour chiffrer des messages destinés à des tiers spécifiques et pour vérifier les signatures numériques des messages entrants envoyés par des tiers spécifiques.</span><span class="sxs-lookup"><span data-stu-id="f67fb-106">BizTalk Server uses public key certificates to encrypt messages to specific parties and to verify the digital signatures for incoming messages from specific parties.</span></span> <span data-ttu-id="f67fb-107">Windows fournit le magasin de certificats Autres personnes pour stocker les certificats de clé publique utilisés sur l'ordinateur.</span><span class="sxs-lookup"><span data-stu-id="f67fb-107">Windows provides the Other People certificate store to store the public key certificates used on the computer.</span></span> <span data-ttu-id="f67fb-108">Tous les utilisateurs peuvent lire et utiliser les certificats de ce magasin et les administrateurs Windows ont les autorisations pour maintenir ce magasin.</span><span class="sxs-lookup"><span data-stu-id="f67fb-108">All users can read and use the certificates in this store, and Windows administrators have permissions to maintain this store.</span></span>
