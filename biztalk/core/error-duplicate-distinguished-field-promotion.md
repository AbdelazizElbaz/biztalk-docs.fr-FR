# <a name="error---duplicate-distinguished-field-promotion"></a>Erreur : Promotion de champ distinctif dupliquée
**Code d’erreur**  
  
 BEC2015  
  
 **Explication**  
  
 Le nœud indiqué dans le schéma est promu plusieurs fois en tant que champ distinctif.  
  
 **Action de l’utilisateur**  
  
 Utilisez le **champs distinctifs** onglet de la **promouvoir les propriétés** boîte de dialogue, accédé via la **promouvoir les propriétés** propriété de la **schéma** nœud afin de supprimer tous les mais parmi les promotions de champ distinctif qui sont associées au même nœud de ce schéma.
