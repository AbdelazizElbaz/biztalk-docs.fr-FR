# <a name="fragment-not-allowed-in-target-namespace"></a>Fragment non autorisé dans l'espace de noms cible
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]|

|ID d'événement|0|  
|Source de l'événement|0|  
|Composant|0|  
|Nom symbolique|0|  
|Texte du message|Fragment '#' non autorisé dans l'espace de noms cible|  
  
## <a name="explanation"></a>Explication  
 Cette erreur indique que l'espace de noms cible spécifié ne peut pas contenir de fragments.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Spécifiez un espace de noms cible différent sans fragment.
