# <a name="receiveportref-port-node"></a>ReceivePortRef (nœud Port)
Le nœud ReceivePortRef du nœud Port d'un fichier de liaison contient des informations sur un port de réception référencé par un service.  
  
## <a name="nodes-in-the-receiveportref-node"></a>Nœuds du nœud ReceivePortRef  
 Le tableau suivant répertorie les propriétés que vous pouvez définir pour ce nœud d'un fichier de liaison :  
  
|**Nom**|**Type de nœud**|**Type de données**|**Description**|**Restrictions**|**Commentaires**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|Nom|Attribut|xs:string|Spécifie le nom d'un port de réception référencé par un service.|Facultatif|Valeur par défaut : vide|
