# <a name="query-not-allowed-in-target-namespace"></a>Requête non autorisée dans l'espace de noms cible
## <a name="details"></a>Détails  
  
|||  
|-|-|

|Nom symbolique|0|  
|Texte du message|Requête '?' non autorisée dans l'espace de noms cible.|  
  
## <a name="explanation"></a>Explication  
 Cette erreur indique que l'espace de noms cible spécifié ne peut pas contenir de requêtes.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Spécifiez un espace de noms cible différent sans requête.
