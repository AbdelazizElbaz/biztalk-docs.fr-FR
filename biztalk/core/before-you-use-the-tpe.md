# <a name="before-you-use-the-tpe"></a>Avant de commencer à utiliser l'Éditeur TPE
Avant de commencer à travailler avec l'Éditeur de modèle de suivi (TPE), nous vous recommandons de vous familiariser avec BizTalk Server et de comprendre comment il facilite les communications à travers diverses plateformes et applications.  
  
 En tant que développeur de solutions, vous devez également disposer de connaissances spécifiques au sujet de l'implémentation des processus d'entreprise de votre organisation avec des orchestrations Microsoft BizTalk Server. Vous devez savoir comment utiliser le Concepteur d'orchestration pour produire des orchestrations. Pour plus d’informations sur [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] orchestrations, consultez [sur les Orchestrations](../core/about-orchestrations.md).  
  
> [!IMPORTANT]
>  Pour pouvoir utiliser l'Éditeur de modèle de suivi, vous devez choisir une installation complète ou personnalisée de BizTalk Server et veiller à sélectionner les options « Outils et kit de développement » ou « Outils d'administration ». Pour plus d’informations, consultez [vue d’ensemble de l’Installation de BizTalk Server 2013 et 2013 R2](http://msdn.microsoft.com/library/8041926c-cfc9-4eaf-9c28-a2c6e8015bc5).  
  
> [!NOTE]
>  L’Éditeur de modèle de suivi n'assure pas le suivi des parties de messages .Net. Il est impossible de visualiser le schéma de message de ces types de messages.
