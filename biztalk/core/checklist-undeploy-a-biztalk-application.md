# <a name="checklist-undeploy-a-biztalk-application"></a>Liste de vérification : Annuler le déploiement d’une Application BizTalk
|Étape|Référence|  
|----------|---------------|  
|Supprimez l'application BizTalk du groupe BizTalk. Cette opération supprime l'application de la console Administration de BizTalk et des bases de données BizTalk Server.|[Comment supprimer une Application BizTalk du groupe BizTalk](../core/how-to-delete-a-biztalk-application-from-the-biztalk-group.md)|  
|Désinstallez l'application de tous les ordinateurs sur lesquels elle était installée.|[Comment désinstaller une Application BizTalk](../core/how-to-uninstall-a-biztalk-application.md)|  
|Le cas échéant, supprimez du GAC (Global Assembly Cache) les assemblys inclus dans l'application.|[Comment désinstaller un Assembly du GAC](http://msdn.microsoft.com/library/464706a8-f902-4d05-a724-19169facd2b4)|  
|Supprimez de l'ordinateur local tous les fichiers et paramètres qui n'ont pas été automatiquement supprimés lors des précédentes étapes.|[Comment supprimer les autres fichiers et paramètres d’une Application BizTalk](../core/how-to-remove-other-files-and-settings-for-a-biztalk-application.md)|
