# <a name="single-sign-on-event-10810"></a>Single Sign-On : Événement 10810
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10810|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|ENTSSO_E_HISSO_INVALID_CREDENTIALS|  
|Texte du message|Les informations d'identification spécifiées ne sont pas valides ou ne correspondent pas à celles dans la base de données SSO.|  
  
## <a name="explanation"></a>Explication  
 Les informations d'identification spécifiées ne sont pas valides ou ne correspondent pas à celles dans la base de données SSO.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Définissez les informations dans la base de données SSO afin qu'elles correspondent à celles du système externe.
