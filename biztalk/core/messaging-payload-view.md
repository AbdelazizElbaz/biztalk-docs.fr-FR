# <a name="messaging-payload-view"></a>Affichage de la charge de messagerie
L'écran de la charge de messagerie affiche le schéma du message XML associé à l'action sélectionnée (Message envoyé ou Message reçu, par exemple). Cette vue peut s'afficher à partir du menu contextuel de certaines des formes de la vue de planification d'orchestration.  
  
## <a name="working-with-the-messaging-payload-view"></a>Utilisation de l'affichage de la charge de messagerie  
 Vous sélectionnez votre affichage de la planification d’Orchestration en cliquant sur le bouton Sélectionner Source d’événement et en cliquant sur l’élément de menu Sélectionner le schéma de messagerie charge utile. Une fois le schéma sélectionné, vous pouvez faire glisser les éléments du schéma vers les dossiers des éléments de données de l'activité, indiquant ainsi que vous souhaitez extraire ces données de l'expression XPath contenue dans le message associé à l'action.  
  
 Vous pouvez ouvrir les écrans de charge de messagerie à partir de l'écran de la planification d'orchestration en cliquant avec le bouton droit sur une forme contenant la charge d'un message. Dans le menu contextuel qui s'affiche, vous pouvez cliquer sur l'élément de menu Charge de messagerie pour extraire le schéma de propriété de la messagerie.  
  
 Pour plus d’informations sur les expressions XPath, consultez [à l’aide de XPath dans une assignation de Message](../core/using-xpaths-in-message-assignments.md).
