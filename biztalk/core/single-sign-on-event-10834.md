# <a name="single-sign-on-event-10834"></a>Single Sign-On : Événement 10834
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10834|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|ENTSSO_E_SSO_NOT_SSO_ADMIN|  
|Texte du message|L'utilisateur actuel n'est pas membre du compte Administrateurs de l'authentification unique spécifié.|  
  
## <a name="explanation"></a>Explication  
 Pour exécuter cette action, l'utilisateur doit être membre du compte Administrateurs de l'authentification unique spécifié.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Pour plus d’informations sur les comptes, consultez [comment spécifier les administrateurs de l’authentification unique et Affiliate Administrators Accounts](../core/how-to-specify-sso-administrators-and-affiliate-administrators-accounts.md).
