# <a name="single-sign-on-event-11045"></a>Single Sign-On : Événement 11045
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|11045|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|SSO_WARN_EXISTING_MAPPING_WINDOWS|  
|Texte du message|Impossible de créer un mappage car il en existe déjà un pour le compte Windows spécifié.%r<br /><br /> ID de suivi : %1 %r<br /><br /> Nom de l’application : %2 %r<br /><br /> Compte Windows : %3|  
  
## <a name="explanation"></a>Explication  
 Un mappage existe déjà pour le compte Windows spécifié et les doublons ne sont pas autorisés.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Aucune action utilisateur n’est nécessaire.
