# <a name="how-to-rename-nodes"></a>Comment renommer des nœuds
Il pourra vous arriver de vouloir renommer des nœuds existants. Cette rubrique décrit deux variations de cette tâche.  
  
### <a name="to-rename-a-single-node"></a>Pour renommer un seul nœud  
  
1.  Sélectionnez le **enregistrement** nœud, **Fieldelement** nœud, ou **attribut de champ** nœud que vous souhaitez renommer.  
  
2.  Sur le **BizTalk** menu, cliquez sur **renommer**.  
  
3.  Attribuez un nouveau nom au nœud, puis appuyez sur Entrée.  
  
    > [!NOTE]
    >  Vous pouvez également renommer un nœud en modifiant la valeur de sa **nom de nœud** propriété.  
  
### <a name="to-rename-multiple-nodes-to-have-the-same-name"></a>Pour donner le même nom à plusieurs nœuds  
  
1.  Sélectionnez le premier nœud de l'ensemble des nœuds en attente d'un même nom.  
  
2.  Appuyez sur CTRL et maintenez la touche enfoncée, puis sélectionnez les nœuds restants.  
  
3.  Dans la fenêtre Propriétés, modifiez la valeur de la **nom de nœud** propriété et terminez en appuyant sur ENTRÉE.  
  
     Tous les nœuds sélectionnés sont mis à jour dans l'arborescence de schéma avec le nouveau nom.
