# <a name="using-host-initiated-sso-functionality"></a>Utilisation de l’hôte initiée par les fonctionnalités de l’authentification unique
L'authentification unique initiée par l'hôte utilise la fonctionnalité de transition de protocole de Windows Server 2008 pour effectuer une authentification unique pour l'utilisateur non-Windows. Cette fonctionnalité nécessite Windows Server 2008 et doit être dans un domaine qui a son **niveau fonctionnel du domaine** la valeur **Windows Server 2008**.
