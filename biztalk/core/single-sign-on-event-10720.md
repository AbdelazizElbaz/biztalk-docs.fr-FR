# <a name="single-sign-on-event-10720"></a>Single Sign-On : Événement 10720

|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10720|  
|Source de l'événement|ENTSSO|  
|Composant|N\A|  
|Nom symbolique|SSO_INFO_REPLAY_RECORD_WRITTEN|  
|Texte du message|La modification d'un mot de passe externe a été stockée dans le fichier de relecture.%r<br /><br /> ID de suivi : %1 %r<br /><br /> Carte : %2 %r<br /><br /> Compte externe : %3 %r<br /><br /> Nom du fichier : %4 %r<br /><br /> Enregistrer le nombre : %5|  
  
## <a name="explanation"></a>Explication  
 Cet événement d'informations indique que la modification d'un mot de passe externe a été stockée dans le fichier de relecture.
