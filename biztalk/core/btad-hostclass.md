# <a name="btadhostclass"></a>BTAD_HostClass
BTAD_HostClass indique si l'opération est effectuée sur la base de données de gestion BizTalk ou sur l'ordinateur local.  
  
## <a name="remarks"></a>Notes  
 Le tableau suivant répertorie les valeurs possibles de BTAD_HostClass.  
  
|Valeur|Description|  
|-----------|-----------------|  
|ConfigurationDb|Indique que l'opération d'importation est effectuée sur la base de données de gestion BizTalk pour le groupe|  
|BizTalkHostInstance|Indique que l'installation ou la désinstallation est effectuée sur l'ordinateur local|
