# <a name="testing-policies"></a>Test des stratégies
Cette section explique comment tester les stratégies et apporte des éclaircissements sur ce que sont les informations de suivi.  
  
 Vous pouvez utiliser Microsoft [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] pour analyser les activités des règles et effectuer le suivi du progrès global d'une orchestration utilisant l'infrastructure des règles d'entreprise. Les informations de suivi des règles que vous trouvez dans la page Présentation du groupe sont les mêmes que la sortie des tests de stratégie de l'Éditeur des règles d'entreprise.  
  
 Pour plus d’informations sur l’utilisation du suivi, consultez [comment configurer le suivi d’une stratégie](../core/how-to-configure-tracking-for-a-policy.md).
