# <a name="btadserver"></a>BTAD_Server
Lorsque les scripts de pré-traitement et de post-traitement sont exécutés lors de l'importation de l'application, la variable d'environnement BTAD_Server est définie. Elle spécifie le nom de l'instance de SQL Server exécutant la base de données de gestion BizTalk pour le groupe.
