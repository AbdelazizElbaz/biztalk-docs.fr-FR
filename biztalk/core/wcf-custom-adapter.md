# <a name="wcf-custom-adapter"></a>Adaptateur WCF-Custom
L’adaptateur WCF-Custom est un adaptateur in-process qui permet de configurer les emplacements de réception et tous les paramètres pour Windows Communication Foundation (WCF) liaison et le comportement qui sont appliquent aux ports d’envoi [!INCLUDE[btsCoName](../includes/btsconame-md.md)] [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]. Vous pouvez utiliser cet adaptateur pour les scénarios que les adaptateurs WCF standard ne prennent pas en charge. L'adaptateur WCF-Custom permet également d'utiliser les points d'extensibilité WCF dans [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)].
