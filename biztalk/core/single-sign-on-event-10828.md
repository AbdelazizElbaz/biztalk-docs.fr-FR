# <a name="single-sign-on-event-10828"></a>Single Sign-On : Événement 10828
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10828|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|ENTSSO_E_PSADMIN_ADAPTER_ALREADY_ASSIGNED|  
|Texte du message|L'adaptateur est déjà affecté à un adaptateur de groupe.|  
  
## <a name="explanation"></a>Explication  
 Un adaptateur peut uniquement être affecté à un adaptateur de groupe.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Aucune action utilisateur n’est nécessaire.
