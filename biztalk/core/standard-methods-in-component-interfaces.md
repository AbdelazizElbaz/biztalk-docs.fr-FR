# <a name="standard-methods-in-component-interfaces"></a>Méthodes standard dans les Interfaces de composant
Les méthodes standard pour l'interface de composant sont les suivantes :  
  
-   `Create`  
  
-   `Find`  
  
-   `Get`  
  
-   `Save`  
  
 Seules les méthodes du composant sous-jacent sont disponibles. Par exemple, si le composant sous-jacent ne contient pas de fonctionnalités `Add`, `Create` n'est pas disponible.  
  
## <a name="viewing-or-changing-available-methods"></a>Affichage et modification des méthodes disponibles  
  
#### <a name="to-view-or-change-available-methods"></a>Pour afficher et modifier les méthodes disponibles  
  
1.  Ouvrez l’interface de composant **propriétés** boîte de dialogue.  
  
     ![](../core/media/psadapter-46-ps-properties.gif "PSAdapter_46_PS_Properties")  
  
2.  Cliquez sur le **méthodes Standard** onglet.  
  
3.  Sélectionnez les méthodes souhaitées, puis cliquez sur **OK**.
