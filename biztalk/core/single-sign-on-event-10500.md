# <a name="single-sign-on-event-10500"></a>Single Sign-On : Événement 10500
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10500|  
|Source de l'événement|ENTSSO|  
|Composant|N\A|  
|Nom symbolique|SSO_ERROR_EVENT_LOG_FAILED|  
|Texte du message|Le système d'enregistrement des événements a rencontré une erreur. Il se peut que les messages du journal des événements soient incorrects pour cette application.|  
  
## <a name="explanation"></a>Explication  
 Cet événement d'erreur indique que la journalisation des événements a échoué.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Pour résoudre cette erreur, procédez comme suit :  
  
-   Aucune action n'est nécessaire.
