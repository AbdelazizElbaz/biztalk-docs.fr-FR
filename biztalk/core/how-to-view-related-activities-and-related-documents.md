# <a name="how-to-view-related-activities-and-related-documents"></a>Affichage des activités et des documents associés
Dans le portail Web BAM, vous pouvez afficher les activités et les documents associés par le biais de la page des résultats de la recherche d'activités d'entreprise. Vous pouvez naviguer d'une activité associée à l'autre, par exemple, si vous recherchez un bon de commande déterminé, vous pouvez examiner les diverses livraisons figurant sur le bon de commande. Vous pouvez également vérifier les documents associés, par exemple, pour le même bon de commande, vous pouvez examiner le bon de commande associé.  
  
 Pour connaître les étapes rechercher des activités, consultez [recherche d’activités](../core/how-to-search-for-activities.md)  
  
### <a name="to-view-related-activities"></a>Pour afficher des activités associées  
  
1.  Dans le portail Web BAM, recherchez une activité spécifique.  
  
2.  Dans la page État de l’activité, cliquez sur le signe plus (+) icônes dans le **des activités connexes** zone du cadre contenu.  
  
     La liste des activités associées apparaît.  
  
3.  Cliquez sur une activité associée dans la liste.  
  
### <a name="to-view-related-documents"></a>Pour afficher des documents associés  
  
1.  Dans le portail Web BAM, recherchez une activité spécifique...  
  
2.  Dans la page État de l’activité, cliquez sur le signe plus (+) icônes dans le **Documents associés** zone du cadre contenu.  
  
     La liste des documents associés apparaît.
