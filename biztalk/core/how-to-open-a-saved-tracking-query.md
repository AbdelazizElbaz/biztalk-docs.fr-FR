# <a name="how-to-open-a-saved-tracking-query"></a>Ouverture d'une requête de suivi enregistrée
Vous pouvez ouvrir une requête de message ou d'instance de service pour des données de suivi actives (messages et instances de service en cours d'exécution). Cette opération est effectuée sous l'onglet Requête de la page Hub du groupe dans la console Administration de BizTalk Server.  
  
## <a name="prerequisites"></a>Conditions préalables  
 Pour exécuter cette procédure, vous devez ouvrir une session en tant que membre du groupe Administrateurs de BizTalk Server.  
  
### <a name="to-open-a-saved-query"></a>Pour ouvrir une requête enregistrée  
  
1.  Cliquez sur **Démarrer**, cliquez sur **tous les programmes**, cliquez sur [!INCLUDE[btsBizTalkServerStartMenuItemui](../includes/btsbiztalkserverstartmenuitemui-md.md)], puis cliquez sur **Administration de BizTalk Server**.  
  
2.  Sur le **Hub du groupe** sous le **nouvelle requête** , cliquez sur **ouvrir la requête**.  
  
3.  Cliquez sur la requête de votre choix, puis cliquez sur **ouvrir**.  
  
4.  Si un avertissement de sécurité s’affiche la première fois que vous ouvrez une requête, vérifiez éventuellement **ne plus afficher ce message** pour l’empêcher de se reproduire. Cliquez sur **OK** pour continuer.
