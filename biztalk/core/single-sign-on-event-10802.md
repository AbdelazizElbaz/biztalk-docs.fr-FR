# <a name="single-sign-on-event-10802"></a>Single Sign-On : Événement 10802
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10802|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|ENTSSO_E_WRONG_SESSION|  
|Texte du message|Une demande a été reçue pour la session erronée. Il se peut qu'elle ait été déconnectée.|  
  
## <a name="explanation"></a>Explication  
 Il se peut que la session ait été déconnectée.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Arrêtez l'interface, puis redémarrez-la.
