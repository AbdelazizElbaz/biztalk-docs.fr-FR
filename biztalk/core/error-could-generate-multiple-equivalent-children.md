# <a name="error---could-generate-multiple-equivalent-children"></a>Erreur : génération de plusieurs enfants équivalent possible
**Code d’erreur**  
  
 btm1026  
  
 **Explication**  
  
 Des liens vers le schéma de destination inappropriée permettent à plusieurs nœuds au sein d’un **équivalent** nœud de groupe.  
  
 **Action de l’utilisateur**  
  
 Corrigez les liens vers le schéma de destination, à l’aide des fonctoids logiques, ainsi qu’un seul nœud dans le **équivalent** nœud de groupe peut être généré lors du mappage.
