# <a name="distributionlistcollection-node"></a>Nœud DistributionListCollection
Le nœud DistributionListCollection d'un fichier de liaison est le nœud parent de tous les nœuds DistributionList qui contiennent des informations spécifiques sur les listes de distribution exportées avec le fichier de liaison.  
  
## <a name="nodes-in-the-distributionlistcollection-node"></a>Nœuds du nœud DistributionListCollection  
 Le tableau suivant répertorie les propriétés que vous pouvez définir pour ce nœud d'un fichier de liaison :  
  
|**Nom**|**Type de nœud**|**Type de données**|**Description**|**Restrictions**|**Commentaires**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|[DistributionList](../core/distributionlist-distributionlistcollection-node.md)|Record|DistributionList (ComplexType)|Spécifie les informations relatives à une liste de distribution exportée avec le fichier de liaison.|Facultatif|Valeur par défaut : Aucun|
