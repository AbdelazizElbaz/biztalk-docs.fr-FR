# <a name="error---node-not-valid-for-property-field-promotion"></a>Erreur : nœud non valide pour la Promotion de champ de propriété
**Code d’erreur**  
  
 BEC2001  
  
 **Explication**  
  
 Le nœud indiqué ne peut pas être promu en tant que champ de propriété, car il ne comporte pas le contenu de texte requis. Uniquement **Fieldelement** nœuds, **attribut de champ** nœuds, ou **enregistrement** comportant un contenu mixte ou simple peut être promu en tant que champs de propriété.  
  
 **Action de l’utilisateur**  
  
 Utilisez le **champs de propriété** onglet dans le **promouvoir les propriétés** boîte de dialogue pour supprimer la promotion de propriétés non autorisée.
