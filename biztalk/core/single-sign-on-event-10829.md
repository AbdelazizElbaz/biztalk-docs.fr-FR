# <a name="single-sign-on-event-10829"></a>Single Sign-On : Événement 10829
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10829|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|ENTSSO_E_PSADMIN_INVALID_ADAPTER_TYPE_2|  
|Texte du message|Type d'application non valide pour l'adaptateur. « Adaptateur de synchronisation de mot de passe » et « Adaptateur de groupe de synchronisation de mot de passe » sont les types d'applications valides.|  
  
## <a name="explanation"></a>Explication  
 Type d'application non valide pour l'adaptateur. « Adaptateur de synchronisation de mot de passe » et « Adaptateur de groupe de synchronisation de mot de passe » sont les types d'applications valides.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Utilisez l'un des types d'applications valides.
