# <a name="unable-to-create-one-or-more-biztalk-receive-locations"></a>Impossible de créer un ou plusieurs emplacements de réception BizTalk
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]|  
|Version du produit|[!INCLUDE[btsWCFVersion](../includes/btswcfversion-md.md)]|  
|ID d'événement|0|  
|Source de l'événement|0|  
|Composant|0|  
|Nom symbolique|0|  
|Texte du message|Impossible de créer un ou plusieurs emplacements de réception BizTalk.|  
  
## <a name="explanation"></a>Explication  
 Cette erreur indique que l'Assistant Publication WCF BizTalk n'a pas pu créer les emplacements de réception hébergeant un adaptateur WCF isolé.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Vérifiez que cette instance BizTalk est démarrée.
