## <a name="explanation"></a>Explication  
 Une des situations suivantes s’est produite : l’adresse spécifiée dans le champ URI (uniform resource identifier) doit commencer par http:// ou https://. ou le schéma ne correspond pas aux données spécifiées dans l'implémentation de l'élément de transport de liaison.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Entrez un URI d'adresse proxy valide commençant par http:// ou https://
