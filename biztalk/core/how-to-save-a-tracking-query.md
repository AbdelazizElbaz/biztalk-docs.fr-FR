# <a name="how-to-save-a-tracking-query"></a>Enregistrement d'une requête de suivi
Vous pouvez enregistrer une requête d'instance de message ou de service à l'aide de l'interface Générateur de requête dans la console Administration de BizTalk Server.  
  
## <a name="prerequisites"></a>Conditions préalables  
 Vous devez être connecté en tant que membre du groupe opérateurs BizTalk Server pour effectuer cette procédure.  
  
### <a name="to-save-a-tracking-query"></a>Pour enregistrer une requête de suivi  
  
1.  Cliquez sur **Démarrer**, cliquez sur **programmes**, cliquez sur [!INCLUDE[btsBizTalkServerStartMenuItemui](../includes/btsbiztalkserverstartmenuitemui-md.md)], puis cliquez sur **Administration de BizTalk Server**.  
  
2.  Sur le **Hub du groupe** page, sur le **nouvelle requête** , cliquez sur **enregistrer en tant que** pour enregistrer les commandes SQL dans le **Expression de requête** zone.  
  
3.  Dans le **enregistrer la requête sous** boîte de dialogue, accédez au répertoire de destination où vous souhaitez enregistrer la requête. Veillez à inclure un nom de fichier approprié pour la requête.
