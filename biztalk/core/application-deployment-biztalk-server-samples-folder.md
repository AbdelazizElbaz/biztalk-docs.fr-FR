# <a name="application-deployment-biztalk-server-samples-folder"></a>Déploiement d'applications (dossier d'exemples BizTalk Server)
Le kit de développement logiciel (SDK) de Microsoft [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] inclut plusieurs exemples de déploiement d'application. Cette section fournit des informations détaillées sur les fonctionnalités présentées dans les exemples, des instructions sur la création et l'exécution des exemples et les résultats attendus.  
  
## <a name="in-this-section"></a>Dans cette section  
  
-   [Modèle (exemple de déploiement d’Application)](../core/template-application-deployment-sample.md). Cet exemple montre comment créer des scripts de pré-traitement et de post-traitement qui s’exécutent pendant une ou plusieurs phases de déploiement de l’application et effectuer des tâches personnalisés.  
  
-   [CreateApp (exemple de déploiement d’Application)](../core/createapp-application-deployment-sample.md). Cet exemple illustre comment utiliser l’outil de ligne de commande BTSTask pour automatiser les tâches de déploiement d’application.  
  
-   [SelectiveBindingImport (exemple de déploiement d’Application)](../core/selectivebindingimport-application-deployment-sample.md). Cet exemple illustre comment utiliser des scripts de pré-traitement et post-traitement pour importer des fichiers de liaison dans une application qui sont appropriées pour l’environnement de destination.  
  
-   [VisualStudioHostRestart (exemple de déploiement d’Application)](../core/visualstudiohostrestart-application-deployment-sample.md). Cet exemple fournit illustre comment redémarrer toutes les instances d’hôte sur l’ordinateur local à l’aide d’un script.
