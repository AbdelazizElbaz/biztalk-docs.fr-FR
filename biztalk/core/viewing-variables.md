# <a name="viewing-variables"></a>Affichage des variables
Vous pouvez afficher les variables de deux manières :  
  
-   lorsque vous effectuez le suivi de données archivées, hors connexion ;  
  
-   lorsque vous réalisez un débogage interactif sur des données actives.  
  
 Lorsque vous utilisez des données archivées, les variables affichées peuvent être les suivantes :  
  
-   des variables pas encore incluses dans l'étendue ;  
  
-   actives ;  
  
-   des valeurs sur lesquelles vous pouvez cliquer pour obtenir plus de détails ;  
  
-   des valeurs hors de portée, dont vous ne pouvez plus accéder aux détails.
