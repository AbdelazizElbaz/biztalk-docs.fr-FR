# <a name="document-structure-enforcement-in-the-xml-disassembler-pipeline-component"></a>Mise en œuvre de Structure de document dans le composant de Pipeline désassembleur XML
Si un schéma de document ou d’enveloppe est référencé explicitement dans le Désassembleur XML, ce dernier ne traite que les documents dont les types de messages se conforment au schéma. Aucun autre document n’est traité, même si un schéma déployé est référencé pour lui.  
  
 Le Désassembleur XML respecte l’ordre spécifié pour les schémas d’enveloppe, mais pas celui des schémas de document.
