# <a name="soap-headers-with-published-web-services"></a>En-têtes SOAP avec les services Web publiés
Vous pouvez ajouter des en-têtes SOAP à vos services Web publiés. Dans l’Assistant de publication de Services Web de BizTalk sur le **propriétés des Services Web** page, vous pouvez vérifier le **ajouter des en-têtes SOAP** case pour ajouter des en-têtes SOAP à vos services Web publiés.
