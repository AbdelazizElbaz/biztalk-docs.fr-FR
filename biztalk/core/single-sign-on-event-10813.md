# <a name="single-sign-on-event-10813"></a>Single Sign-On : Événement 10813
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10813|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|ENTSSO_E_WINDOWS_UPDATE_FAILED_MORE_RECENT|  
|Texte du message|Le mot de passe Windows dans la base de données SSO est plus récent.|  
  
## <a name="explanation"></a>Explication  
 Un mot de passe obsolète a été reçu du système Windows et sera ignoré.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Aucune action utilisateur n’est nécessaire.
