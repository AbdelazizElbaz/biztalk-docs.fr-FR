# <a name="single-sign-on-event-10505"></a>Single Sign-On : Événement 10505
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10505|  
|Source de l'événement|ENTSSO|  
|Composant|N\A|  
|Nom symbolique|SSO_INFO_SERVICE_STOPPED|  
|Texte du message|Le service d'authentification unique s'est arrêté.|  
  
## <a name="explanation"></a>Explication  
 Cet événement d'informations indique que le service d'authentification unique de l'entreprise s'est arrêté.
