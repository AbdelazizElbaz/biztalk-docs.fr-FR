# <a name="error---no-script-type-for-scripting-functoid"></a>Erreur : aucun Type de Script pour le fonctoid script
**Code d’erreur**  
  
 btm1003  
  
 **Explication**  
  
 Le **le Type de Script** propriété du fonctoid **script** fonctoid n’a pas été défini.  
  
 **Action de l’utilisateur**  
  
 Sélectionnez le **script** fonctoid, cliquez sur le bouton de sélection (**...** ) associés à la **Script** propriété de Microsoft [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)] fenêtre Propriétés, puis définissez les valeurs appropriées dans le **configurer le Script du fonctoid** boîte de dialogue, y compris le **le Type de Script** propriété.
