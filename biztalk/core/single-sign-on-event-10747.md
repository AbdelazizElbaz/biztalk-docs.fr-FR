# <a name="single-sign-on-event-10747"></a>Single Sign-On : Événement 10747
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10747|  
|Source de l'événement|N\A|  
|Composant|N\A|  
|Nom symbolique|SSO_ERROR_UNKNOWN_NOTIFICATION|  
|Texte du message|Un type de notification inconnu a été reçu.%r<br /><br /> ID de suivi : %1 %r<br /><br /> Carte : %2 %r<br /><br /> Type de notification : %3|  
  
## <a name="explanation"></a>Explication  
 Cet événement d'erreur indique qu'une erreur interne avec un type de notification non valide a été détectée par le service d'authentification unique.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Pour résoudre cette erreur, procédez comme suit :  
  
-   Recherchez les événements associés dans les journaux des événements système et de l'application.  
  
 Pour plus d'informations, consultez les ressources ci-dessous.
