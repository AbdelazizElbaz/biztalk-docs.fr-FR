# <a name="biztalk-project-properties-window"></a>Fenêtre Propriétés du projet BizTalk
La fenêtre de propriétés d'un projet BizTalk dans [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)] contient les propriétés suivantes :  
  
## <a name="bpel-compliance"></a>Compatibilité BPEL  
 Spécifie s'il faut générer le résultat de la compatibilité avec Business Process Execution Language (BPEL).  
  
## <a name="project-file"></a>Fichier projet  
 Nom du fichier contenant la version, la configuration et d'autres informations sur le projet.  
  
## <a name="project-folder"></a>Dossier du projet  
 Emplacement du fichier de projet.
