# <a name="single-sign-on-event-10582"></a>Single Sign-On : Événement 10582
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10582|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|SSO_INFO_SSO_ENABLED|  
|Texte du message|Le système SSO a été activé.%r<br /><br /> ID de suivi : %1 %r<br /><br /> Ordinateur client : %2 %r<br /><br /> Utilisateur client : %3|  
  
## <a name="explanation"></a>Explication  
 Ce message d'information peut s'avérer utile pour effectuer le suivi d'événements majeurs liés à la sécurité qui surviennent dans le système SSO. Ce message indique que le système SSO a été activé.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Aucune action de l'utilisateur n'est requise.
