# <a name="edi-migration-utilities"></a>Utilitaires de migration EDI
Utilitaires de migration EDI inclus dans les versions précédentes de [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] ont été utilisés pour migrer des applications EDI de Base à utiliser la solution EDI dans BizTalk Server 2006 R2 ou BizTalk Server 2009. Migration EDI de Base applications directement à BizTalk Server n’est pas pris en charge. Vous devez commencer par migrer les applications EDI de Base BizTalk Server 2006 R2 ou BizTalk Server 2009 à l’aide des utilitaires de migration EDI inclus dans ces versions de [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] , puis utilisez l’outil de Migration de tiers inclus dans BizTalk Server pour migrer le tiers définitions à partir de BizTalk Server 2006 R2 ou BizTalk Server 2009 vers BizTalk Server.  
  
 Pour plus d’informations sur l’outil de Migration de tiers, consultez [migration des artefacts EDI à partir d’une Version précédente de BizTalk Server](http://msdn.microsoft.com/library/b956a97e-03d0-47ea-a2ce-c07a339c0f2c).
