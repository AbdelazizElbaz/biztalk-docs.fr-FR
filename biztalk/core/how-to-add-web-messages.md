# <a name="how-to-add-web-messages"></a>Ajout de messages web
Vous créez des variables de message Web dans la fenêtre Vue Orchestration. Vous devez définir le type du message Web sur un type de message Web.  
  
### <a name="to-add-a-web-message"></a>Pour ajouter un message Web  
  
1.  Avec une orchestration ouverte, dans le **vue** menu, cliquez sur **autres fenêtres**, puis cliquez sur **Vue Orchestration**.  
  
2.  Cliquez sur le **Message** nœud, puis cliquez sur **nouveau Message**.  
  
3.  Message_*x* apparaît sous le **Message** nœud.  
  
4.  Dans le **propriétés** fenêtre, à partir de la **Type de Message** liste déroulante, sélectionnez un type de message Web.  
  
5.  Faites défiler la liste déroulante jusqu'au type de message Web que vous voulez utiliser, puis cliquez dessus pour le sélectionner.  
  
     L’illustration suivante montre le **Vue Orchestration** fenêtre avec Message_1 sélectionnée et la fenêtre Propriétés, sélectionnez un site Web de type de message pour Message_1.
