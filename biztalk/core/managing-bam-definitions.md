# <a name="managing-bam-definitions"></a>Gestion  des définitions BAM
Les définitions BAM font partie de l'infrastructure BAM. Elles définissent les données à suivre et à agréger, ainsi que la vue de l'utilisateur final du processus d'entreprise dans les données de suivi. Les rubriques de cette section indiquent des procédures de gestion des éléments de définitions BAM que sont les activités, les vues, les artefacts et les alertes.
