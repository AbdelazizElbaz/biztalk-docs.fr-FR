# <a name="soap-adapter"></a>Adaptateur SOAP
Les services Web de Microsoft BizTalk Server utilisent l'adaptateur SOAP lors de la réception et de l'envoi de demandes de service Web. L'adaptateur SOAP vous permet de publier des orchestrations en tant que services Web et d'utiliser des services Web externes.
