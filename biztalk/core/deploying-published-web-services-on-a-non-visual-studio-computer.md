# <a name="deploying-published-web-services-on-a-non-visual-studio-computer"></a>Déploiement de services Web publiés sur un ordinateur n'exécutant pas Visual Studio
Pour déployer votre service Web publié sur un ordinateur sur lequel [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)] n'est pas installé, vous devez créer un projet d'installation Web, distribuer le package d'installation Web (fichier .msi), l'installer sur l'ordinateur non-[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)] et configurer le service Web installé.  
  
> [!NOTE]
>  La console Administration de BizTalk Server permet également de créer des applications BizTalk de lot de fichiers MSI contenant les projets de services Web publiés. Ces fichiers MSI permettent de déployer les projets de service Web sur un ordinateur non-[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]. Pour plus d’informations sur la création de fichiers MSI à l’aide de la console d’Administration de BizTalk Server, consultez [comment exporter une Application BizTalk](../core/how-to-export-a-biztalk-application.md).
