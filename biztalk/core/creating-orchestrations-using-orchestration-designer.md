# <a name="creating-orchestrations-using-orchestration-designer"></a>Création d’Orchestrations à l’aide du Concepteur d’Orchestration
Orchestration dans le workflow [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]  
  
 ![Orchestration de BizTalk Server](../core/media/ebiz-dev-busprcsf.gif "ebiz_dev_busprcsf")  
  
 Le Concepteur d'orchestration BizTalk est un utilitaire qui permet de créer facilement des représentations visuelles de vos processus d'entreprise qui sont automatiquement reflétées dans le code sous-jacent et que vous créez dans un module exécutable. Il fournit une grande variété de formes correspondant aux différentes actions que vous pouvez effectuer.  
  
 Cette section présente les concepts, les procédures et des informations de référence vous permettant de comprendre et d'utiliser l'orchestration BizTalk et le Concepteur d'orchestration BizTalk.  
  
 Pour plus d’informations sur l’utilisation des raccourcis clavier pour le Concepteur d’Orchestration, consultez [raccourcis clavier de concepteur d’Orchestration](../core/orchestration-designer-keyboard-shortcuts.md).
