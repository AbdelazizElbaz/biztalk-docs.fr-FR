# <a name="appendix-a-sample-files"></a>Annexe a : exemples de fichiers
L'adaptateur Microsoft BizTalk pour JD Edwards EnterpriseOne prend en charge les structures XML jdeRequest et jdeResponse pour exécuter des fonctions commerciales au sein de JD Edwards EnterpriseOne.  
  
 Les structures XML JD Edwards EnterpriseOne permettent d'effectuer les opérations suivantes :  
  
-   agréger les appels des fonctions commerciales en un seul objet ;  
  
-   utiliser l'API JD Edwards EnterpriseOne ThinNet ;  
  
-   accéder aux fichiers et fonctions commerciales Z.  
  
 Cette section fournit des exemples des structures XML jdeRequest et jdeResponse pour exécuter des fonctions commerciales au sein de JD Edwards EnterpriseOne.
