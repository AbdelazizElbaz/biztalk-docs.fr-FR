# <a name="single-sign-on-applications"></a>Applications à authentification uniques
Du point de vue de la programmation, vous pouvez écrire deux différents types d’applications à l’aide de l’authentification unique : une application de l’authentification unique de traditionnelle qui utilise l’interface de l’authentification unique pour interagir avec les applications distantes et un adaptateur de synchronisation de mot de passe qui utilise le Interface d’assistance de synchronisation de mot de passe pour synchroniser les mots de passe au sein de votre entreprise.
