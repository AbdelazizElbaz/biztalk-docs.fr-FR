# <a name="troubleshooting-jd-edwards-enterpriseone"></a>Dépannage de JD Edwards EnterpriseOne
Cette section décrit les problèmes et messages d'erreur courants que vous pouvez rencontrer dans l'adaptateur BizTalk pour JD Edwards EnterpriseOne, et indique les corrections possibles. Elle décrit également l'utilisation de l'outil Suivi d'événements pour Windows.  
  
 Vous pouvez utiliser les outils de débogage/suivi suivants pour dépanner l'adaptateur :  
  
-   débogage BizTalk Server natif (par exemple, type de suivi sur votre port ou débogueur d'orchestration) ;  
  
-   messages d'erreur envoyés au journal des événements ;  
  
-   capture des messages de l'émetteur, du récepteur et de gestion via BTAJDEEnterpriseOneTrace.cmd et un outil qui convertit les fichiers .etl (tels que tracerpt.exe ou tracedmp.exe).
