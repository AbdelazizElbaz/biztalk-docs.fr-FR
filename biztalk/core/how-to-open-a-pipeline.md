# <a name="how-to-open-a-pipeline"></a>Comment ouvrir un Pipeline
L’ouverture d’un pipeline entraîne l’affichage du Concepteur de pipeline avec le pipeline défini et ses étapes dans la zone de conception. Par ailleurs, la boîte à outils du Concepteur de pipeline et la fenêtre Propriétés s’affichent si elles ne sont pas déjà ouvertes.  
  
### <a name="to-open-a-pipeline"></a>Pour ouvrir un pipeline  
  
1.  Dans l'Explorateur de solutions, double-cliquez sur le pipeline que vous voulez ouvrir.  
  
     — Ou :  
  
2.  Dans l’Explorateur de solutions, sélectionnez le pipeline, puis, dans le **vue** menu, cliquez sur **ouvrir**.
