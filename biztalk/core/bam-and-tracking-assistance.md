# <a name="bam-and-tracking-assistance"></a>Assistance en matière de suivi et d'analyse BAM
Vous pouvez demander une assistance via le portail BAM en cliquant sur le **Assistance** bouton sur la page État de l’activité.  
  
 La requête d'assistance est ensuite placée dans le journal des événements de l'application. La source des requêtes est le service Web BAM et l'ID de l'événement est 1000.  
  
 Le corps du message figure dans le champ Description et contient l'objet de la requête, le texte de description du problème, l'ID d'instance du service BizTalk, l'ID du message, le serveur de la base de données de gestion BizTalk et le nom de la base de données de gestion BizTalk.
