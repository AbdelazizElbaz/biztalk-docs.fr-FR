|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|

## <a name="explanation"></a>Explication  
 L'utilisateur du client doit être membre du nouveau compte Administrateurs de l'authentification unique pour modifier le nom du compte Administrateurs de l'authentification unique.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Vous devez être membre du nouveau compte Administrateurs de l'authentification unique pour modifier le nom du compte Administrateurs de l'authentification unique.
