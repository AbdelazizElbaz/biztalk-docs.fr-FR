# <a name="how-to-create-expressions"></a>Création des expressions
L'Éditeur d'expression BizTalk vous permet de créer des expressions XLANG/s afin de développer les capacités de plusieurs formes du Concepteur d'orchestration. Ces expressions vous permettent de prendre des décisions, d'établir des délais, d'exécuter des appels .NET et de tester des conditions de boucle while.  
  
 Il vous permet de pour affecter des valeurs aux messages ou de parties de message dans la forme assignation du Message et exécuter des appels .NET et de manipuler les valeurs des variables dans le **Expression** forme. Vous pouvez également l’utiliser pour construire des expressions booléennes complexes dans le **boucle** et **décider** formes et pour définir un délai dans le **délai** forme.  
  
### <a name="to-open-biztalk-expression-editor"></a>Pour ouvrir l'Éditeur d'expression BizTalk  
  
1.  Avec le bouton droit une **Expression** forme, un **boucle** mettre en forme un **assignation du Message** forme, un **délai** forme ou une branche d’un **Décider** forme.  
  
2.  Cliquez sur **modifier l’Expression**.  
  
3.  Éditeur d’Expression BizTalk s’affiche.  
  
### <a name="to-create-or-edit-an-expression"></a>Pour créer ou modifier une expression  
  
1.  Tapez ou modifiez l'expression dans la zone de texte de l'Éditeur d'expression BizTalk.  
  
2.  Cliquez sur OK. L'expression est enregistrée et appliquée à la forme sélectionnée.
