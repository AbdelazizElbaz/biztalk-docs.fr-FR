# <a name="how-to-create-multi-part-mappings"></a>Comment créer des mappages à parties multiples
Si vous utilisez plusieurs mappages conjointement, vous devez les associer dans une orchestration pour les tester ensemble. Le **Test Map** fonctionne sur un seul mappage à la fois.  
  
 Pour plus d’informations sur la conception et l’utilisation des orchestrations, consultez [création d’Orchestrations à l’aide de concepteur d’Orchestration](../core/creating-orchestrations-using-orchestration-designer.md).
