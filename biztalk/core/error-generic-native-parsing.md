# <a name="error---generic-native-parsing"></a>Erreur : analyse Native générique
**Code d’erreur**  
  
 btm1042  
  
 **Explication**  
  
 Le fichier de message de l'instance d'entrée natif associé à l'opération Test Map n'a pas été analysé. Une erreur d'analyse générique irrécupérable a été renvoyée.  
  
 **Action de l’utilisateur**  
  
 Si nécessaire, corrigez le schéma source associé au mappage ou le message de l'instance d'entrée natif du fichier indiqué, ou bien les deux. Pour identifier le problème, pensez à utiliser l'Éditeur BizTalk. Le **valider le schéma** et **générer l’Instance** commandes, disponibles lorsque vous cliquez sur un schéma dans l’Explorateur de solutions, sont utiles pour la recherche d’erreurs de schéma.
