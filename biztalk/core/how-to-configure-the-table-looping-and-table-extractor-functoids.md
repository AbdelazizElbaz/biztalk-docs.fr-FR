# <a name="how-to-configure-the-table-looping-and-table-extractor-functoids"></a>Configuration des fonctoids Bouclage de table et Extracteur de table
Des instructions détaillées pour la configuration de la **grille Bouclage de Table** propriété d’un **bouclage de Table** fonctoid sont fournies dans les étapes 9 et 10 de [comment ajouter le bouclage de Table et extracteur de Table Fonctoids à un mappage](../core/how-to-add-table-looping-and-table-extractor-functoids-to-a-map.md). Les étapes 3 à 8 sont également pertinentes pour configurer la grille de table, car elles doivent être exécutées en premier pour alimenter les listes déroulantes disponibles dans chaque cellule de la grille de table.
