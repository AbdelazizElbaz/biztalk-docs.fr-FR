# <a name="how-to-add-or-remove-messages-in-orchestrations"></a>Ajout ou suppression de messages dans des orchestrations
Un message est l'unité de base de communication au sein d'une orchestration. Il se compose d'une ou de plusieurs parties, complétées par des données de contexte décrivant les propriétés du message et son contenu.  
  
### <a name="to-add-a-message"></a>Pour ajouter un message  
  
1.  Dans la fenêtre Vue Orchestration, cliquez sur **Messages** puis cliquez sur **nouveau Message**. Le **Messages** dossier se développe, s’il était réduit et un nouveau message est ajouté.  
  
2.  Attribuez un nom au message.  
  
3.  Spécifiez le type du message.  
  
### <a name="to-remove-a-message"></a>Pour supprimer un message  
  
-   Dans la fenêtre Vue Orchestration, cliquez sur le message que vous souhaitez supprimer, puis cliquez sur **supprimer**.
