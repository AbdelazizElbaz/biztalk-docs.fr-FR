## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|

|Composant|Néant|

|Texte du message|Le compte Administrateurs de l'authentification unique n'est pas valide pour la mise à jour des informations globales.%r<br /><br /> Les administrateurs de l’authentification unique : %1 %r<br /><br /> Comptes non valides : %2 %r<br /><br /> Code d’erreur : %3|  
  
## <a name="explanation"></a>Explication  
 Le compte Administrateurs de l'authentification unique n'est pas valide pour la mise à jour des informations globales.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Ce message d'avertissement contient des informations sur le compte Administrateurs de l'authentification unique et les comptes non valides. Consultez ces informations, apportez les corrections nécessaires, puis effectuez de nouveau la mise à jour.
