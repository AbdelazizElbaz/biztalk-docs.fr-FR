# <a name="error---unreferenced-source-schema"></a>Erreur : schéma Source non référencé
**Code d’erreur**  
  
 btm1051  
  
 **Explication**  
  
 Le schéma source requis ne fait pas partie du projet Microsoft BizTalk Server qui contient le mappage.  
  
 **Action de l’utilisateur**  
  
 Ajoutez le schéma source au projet BizTalk qui contient le mappage à l’aide de la **ajouter un élément existant** commande est disponible sur le **fichier** menu et dans le menu contextuel associé au projet BizTalk dans la Solution Explorateur.
