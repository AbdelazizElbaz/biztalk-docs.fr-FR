# <a name="components-of-the-bam-portal"></a>Composants du portail BAM
Le portail BAM est une page Web ASP.NET divisée en trois zones fonctionnelles distinctes, correspondant chacune à une page spécifique du portail. Ces pages sont décrites dans cette section : page recherche d’activité, page agrégations et page Gestionnaire d’alertes. Vous pouvez les utiliser pour créer des activités ou pour rechercher et modifier des activités existantes.
