# <a name="windows-sharepoint-services-adapter-walkthroughs"></a>Procédures pas à pas de carte de Windows SharePoint Services
Cette section se compose de trois procédures pas à pas. La première procédure décrit la configuration de [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] de manière à envoyer et recevoir des messages à l'aide de l'adaptateur Windows SharePoint Services et du routage basé sur le contenu. La deuxième détaille l'intégration de BizTalk Server à Microsoft Office. La troisième illustre la manière d'accéder aux propriétés de contexte Windows SharePoint Services d'un message entrant au moment de l'exécution, puis de déterminer la destination de ce message sur la base d'une propriété utilisant des ports dynamiques dans une orchestration.
