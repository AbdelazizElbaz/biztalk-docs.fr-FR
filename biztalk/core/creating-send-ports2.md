# <a name="creating-send-ports"></a>Création de Ports d’envoi
Lorsque vous créez un port d'envoi qui fait appel à l'authentification unique (SSO) et à l'application associée, vous ne devez pas entrer les paramètres de nom d'utilisateur et de mot de passe. Vous sélectionnez **Oui** pour **utiliser SSO**, puis vous sélectionnez l’application associée SSO que vous avez créé pour représenter le système TIBCO Rendezvous dans la liste.
