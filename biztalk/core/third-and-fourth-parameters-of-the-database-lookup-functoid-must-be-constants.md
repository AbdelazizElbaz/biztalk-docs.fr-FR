# <a name="error---third-and-fourth-parameters-of-the-database-lookup-functoid-must-be-constants"></a>Erreur : Les troisième et quatrième paramètres du fonctoid Recherche dans la base de données doivent être des valeurs constantes.
**Code d’erreur**  
  
 btm1078  
  
 **Explication**  
  
 Les troisième et quatrième paramètres de la **recherche de base de données** fonctoid sont les noms de table et de colonne. Ces arguments doivent être des valeurs constantes.  
  
 **Action de l’utilisateur**  
  
 Modifiez les paramètres d’entrée du fonctoid dans le **configurer les entrées de fonctoid** boîte de dialogue. Cliquez sur le fonctoid de la grille. Dans la fenêtre Propriétés, sélectionnez le **paramètres d’entrée** propriété et cliquez sur le bouton de sélection (**...** ) bouton. Le **configurer les entrées de fonctoid** boîte de dialogue s’affiche.
