# <a name="application-not-found"></a>Application introuvable

|Texte du message|Application « {0} » introuvable. Vérifiez que l'application existe dans la base de données de configuration BizTalk par défaut|  
  
## <a name="explanation"></a>Explication  
 Cette erreur indique que BizTalk utilise une application qui n'existe pas dans les bases de données BizTalk.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 La procédure suivante permet de configurer une application valide.  
  
#### <a name="to-configure-a-valid-application"></a>Pour configurer une application valide  
  
1.  Cliquez sur **Démarrer**, cliquez sur **tous les programmes**, cliquez sur [!INCLUDE[btsBizTalkServerStartMenuItemui](../includes/btsbiztalkserverstartmenuitemui-md.md)], puis cliquez sur **Administration de BizTalk Server**.  
  
2.  Dans la racine de la Console, développez [!INCLUDE[btsBizTalkServerAdminConsoleui](../includes/btsbiztalkserveradminconsoleui-md.md)], développez **groupe BizTalk**, développez **Applications**.  
  
3.  Assurez-vous que l'application existe. Sinon, sélectionnez-en une valide.
