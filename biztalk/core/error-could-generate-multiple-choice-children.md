# <a name="error---could-generate-multiple-choice-children"></a>Erreur : génération de plusieurs enfants choix possible
**Code d’erreur**  
  
 btm1027  
  
 **Explication**  
  
 Des liens vers le schéma de destination inappropriée permettent à plusieurs nœuds au sein d’un **choix** nœud de groupe.  
  
 **Action de l’utilisateur**  
  
 Corrigez les liens dans le schéma de destination, ainsi qu’un seul nœud dans le **choix** nœud de groupe peut être généré.
