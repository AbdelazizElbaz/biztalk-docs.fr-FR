# <a name="how-to-search-for-activities"></a>Recherche d'activités
Utilisez la recherche d'activités d'entreprise sur le portail Web BAM pour rechercher les données de suivi, également appelées instances, stockées dans la base de données BAM.  
  
 La recherche porte à la fois sur les données actives pour les processus en cours d'exécution et sur les données stockées.  
  
### <a name="to-search-the-bam-database"></a>Pour effectuer des recherches dans la base de données BAM  
  
1.  Dans Internet Explorer, dans le **adresse** , tapez l’URL du portail Web BAM.  
  
2.  À partir de la **Mes vues** volet de navigation, cliquez sur une vue pour développer la vue pour afficher la liste des tâches.  
  
3.  Cliquez sur le **recherche d’activité** tâche pour afficher une liste d’activités...  
  
4.  Cliquez sur une activité pour charger la page Recherche d'activité dans le cadre Contenu.  
  
5.  Dans le **requête** Générateur de choisir les paramètres de requête pour la recherche.  
  
6.  Dans le **sélecteur de colonne** sélectionner les colonnes représentant les éléments de données à retourner dans la recherche.  
  
7.  Cliquez sur le **exécuter la requête** bouton en haut à droite du cadre contenu.  
  
     Les résultats sont affichés dans le **résultats** zone de la page contenu recherche d’activité.  
  
> [!NOTE]
>  Vous pouvez sélectionner des colonnes au maximum 20.  
  
> [!NOTE]
>  Cette version ne prend pas en charge le regroupement d'opérateurs logiques.
