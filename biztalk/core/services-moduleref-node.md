# <a name="services-moduleref-node"></a>Services (nœud ModuleRef)
Le nœud Services d'un fichier de liaison contient des informations spécifiques sur chaque service exporté avec ce fichier.  
  
## <a name="nodes-in-the-services-node"></a>Nœuds du nœud Services  
 Le tableau suivant répertorie les propriétés que vous pouvez définir pour ce nœud d'un fichier de liaison :  
  
|**Nom**|**Type de nœud**|**Type de données**|**Description**|**Restrictions**|**Commentaires**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|[Service](../core/service-services-node.md)|Record|ServiceRef (ComplexType)|Nœud contenant les nœuds qui décrivent ce service.|Facultatif|Valeur par défaut : Aucun|
