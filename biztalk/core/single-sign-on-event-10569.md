# <a name="single-sign-on-event-10569"></a>Single Sign-On : Événement 10569
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10569|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|SSO_WARN_NO_UPDATE_BY_APP_ADMIN|  
|Texte du message|Le compte Administrateurs d'application ne peut pas être modifié par un administrateur d'application.%r<br /><br /> Nom de l’application : %1 %r<br /><br /> Administrateurs d’application : %2 %r<br /><br /> Code d’erreur : %3|  
  
## <a name="explanation"></a>Explication  
 Le compte Administrateurs d'application ne peut pas être modifié par un administrateur d'application.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Vous devez être administrateur d'applications associées à authentification unique ou avoir des privilèges supérieurs pour modifier le compte Administrateurs d'application.
