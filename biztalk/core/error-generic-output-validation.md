# <a name="error---generic-output-validation"></a>Erreur : Validation de sortie générique
**Code d’erreur**  
  
 btm1047  
  
 **Explication**  
  
 Le fichier de message de l'instance de sortie créé par l'opération Test Map n'a pas été validé avec le schéma de destination pour une raison inconnue.  
  
 **Action de l’utilisateur**  
  
 Ouvrir le schéma de destination dans l’Éditeur BizTalk et utiliser le **valider le schéma**, **valider l’Instance**, et **générer l’Instance** commandes, disponibles lorsque vous cliquez sur un schéma dans l’Explorateur de solutions, pour isoler le problème.
