# <a name="single-sign-on-event-10760"></a>Single Sign-On : Événement 10760
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10760|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|ENTSSO_E_NO_EXISTING_VALUE|  
|Texte du message|Le jeu de propriétés contenait une valeur VT_NULL pour indiquer que la valeur de la propriété existante devait être conservée, mais il n'y a pas de valeur existante.|  
  
## <a name="explanation"></a>Explication  
 Il s'agit d'une erreur interne.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Si cette erreur se produit, contactez les services de support technique Microsoft.
