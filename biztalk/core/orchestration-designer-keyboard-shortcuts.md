# <a name="orchestration-designer-keyboard-shortcuts"></a>Raccourcis clavier du Concepteur d'orchestration
La surface de conception du Concepteur d'orchestration permet d'accéder à toutes les options à l'aide du clavier. Cette accessibilité est disponible lorsque vous ouvrez l’aire de conception, vous pouvez faire dans Visual Studio en sélectionnant les options de menu à partir de la **nouveau** option sur le **fichier**menu.  
  
 Les options sont également accessibles depuis la surface de conception, dans laquelle vous pouvez par exemple sélectionner différentes formes et parties de formes en appuyant sur les touches de direction HAUT et BAS. Lorsqu’une forme est sélectionnée sur l’aire de conception, il est mis en surbrillance dans l’aire de conception et dans la fenêtre Vue Orchestration (le cas échéant), lors de la **propriétés** grille affiche également des informations sur la forme sélectionnée.  
  
 Le Concepteur d'orchestration permet l'utilisation d'un certain nombre de raccourcis clavier pour accéder aux options. Ces raccourcis diffèrent selon la partie de la surface de conception que vous êtes en train d'utiliser.  
  
 Vous avez également la possibilité d'utiliser la plupart des raccourcis disponibles dans Visual Studio, comme par exemple lorsque vous utilisez Microsoft IntelliSense dans l'Éditeur d'expression BizTalk.
