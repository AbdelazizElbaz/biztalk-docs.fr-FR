# <a name="request-assistance"></a>Demande d’Assistance
Lorsque vous avez besoin de l’assistance de votre administrateur système ou une autre personne de prise en charge avec votre [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] mise en œuvre, vous pouvez utiliser la **Assistance** situé en bas de la **résultats** détail page pour placer une demande d’assistance.  
  
## <a name="how-to-request-assistance"></a>Procédure de demande d'assistance  
 En cliquant sur le **Assistance** bouton ouvre un **Web demander de l’Assistance technique** boîte de dialogue de page. Pour demander une assistance, vous entrez un objet décrivant votre problème dans le **sujet** zone de texte. Dans le **Description du problème** zone de texte vous décrivez le problème que vous rencontrez avec les détails de votre administrateur système doit résoudre le problème. Lorsque vous êtes satisfait de la description du problème, cliquez sur le **envoyer le rapport** bouton pour envoyer la demande d’assistance. Votre requête est alors placée dans le journal des événements de l'application de serveur et votre administrateur système peut l'extraire en utilisant le suivi des instances de service et de messages à partir de la page Présentation du groupe.
