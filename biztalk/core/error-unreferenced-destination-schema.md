# <a name="error---unreferenced-destination-schema"></a>Erreur : schéma de Destination non référencé
**Code d’erreur**  
  
 btm1052  
  
 **Explication**  
  
 Le schéma de destination requis ne fait pas partie du projet Microsoft BizTalk Server. **Action de l’utilisateur**  
  
 Ajoutez le schéma de destination au projet BizTalk qui contient le mappage à l’aide de la **ajouter un élément existant** commande est disponible sur le **fichier** menu et dans le menu contextuel du projet BizTalk dans Explorateur de solutions.
