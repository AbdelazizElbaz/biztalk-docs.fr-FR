# <a name="how-biztalk-server-sends-edi-messages"></a>Envoi des messages EDI par BizTalk Server
Lorsque [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] envoie un message EDI, il recherche l'accord, identifie le schéma, valide le message, envoie un accusé de réception (le cas échéant) et sérialise le lot EDI. Ce traitement est effectué par l'assembleur EDI dans le pipeline d'envoi EDI.
