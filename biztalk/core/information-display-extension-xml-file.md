# <a name="information---display-extension-xml-file"></a>Information : affichage fichier XML avec extensions
**Code d’erreur**  
  
 btm1057  
  
 **Explication**  
  
 L'élément XML associé à l'objet d'extension est stocké dans le fichier indiqué.  
  
 **Action de l’utilisateur**  
  
 Cliquez sur le lien correspondant pour ouvrir le fichier XML avec extensions et obtenir le préfixe du mappage du nom de l'assembly.
