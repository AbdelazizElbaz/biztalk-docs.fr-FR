# <a name="reset-for-control-number-failed"></a>Échec de la réinitialisation du numéro de contrôle
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]|  
|Version du produit|[!INCLUDE[btsEDIVersion](../includes/btsediversion-md.md)]|  
|ID d'événement|-|  
|Source de l'événement|[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] EDI|  
|Composant|Moteur EDI|  
|Nom symbolique|EdiControlNumberResetFailed|  
|Texte du message|Réinitialiser pour {0} a échoué.|  
  
 **{0}** : Définir le contrôle transaction numéro de contrôle du groupe / numéro / numéro de contrôle d’échange  
  
## <a name="explanation"></a>Explication  
 Cet événement d'erreur/d'avertissement/d'informations indique que BizTalk Server n'a pas réussi à réinitialiser le numéro de contrôle.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Pour résoudre cette erreur, rouvrez l'interface utilisateur, puis réessayez.
