# <a name="creating-schemas-using-biztalk-flat-file-schema-wizard"></a>Création de schémas à l'aide de l'Assistant Schéma de fichier plat BizTalk
L'Assistant Schéma de fichier plat est conçu pour simplifier la création des schémas de fichier plat grâce aux fonctionnalités suivantes :  
  
-   la possibilité d'utiliser des instances de fichier plat comme entrée ;  
  
-   une surface de conception visuelle pour le traitement des schémas de fichier plat délimité et positionnel ;  
  
-   un processus interactif basé sur un Assistant permettant d'ajouter des éléments au schéma et de définir des annotations associées au fichier plat.  
  
 Cette section présente l'Assistant Schéma de fichier plat, la procédure de lancement de l'Assistant, des observations relatives à l'utilisation de l'Assistant et une procédure pas à pas vous indiquant comment créer des schémas de fichier plat à partir d'une instance de document.
