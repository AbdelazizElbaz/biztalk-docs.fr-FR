# <a name="access-control-to-business-information"></a>Contrôle d’accès aux informations de l’entreprise
Seuls les administrateurs BizTalk Server peuvent accéder aux données de suivi, telles que les propriétés de message et les données de corps de message, dans la base de données des suivis BizTalk. Les opérateurs BizTalk Server ont accès aux données de flux de messages dans la base de données des suivis BizTalk.
