# <a name="single-sign-on-event-10662"></a>Single Sign-On : Événement 10662
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10662|  
|Source de l'événement|ENTSSO|  
|Composant|N\A|  
|Nom symbolique|SSO_INFO_WINDOWS_PASSWORD_CHANGE_RECEIVED|  
|Texte du message|Une modification de mot de passe Windows a été reçue du service PCNS.%r<br /><br /> ID de suivi : %1 %r<br /><br /> Compte Windows : %2 %r<br /><br /> Données supplémentaires : %3 %r<br /><br /> Utilisateur client : %4|  
  
## <a name="explanation"></a>Explication  
 Cet événement de type information indique qu'une modification de mot de passe Windows a été reçue du service PCNS (Password Change Notification Service).  
  
## <a name="user-action"></a>Action de l'utilisateur  
  
-   Aucune action utilisateur n’est nécessaire.
