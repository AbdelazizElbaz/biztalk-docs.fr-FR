# <a name="bam-wcf-interceptor"></a>Intercepteur WCF BAM
L'intercepteur WCF BAM fournit une prise en charge complète des données de suivi au sein de vos services WCF.  
  
 L'intercepteur WCF BAM permet d'effectuer les actions suivantes :  
  
-   transférer des données d'application de flux de travail vers l'analyse BAM de façon transparente, sans avoir à recompiler votre application WCF ;  
  
-   assurer le suivi des informations des paramètres et des messages dans les appels de service WCF ;  
  
-   assurer le suivi des informations des messages dans les appels de service WCF ;  
  
-   participer aux transactions transmises du client ou initiées en interne par le service d'exécution WCF pour les appels de service traités.
