# <a name="nodes-that-simplify-schema-creation"></a>Nœuds simplifiant la création de schémas
Les nœuds qui simplifient la création de schémas appartiennent à une catégorie de deux types de nœuds qui échappent au classement dans les autres catégories et simplifient la création de schémas. Le premier nœud, le **schéma** , correspond à la **schéma** élément dans le langage XML Schema definition (XSD), qui est l’élément qui sert de l’élément le plus éloigné dans tous les schémas XSD.  
  
 L'autre nœud de simplification joue un rôle plus significatif à cet égard. Le **groupe d’attributs** nœud correspond directement à la **attributeGroup** élément dans le schéma XSD et fournit un mécanisme permettant de définir un groupe d’attributs associés, qui ont toujours lieu ensemble, dans un seul emplacement, puis en utilisant cette définition de groupe dans plusieurs emplacements du schéma. Le reste de cette section propose des informations supplémentaires concernant ces nœuds.
