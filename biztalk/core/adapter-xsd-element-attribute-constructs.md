# <a name="adapter-xsd-element-attribute-constructs"></a>Constructions élément-attribut XSD l’adaptateur
Les tableaux suivants répertorient les constructions élément-attribut XSD prises en charge par l'infrastructure d'adaptateurs. L'axe vertical se compose des types d'éléments pris en charge, tandis que l'axe horizontal indique les types d'attributs valides pour chaque élément.  
  
> [!NOTE]
>  Les éléments et attributs possèdent des noms uniques. Tout doublon entraîne une erreur de compilation.  
  
> [!NOTE]
>  L'attribut fixed avec la facette Énumération n'est pas pris en charge.  
  
> [!NOTE]
>  Le fichier XSD de configuration doit avoir un élément appelé **uri** de type xsd : String.
