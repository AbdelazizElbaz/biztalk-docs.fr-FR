# <a name="how-to-manage-other-visual-studio-windows"></a>Comment gérer les autres fenêtres Visual Studio
Dans l'Éditeur BizTalk, certaines opérations utilisent les autres fenêtres Microsoft [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)] que sont la fenêtre Sortie et la fenêtre Liste des tâches. Ces fenêtres sont utilisées pour afficher la sortie d'opérations telles que celles consistant à générer des exemples de messages d'instance et à valider des schémas et des instances. Cette rubrique contient des instructions détaillées concernant la gestion de ces fenêtres.  
  
### <a name="to-open-the-output-window"></a>Pour ouvrir la fenêtre Sortie  
  
-   Dans [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)], cliquez sur le **vue** menu, puis sur **sortie**.  
  
    > [!NOTE]
    >  Si vous voulez afficher le fichier correspondant à une ligne de la fenêtre Sortie, double-cliquez sur cette ligne.  
  
### <a name="to-open-the-task-list-window"></a>Pour ouvrir la fenêtre Liste des tâches  
  
-   Dans [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)], cliquez sur le **vue** menu, puis sur **liste des tâches**.  
  
    > [!NOTE]
    >  Si vous souhaitez afficher le nœud d'un schéma BizTalk qui correspond à une ligne de tâche dans la fenêtre Liste des tâches, double-cliquez sur cette ligne de tâche.  
  
 Par défaut, la fenêtre Sortie et la fenêtre Liste des tâches partagent la même section du shell [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]. Si les deux fenêtres sont ouvertes en même temps, elles sont affichées avec des onglets nommés qui vous permettent de basculer entre elles.
