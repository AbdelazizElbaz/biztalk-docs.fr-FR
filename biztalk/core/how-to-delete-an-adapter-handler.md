# <a name="how-to-delete-an-adapter-handler"></a>Comment supprimer un gestionnaire d’adaptateur
Vous pouvez supprimer un gestionnaire d'adaptateur d'envoi ou de réception à l'aide de la console Administration de BizTalk Server.  
  
 Avant de supprimer un gestionnaire d'adaptateur, vous devez supprimer tous les emplacements de réception ou les ports d'envoi auxquels il est associé.  
  
> [!NOTE]
>  Pour supprimer un gestionnaire d'adaptateur, vous devez être membre du groupe Administration SSO.  
  
### <a name="to-delete-an-adapter-handler"></a>Pour supprimer un gestionnaire d'adaptateur  
  
1.  Dans la Console Administration de BizTalk Server, développez **Administration de BizTalk Server**, développez **groupe BizTalk**, développez **paramètres de plateforme**, puis développez  **Adaptateurs**.  
  
2.  Dans la liste développée des adaptateurs, sélectionnez l'adaptateur dont vous souhaitez supprimer le gestionnaire.  
  
3.  Cliquez sur le Gestionnaire d’adaptateur que vous souhaitez supprimer, puis cliquez sur **supprimer**.  
  
4.  Cliquez sur **Oui** pour confirmer que vous souhaitez supprimer ce gestionnaire d’adaptateur.  
  
5.  Cliquez sur **OK**.
