# <a name="single-sign-on-event-10774"></a>Single Sign-On : Événement 10774
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10774|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|ENTSSO_E_SSO_NOT_IN_NEW_SSO_ADMIN|  
|Texte du message|Le compte du service SSO ne correspond pas à un membre du compte Administrateurs SSO spécifié.|  
  
## <a name="explanation"></a>Explication  
 Vous devez être membre du compte Administrateurs SSO pour pouvoir y apporter des modifications.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Faites appel à votre administrateur système afin de contacter un membre du compte Administrateurs SSO.
