# <a name="single-sign-on-event-10756"></a>Single Sign-On : Événement 10756
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10756|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|ENTSSO_E_NO_APP|  
|Texte du message|L'application n'existe pas.|  
  
## <a name="explanation"></a>Explication  
 Le nom de l'application a été donné de façon incorrecte ou l'application a été supprimée.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Vérifiez le nom de l'application.
