# <a name="creating-a-net-application-to-test-a-published-web-service"></a>Création d’une Application .NET pour tester un Service Web publié
Pour tester un service Web publié, vous pouvez créer une application cliente Web ASP.NET qui va utiliser celui-ci. La collection d'aide de [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)] inclut une procédure de création d'application cliente Web ASP.NET qui permet de tester un service Web publié.  
  
 La procédure crée une application cliente Web ASP.NET, ajoute une référence Web et fournit un exemple de code qui vous indique comment accéder à votre service Web publié. Elle vous guide dans l'exécution de l'application en mode de débogage.  
  
 Pour plus d’informations et des procédures sur la création d’un projet de client de service Web XML, consultez « Procédure pas à pas : accès à un XML Web Service à l’aide de Visual Basic ou Visual c# » dans le [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)] Collection d’aide à [http://go.microsoft.com/fwlink/?LinkId=62263](http://go.microsoft.com/fwlink/?LinkId=62263). Vous pouvez remplacer le service Web utilisé dans cet exemple, TempConvert1, avec un BizTalk service Web publié.
