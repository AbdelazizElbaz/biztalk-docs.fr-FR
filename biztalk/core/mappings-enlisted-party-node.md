# <a name="mappings-enlisted-party-node"></a>Mappages (nœud Tiers inscrit)
Le nœud Mappages d’un fichier de liaison est le nœud parent de tous les nœuds Mappage qui décrivent le mappage entre un port de tiers et une opération de type de ports de rôle.  
  
## <a name="nodes-in-the-mappings-node"></a>Nœuds du nœud Mappages  
 Le tableau suivant répertorie les propriétés que vous pouvez définir pour ce nœud d'un fichier de liaison :  
  
|**Nom**|**Type de nœud**|**Type de données**|**Description**|**Restrictions**|**Commentaires**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|[Mappage](../core/mapping-mappings-node.md)|Record|EnlistedPartyMapping (ComplexType)|Spécifie le mappage entre un port de tiers et une opération de type de ports de rôle.|Facultatif|Valeur par défaut : Aucun|
