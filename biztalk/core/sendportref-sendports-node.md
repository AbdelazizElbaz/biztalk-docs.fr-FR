# <a name="sendportref-sendports-node"></a>SendPortRef (Nœud SendPorts)
Le nœud SendPortRef du nœud SendPorts d'un fichier de liaison spécifie le nom d'un port d'envoi référencé par une liste de distribution.  
  
> [!NOTE]
>  Une liste de distribution est appelée groupe de ports d'envoi dans l'Administrateur BizTalk.  
  
## <a name="nodes-in-the-sendportref-node"></a>Nœuds du nœud SendPortRef  
 Le tableau suivant répertorie les propriétés que vous pouvez définir pour ce nœud d'un fichier de liaison :  
  
|**Nom**|**Type de nœud**|**Type de données**|**Description**|**Restrictions**|**Commentaires**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|Nom|Attribut|xs:string|Spécifie le nom du port d'envoi référencé par la liste de distribution.|Facultatif|Valeur par défaut : vide|
