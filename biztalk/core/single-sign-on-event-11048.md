## <a name="explanation"></a>Explication  
 Ce problème est probablement dû à une erreur d'installation.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Pour résoudre le problème, réinstallez l'authentification unique ou installez SSOCSTX.dll dans une application de la bibliothèque COM+.
