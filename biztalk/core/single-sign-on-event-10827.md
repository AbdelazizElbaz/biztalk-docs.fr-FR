# <a name="single-sign-on-event-10827"></a>Single Sign-On : Événement 10827
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10827|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|ENTSSO_E_PSADMIN_INVALID_GROUP_ADAPTER_TYPE|  
|Texte du message|Type d'application non valide pour l'adaptateur de groupe. « Adaptateur de groupe de synchronisation de mot de passe » est le type d'application valide.|  
  
## <a name="explanation"></a>Explication  
 Type d'application non valide pour l'adaptateur de groupe. « Adaptateur de groupe de synchronisation de mot de passe » est le type d'application valide.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Utilisez l'un des types d'applications valides.
