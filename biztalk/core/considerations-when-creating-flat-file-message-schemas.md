# <a name="considerations-when-creating-flat-file-message-schemas"></a>Considérations relatives à la création de plat fichier schémas de Message
Il faut tenir compte d'un certain nombre de considérations lorsque vous utilisez des schémas de message de fichier plat. Certaines concernent l'ensemble des schémas de fichier plat tandis que d'autres ont trait spécifiquement aux enregistrements positionnels, aux enregistrements délimités, aux champs positionnels ou aux champs délimités. Il existe également des considérations sur la façon d'interpréter des caractères considérés généralement comme spéciaux en tant que données normales. Cette section contient des informations sur ces points :
