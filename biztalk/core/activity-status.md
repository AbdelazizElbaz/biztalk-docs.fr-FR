# <a name="activity-status"></a>État de l'activité
La zone État de l'activité des détails des résultats vous permet d'afficher sur une seule page tous les détails de l'élément des résultats.  
  
 Elle contient deux sous-zones : Étapes majeures et Données.  
  
## <a name="milestones"></a>Milestones  
 La zone Étapes majeures affiche toutes les étapes majeures définies pour l'activité sélectionnée. Les données concernant les étapes majeures comportent la date et l'heure auxquelles l'étape majeure a été atteinte. Certaines étapes majeures, comme les étapes majeures de commandes approuvées ou de commandes refusées, s'excluent mutuellement. Dans ce cas, les données de date et d'heure d'une des étapes majeures sont affichées, tandis que la seconde ne sera pas spécifiée, indiquant qu'il n'existe aucune donnée pour cette étape majeure.  
  
## <a name="data"></a>Données  
 La zone Données affiche tous les éléments de données définis pour l'activité. Les éléments de données sont affichés par ordre alphabétique.
