# <a name="resend-status-details-report"></a>Rapport Détails de l'état des renvois
Ce rapport d'état présente des informations sur les tentatives de reprise effectuées quand les propriétés Tiers en tant que récepteur des messages AS2 d'un tiers sont configurées pour renvoyer le message AS2 si un MDN n'est pas reçu.  
  
 Afficher ce rapport en cliquant sur un message dans le rapport d’état AS2/MDN, puis en cliquant sur **afficher l’état de la messagerie fiable**. La page Détails de l'état des renvois affiche des informations sur les tentatives de renvoi effectuées pour ce message.  
  
 Ce rapport est uniquement disponible si vous avez sélectionné **renvoyer le message AS2 si MDN non reçu** dans le tiers en tant que propriétés du récepteur des messages AS2 pour le tiers associé.  
  
 Le rapport présente des informations sur les tentatives effectuées pour le message dans les pages suivantes :  
  
|Radiomessagerie|Données affichées|  
|----------|--------------------|  
|**Général**|Index de la tentative d'envoi en cours ainsi que la configuration de renvoi.|  
|**Historique des renvois**|Historique des tentatives d'envoi du message.|
