# <a name="error---native-serialization"></a>Erreur : sérialisation Native
**Code d’erreur**  
  
 btm1048  
  
 **Explication**  
  
 Erreur de sérialisation lors de la tentative de conversion du message de l'instance de sortie XML généré par le mappage au format natif indiqué par le schéma de destination.  
  
 **Action de l’utilisateur**  
  
 En fonction de l'erreur indiquée, corrigez les transformations spécifiées dans le mappage ou le schéma de destination, ou bien les deux. Il peut être utile ouvrir le schéma de destination dans l’Éditeur BizTalk et utiliser le **valider le schéma**, **valider l’Instance**, et **générer l’Instance** commandes disponibles lorsque vous cliquez sur un schéma dans l’Explorateur de solutions.
