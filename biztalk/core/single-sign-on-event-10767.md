# <a name="single-sign-on-event-10767"></a>Single Sign-On : Événement 10767
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10767|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|ENTSSO_E_NO_SSO_SERVER|  
|Texte du message|Impossible de contacter le serveur SSO « %1 ». Vérifiez que SSO est configuré et que le service SSO est exécuté sur ce serveur.|  
  
## <a name="explanation"></a>Explication  
 Le client ENTSSO ne parvient pas à contacter le serveur ENTSSO.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Vérifiez la connectivité réseau et assurez-vous d'avoir bien orthographié le nom du serveur.
