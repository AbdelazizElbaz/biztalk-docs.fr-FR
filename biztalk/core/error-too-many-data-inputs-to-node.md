# <a name="error---too-many-data-inputs-to-node"></a>Erreur : trop d’entrées de données pour ce nœud
**Code d’erreur**  
  
 btm1005  
  
 **Explication**  
  
 Il existe un plus grand nombre de liens connectés au nœud indiqué dans le schéma de destination que le nombre de liens d’entrée le **bouclage** fonctoid qui est connecté à un nœud d’ancêtre du nœud indiqué. Ces deux nombres doivent être identiques.  
  
 **Action de l’utilisateur**  
  
 Reprendre le nombre de liens connecté au nœud indiqué et en le **bouclage** connectés au nœud de niveau supérieur afin qu’elles correspondent.
