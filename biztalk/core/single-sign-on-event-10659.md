# <a name="single-sign-on-event-10659"></a>Single Sign-On : Événement 10659
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10659|  
|Source de l'événement|ENTSSO|  
|Composant|N\A|  
|Nom symbolique|SSO_INFO_PASSWORD_SYNC_WINDOWS_STARTING|  
|Texte du message|La synchronisation des mots de passe pour Windows (à partir de PCNS) est en cours de démarrage.%r|  
  
## <a name="explanation"></a>Explication  
 Cet événement d'informations indique que la synchronisation des mots de passe pour Windows est en cours de démarrage.

-   Aucune action utilisateur n’est nécessaire.
