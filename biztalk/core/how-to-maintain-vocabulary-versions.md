# <a name="how-to-maintain-vocabulary-versions"></a>Comment mettre à jour les Versions de vocabulaire
Lorsque vous avez ajouté des définitions de vocabulaire à une version de votre vocabulaire, vous pouvez enregistrer la version dans le magasin de règles en vue de la développer, ou la publier afin de fournir aux utilisateurs un ensemble de termes bien définis et immuables à ajouter aux règles pendant qu'ils développent leurs stratégies. Notez que les règles existantes pointeront toujours vers les anciennes versions.  
  
 Cette rubrique contient les procédures des tâches suivantes :  
  
-   Pour enregistrer une version de vocabulaire  
  
-   Pour publier une version de vocabulaire  
  
-   Pour créer une version mise à jour d'un vocabulaire  
  
-   Pour créer une version vide d'un vocabulaire  
  
### <a name="to-save-a-vocabulary-version"></a>Pour enregistrer une version de vocabulaire  
  
-   Avec le bouton droit de la version, puis cliquez sur **enregistrer**.  
  
### <a name="to-publish-a-vocabulary-version"></a>Pour publier une version de vocabulaire  
  
-   Avec le bouton droit de la version, puis cliquez sur **publier**.  
  
### <a name="to-create-an-updated-version-of-a-vocabulary"></a>Pour créer une version mise à jour d'un vocabulaire  
  
1.  Cliquez sur une version existante, puis cliquez sur **copie**.  
  
2.  Cliquez sur le dossier de vocabulaire, puis cliquez sur **coller**.  
  
     Une nouvelle version est alors créée. Ses éléments sont les mêmes que ceux de la version copiée.  
  
### <a name="to-create-an-empty-new-version-of-a-vocabulary"></a>Pour créer une version vide d'un vocabulaire  
  
-   Cliquez sur le dossier de vocabulaire, puis cliquez sur **ajouter une nouvelle Version**.  
  
    > [!NOTE]
    >  Vous ne pouvez utiliser que des définitions de vocabulaire issues de vocabulaires publiés.
