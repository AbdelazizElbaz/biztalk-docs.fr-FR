# <a name="wcf-customisolated-adapter"></a>Adaptateur WCF-CustomIsolated
L’adaptateur WCF-CustomIsolated est un adaptateur isolé pour permettre la configuration des emplacements de réception via le transport HTTP avec les paramètres de Windows Communication Foundation (WCF) liaison et le comportement qui sont appliquent aux [!INCLUDE[btsCoName](../includes/btsconame-md.md)] [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]. Vous pouvez utiliser l’adaptateur WCF-CustomIsolated pour les adaptateurs WCF standard ne prenant pas en charge les scénarios. L'adaptateur WCF-CustomIsolated permet également d'utiliser les points d'extensibilité WCF dans [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)].  
  
 L'adaptateur WCF-CustomIsolated est utilisé pour l'emplacement de réception exécuté dans un hôte isolé. Pour utiliser l'adaptateur WCF-CustomIsolated, vous devez créer, dans Microsoft Internet Information Services (IIS), le répertoire virtuel correspondant à cet emplacement de réception, par l'intermédiaire de l'Assistant Publication de services WCF BizTalk.
