# <a name="distributionlistref-port-node"></a>DistributionListRef (Nœud Port)
Le nœud DistributionListRef du nœud Port d'un fichier de liaison contient des informations sur une liste de distribution référencée par un service.  
  
> [!NOTE]
>  Les listes de distribution sont appelées « groupes de ports d'envoi » dans la console Administration de BizTalk Server.  
  
## <a name="nodes-in-the-distributionlistref-node"></a>Nœuds du nœud DistributionListRef  
 Le tableau suivant répertorie les propriétés que vous pouvez définir pour ce nœud d'un fichier de liaison :  
  
|**Nom**|**Type de nœud**|**Type de données**|**Description**|**Restrictions**|**Commentaires**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|Nom|Attribut|xs:string|Spécifie le nom d'une liste de distribution référencée par un service.|Facultatif|Valeur par défaut : vide|
