# <a name="single-sign-on-event-10658"></a>Single Sign-On : Événement 10658
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10658|  
|Source de l'événement|ENTSSO|  
|Composant|N\A|  
|Nom symbolique|SSO_ERROR_PASSWORD_SYNC_ADAPTERS_START_FAILED|  
|Texte du message|Échec du démarrage de la synchronisation des mots de passe pour les cartes réseau.%r<br /><br /> Code d’erreur : %1|  
  
## <a name="explanation"></a>Explication  
 Cet événement de type erreur indique que la synchronisation des mots de passe pour les cartes réseau n'a pas démarré.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Pour résoudre cette erreur, procédez de l'une des manières suivantes :  
  
-   Consultez les journaux des événements de l'application et du système pour connaître les événements associés.  
  
-   Vérifiez les propriétés de la configuration de la carte.
