# <a name="defining-a-bam-view"></a>Définition d'une vue BAM
Vous pouvez parfois souhaiter que tous les utilisateurs des activités ne puissent pas voir l'intégralité de vos données d'entreprise. Vous pouvez créer des vues BAM différentes présentant différents types d'informations.  
  
> [!IMPORTANT]
>  Tous les noms d'alias, de durée, de groupe, de dimension et de mesure doivent être uniques, car ces éléments seront intégrés dans la même vue.  
  
> [!NOTE]
>  Pour définir une vue BAM, vous devez d'abord avoir créé au moins une activité BAM. Pour plus d’informations sur la création d’activités, consultez [la définition d’une activité d’entreprise](../core/how-to-define-a-business-activity.md).  
  
> [!NOTE]
>  Les vues BAM apparaissent dans le classeur Excel en tant que feuilles de calcul. Vous pouvez créer plusieurs vues pour une activité BAM. Utilisez les onglets de feuille de calcul en bas de la fenêtre de l'application Excel pour sélectionner une vue BAM afin de la modifier.
