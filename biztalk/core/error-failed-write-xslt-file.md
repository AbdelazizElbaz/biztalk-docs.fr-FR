# <a name="error---failed-write-xslt-file"></a>Erreur : Échec de l’écriture du fichier XSLT
**Code d’erreur**  
  
 btm1056  
  
 **Explication**  
  
 Erreur lors de la tentative d'écriture de l'élément XSLT généré par ce mappage dans le fichier indiqué.  
  
 **Action de l’utilisateur**  
  
 En fonction de l'erreur indiquée, définissez la mesure adéquate pour corriger le problème. Par exemple, libérez de l'espace supplémentaire sur votre disque dur.
