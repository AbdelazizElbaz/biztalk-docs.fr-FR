# <a name="single-sign-on-event-10785"></a>Single Sign-On : Événement 10785
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10785|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|ENTSSO_E_DB_ACCESS|  
|Texte du message|Une erreur s'est produite lors de la tentative d'accès à la base de données SSO.|  
  
## <a name="explanation"></a>Explication  
 Il se peut que la base de données ENTSSO soit hors ligne.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Demandez à votre administrateur système de vérifier le journal des événements sur le serveur ENTSSO.
