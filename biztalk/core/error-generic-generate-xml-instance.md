# <a name="error---generic-generate-xml-instance"></a>Erreur : génération d’Instance XML générique
**Code d’erreur**  
  
 btm1038  
  
 **Explication**  
  
 Le Mappeur BizTalk n'a pas été en mesure de générer un fichier de message de l'instance XML pour le schéma source du mappage. Par conséquent, un problème existe au niveau du schéma source associé au mappage.  
  
 **Action de l’utilisateur**  
  
 Utilisez l'Éditeur BizTalk pour ouvrir le schéma source associé au mappage et rechercher les problèmes éventuels liés au schéma source. En outre, le **valider le schéma** et **générer l’Instance** commandes, disponibles lorsque vous cliquez sur un schéma dans l’Explorateur de solutions, sont utiles pour la recherche d’erreurs de schéma.
