# <a name="single-sign-on-event-10562"></a>Single Sign-On : Événement 10562
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10562|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|SSO_INFO_REENCRYPT_STARTING|  
|Texte du message|Le nouveau cryptage de la base de données SSO est en cours.|  
  
## <a name="explanation"></a>Explication  
 Ce message est fourni uniquement. Le secret a été modifié et le système SSO commence à rechiffrer la base de données avec le nouveau secret.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Aucune action de l'utilisateur n'est requise.
