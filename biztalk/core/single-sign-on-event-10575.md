# <a name="single-sign-on-event-10575"></a>Single Sign-On : Événement 10575
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10575|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|SSO_INFO_CHANGED_SECRET_SERVER|  
|Texte du message|Nom du serveur de secret mis à jour.%r<br /><br /> Nouveau serveur de secret principal : %1 %r<br /><br /> Ancien serveur de Secret : %2 %r<br /><br /> ID de suivi : %3 %r<br /><br /> Ordinateur client : %4 %r<br /><br /> Utilisateur client : %5|  
  
## <a name="explanation"></a>Explication  
 Ce message d'information peut s'avérer utile pour effectuer le suivi d'événements majeurs liés à la sécurité qui surviennent dans le système SSO. Il indique que le nom du serveur de secret a été mis à jour.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Aucune action de l'utilisateur n'est requise.
