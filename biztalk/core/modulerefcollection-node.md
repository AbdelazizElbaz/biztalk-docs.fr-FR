# <a name="modulerefcollection-node"></a>Nœud ModuleRefCollection
La section ModuleRefCollection  d'un fichier de liaison est le nœud parent de tous les nœuds ModuleRef qui contiennent des informations spécifiques concernant les assemblys .NET exportés avec le fichier de liaison.  
  
## <a name="entries-in-the-modulerefcollection-section"></a>Entrées dans la section ModuleRefCollection.  
 Le tableau suivant répertorie les propriétés que vous pouvez définir pour les nœuds de cette section d'un fichier de liaison :  
  
|**Nom**|**Type de nœud**|**Type de données**|**Description**|**Restrictions**|**Commentaires**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|[ModuleRef](../core/moduleref-modulerefcollection-node.md)|Record|ModuleRef (ComplexType)|Nœud conteneur pour un module d’assembly .NET exporté avec le fichier de liaison.|Facultatif|Valeur par défaut : Aucun|
