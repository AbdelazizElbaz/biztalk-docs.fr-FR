# <a name="error---input-count-for-xslt-call-template-scripting-functoid-not-valid"></a>Erreur : nombre d’entrées pour XSLT appel fonctoid script du modèle non valide
**Code d’erreur**  
  
 btm1076  
  
 **Explication**  
  
 Le nombre de paramètres d’entrée au **script** fonctoid, configuré pour utiliser un modèle d’appel XSLT inline ne correspond pas au nombre de paramètres attendu par ce modèle.  
  
 **Action de l’utilisateur**  
  
 Vérifiez que le nombre de paramètres d’entrée à ce **script** fonctoid correspond au nombre de paramètres attendu par le modèle d’appel XSLT configuré pour ce inline **script** fonctoid.
