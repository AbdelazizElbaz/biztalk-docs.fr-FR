# <a name="single-sign-on-event-10579"></a>Single Sign-On : Événement 10579
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10579|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|SSO_INFO_CHANGED_APP_ADMIN_GROUP|  
|Texte du message|Compte Administrateurs d'application mis à jour.%r<br /><br /> Nouveaux administrateurs d’Application : %1 %r<br /><br /> Anciens administrateurs d’Application : %2 %r<br /><br /> ID de suivi : %3 %r<br /><br /> L’utilisateur client : %4 %r<br /><br /> Nom de l’application : %5|  
  
## <a name="explanation"></a>Explication  
 Ce message d'information peut s'avérer utile pour effectuer le suivi d'événements majeurs liés à la sécurité qui surviennent dans le système SSO. Il indique que le compte Administrateurs d'application a été mis à jour.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Aucune action de l'utilisateur n'est requise.
