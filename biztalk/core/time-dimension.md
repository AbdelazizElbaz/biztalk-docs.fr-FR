# <a name="time-dimension"></a>Dimension de temps
La dimension de temps permet de créer des agrégations temporelles. Par exemple, une dimension de temps peut être utilisée pour créer le tableau suivant :  
  
|Année|Mois|Compter|  
|----------|-----------|-----------|  
|2003|Janvier|120|  
||February|230|  
||Mars|350|  
||avril|280|  
  
 La dimension de temps peut être combinée avec n’importe quelle autre dimension. Par exemple, vous pouvez utiliser la dimension de temps sur les lignes et la dimension de données sur les colonnes pour créer le tableau suivant :  
  
|||Raquettes de tennis|Ballons de football|  
|------|------|---------------------|------------------|  
||Janvier|50|70|  
||February|120|110|  
||Mars|300|50|  
||avril|220|60|
