# <a name="consuming-web-services"></a>Utilisation des services web
L'utilisation de services Web vous permet d'ajouter des services Web existants à vos processus d'entreprise. Vous pouvez agréger plusieurs services Web au sein d'une seule orchestration.  
  
 Vous pouvez utiliser (appeler) un service Web depuis une orchestration à l'aide de ports Web. Pour utiliser un service Web depuis une orchestration, vous devez au préalable créer un port Web puis construire des messages Web.  
  
 Vous pouvez utiliser des en-têtes SOAP avec le service Web utilisé, modifier l'URI de celui-ci et définir de manière dynamique l'URI d'un service Web utilisé.  
  
 Pour plus d’informations sur la configuration d’un protocole SOAP Gestionnaire de réception, consultez [comment configurer un gestionnaire de réception SOAP](../core/how-to-configure-a-soap-receive-handler.md). Pour plus d’informations sur la configuration d’un protocole SOAP emplacement de réception, consultez [comment configurer un emplacement de réception SOAP](../core/how-to-configure-a-soap-receive-location.md).
