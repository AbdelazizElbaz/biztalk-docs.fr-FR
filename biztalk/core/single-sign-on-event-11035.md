# <a name="single-sign-on-event-11035"></a>Single Sign-On : Événement 11035
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|11035|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|SSO_INFO_PS_WIN_CHANGE_NO_ADAPTER|  
|Texte du message|Changement du mot de passe Windows. Un mappage associé à ce compte Windows a été détecté mais ignoré car la synchronisation de mot de passe n'est pas configurée pour cette application.%r<br /><br /> ID de suivi : %1 %r<br /><br /> Compte Windows : %2 %r<br /><br /> Application : %3 %r<br /><br /> Compte externe : %4 %r<br /><br /> Utilisateur client : %5|  
  
## <a name="explanation"></a>Explication  
 Un mappage associé à ce compte Windows a été détecté mais ignoré car la synchronisation de mot de passe n'est pas configurée pour cette application.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Pour plus d’informations sur la configuration de la synchronisation de mot de passe, consultez [synchronisation de mot de passe](../core/password-synchronization2.md).
