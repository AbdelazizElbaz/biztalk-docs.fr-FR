# <a name="single-sign-on-event-10771"></a>Single Sign-On : Événement 10771
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10771|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|ENTSSO_E_NOT_ADMIN|  
|Texte du message|Seul un administrateur local peut exécuter cette fonction.|  
  
## <a name="explanation"></a>Explication  
 Certaines fonctions ne peuvent être exécutées que par un administrateur sur l'ordinateur local.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Demandez à votre administrateur système de contacter l'administrateur local de cet ordinateur.
