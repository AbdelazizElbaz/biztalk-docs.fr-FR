# <a name="publishing-an-orchestration-as-a-web-service"></a>Publication d'une orchestration en tant que service web
L'Assistant Publication de services Web BizTalk permet de publier une orchestration en tant que service Web. Il crée un service Web sur la base d'une orchestration dans un assembly BizTalk. À l'aide de l'Assistant, vous pouvez sélectionner des orchestrations et des ports de réception pour publier des services Web. Vous pouvez définir des espaces de noms cibles, des configurations d'en-tête SOAP et des emplacements pour le projet de service Web que l'Assistant génère.  
  
 Avant d'exécuter l'Assistant, vous devez compiler les assemblys BizTalk contenant les orchestrations et les schémas à publier en tant que services Web.  
  
 Avant d’exécuter l’Assistant de publication des Services Web BizTalk, vous devez activer les services Web. Pour plus d’informations sur l’activation des services Web pour votre système, consultez [activation des Services Web](../core/enabling-web-services.md).  
  
> [!NOTE]
>  Vous devez installer [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] avant de pouvoir publier une orchestration en tant que service Web. Il n'est pas nécessaire d'installer Microsoft [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)].
