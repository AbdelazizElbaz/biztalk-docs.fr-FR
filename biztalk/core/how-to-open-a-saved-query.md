# <a name="how-to-open-a-saved-query"></a>Ouverture d'une requête enregistrée
Vous pouvez ouvrir une requête enregistrée directement à partir du dossier dans lequel elle est stockée ou bien à partir de la console Administration de BizTalk Server. Si vous optez pour la première méthode, le groupe BizTalk Server approprié est ouvert dans la console Administration. À l'inverse, lorsque vous choisissez d'ouvrir une requête à partir de cette console, vous avez la possibilité de l'exécuter sur n'importe lequel des groupes répertoriés.  
  
## <a name="prerequisites"></a>Conditions préalables  
 Pour exécuter cette procédure, vous devez être connecté en tant que membre du groupe des opérateurs de BizTalk Server.  
  
### <a name="to-open-a-saved-query-from-disk"></a>Pour ouvrir une requête enregistrée à partir du disque  
  
1.  Accédez au dossier dans lequel vous avez enregistré la requête.  
  
    > [!NOTE]
    >  Par défaut, les requêtes enregistrées sont stockées dans le **Mes Documents** dossier de l’utilisateur qui a créé et enregistré.  
  
2.  Double-cliquez sur la requête enregistrée. La console Administration de BizTalk Server est alors ouverte et la requête exécutée.  
  
### <a name="to-open-a-saved-query-from-the-administration-console"></a>Pour ouvrir une requête enregistrée à partir de la console Administration  
  
1.  Cliquez sur **Démarrer**, cliquez sur **programmes**, cliquez sur [!INCLUDE[btsBizTalkServerStartMenuItemui](../includes/btsbiztalkserverstartmenuitemui-md.md)], puis cliquez sur **Administration de BizTalk Server**.  
  
2.  Dans l'arborescence de la console, développez [!INCLUDE[btsBizTalkServerAdminConsoleui](../includes/btsbiztalkserveradminconsoleui-md.md)], puis cliquez sur le groupe BizTalk.  
  
3.  Dans le volet détails, cliquez sur le **nouvelle requête** onglet, puis cliquez sur **ouvrir la requête**.  
  
4.  Dans le **ouvrir** boîte de dialogue Parcourir la requête enregistrée que vous souhaitez ouvrir, sélectionnez-la, puis cliquez sur **ouvrir**.
