# <a name="invalid-identifier"></a>Identificateur non valide
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]|

|Composant|0|  
|Nom symbolique|0|  
|Texte du message|« {0} » n'est pas un identificateur valide. Restauration du nom d'origine. (Un identificateur valide se compose d'une lettre suivie de zéro, un(e) ou plusieurs lettres ou chiffres et ne peut contenir d'espace.)|  
  
## <a name="explanation"></a>Explication  
 Cette erreur indique que les méthodes Web définies lors de la publication d'un schéma ne possèdent pas d'identificateur .net valide.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Renommez les méthodes Web à l'aide d'un identificateur .net valide. Un identificateur valide se compose d'une lettre suivie de zéro, un(e) ou plusieurs lettres ou chiffres et ne peut contenir d'espace.
