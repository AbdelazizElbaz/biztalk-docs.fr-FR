# <a name="developing-and-configuring-biztalk-server-as2-solutions"></a>Développement et configuration de solutions AS2 BizTalk Server
## <a name="overview"></a>Vue d'ensemble
Informations destinées aux développeurs créer des solutions BizTalk AS2. Ces solutions sont créées à l’aide de l’environnement de conception du système de projet BizTalk et le [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] console d’Administration.
  
 Avant de développer une solution AS2 BizTalk Server, assurez-vous d'avoir correctement installé et configuré [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] avec les fonctionnalités AS2. Consultez [BizTalk Server - Nouveautés, Installation, Configuration et mise à niveau](../install-and-config-guides/biztalk-server-what-s-new-installation-configuration-and-upgrade.md). Pour la configuration AS2 spécifiques supplémentaire requis pour développer une solution EDI, consultez [à la configuration des étapes pour optimiser votre environnement](../install-and-config-guides/post-configuration-steps-to-optimize-your-environment.md).
