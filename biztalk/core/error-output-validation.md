# <a name="error---output-validation"></a>Erreur : Validation de sortie
**Code d’erreur**  
  
 btm1046  
  
 **Explication**  
  
 Le fichier de message de l'instance de sortie créé par l'opération Test Map n'a pas été validé avec le schéma de destination en raison de l'erreur indiquée.  
  
 **Action de l’utilisateur**  
  
 En fonction de cette erreur, corrigez les transformations spécifiées dans le mappage ou le schéma de destination, ou bien les deux. Il peut être utile ouvrir le schéma de destination dans l’Éditeur BizTalk et utiliser le **valider le schéma**, **valider l’Instance**, et **générer l’Instance** commandes disponibles lorsque vous cliquez sur un schéma dans l’Explorateur de solutions.
