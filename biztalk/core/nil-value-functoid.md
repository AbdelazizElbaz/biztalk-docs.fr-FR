# <a name="nil-value-functoid"></a>Fonctoid Valeur nil

## <a name="description"></a> Description
Le **valeur Nil** fonctoid définit un élément de destination sur **Nil**. Il est utile pour inclure des éléments vides dans le message de sortie. Il peut être combiné avec d’autres fonctoids pour générer un **Nil** valeur basée sur des conditions spécifiques.
