# <a name="how-to-use-the-group-shape"></a>L’utilisation de la forme de groupe
![](../core/media/ebiz-orch-group.gif "ebiz_orch_group")  
Forme Groupe  
  
## <a name="annotations-and-placeholders"></a>Annotations et indicateurs de position  
 Vous pouvez utiliser la **groupe** forme comme espace réservé pour la fonctionnalité doit être ajouté, ou vous pouvez l’utiliser pour effectuer des annotations sur les actions entreprises dans celui-ci.  
  
 La propriété nom de la **groupe** forme peut comporter jusqu'à 512 caractères. Vous pouvez taper des annotations dans le **nom** propriété et, si la forme est réduite, l’intégralité du nom s’affichera. Si la forme est développée, le nom sera tronqué.
