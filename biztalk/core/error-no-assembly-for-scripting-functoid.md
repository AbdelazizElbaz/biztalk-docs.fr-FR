# <a name="error---no-assembly-for-scripting-functoid"></a>Erreur : aucun Assembly pour le fonctoid script
**Code d’erreur**  
  
 btm1007  
  
 **Explication**  
  
 Le **Assembly du Script** propriété du fonctoid **script** fonctoid n’a pas été défini même si le **le Type de Script** indique qu’un assembly externe contient le script pour ce **script** fonctoid.  
  
 **Action de l’utilisateur**  
  
 Sélectionnez le **script** fonctoid, cliquez sur le bouton de sélection (**...** ) associés à la **Script** propriété de Microsoft [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)] fenêtre Propriétés, puis définissez les valeurs appropriées dans le **configurer le Script du fonctoid** boîte de dialogue, y compris le **Assembly du Script** propriété.
