# <a name="single-sign-on-event-10775"></a>Single Sign-On : Événement 10775
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10775|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|ENTSSO_E_APP_ADMIN_SAME|  
|Texte du message|Le compte Administrateurs d'applications associé à cette application est indiqué comme étant le compte Administrateurs d'applications associées à SSO.|  
  
## <a name="explanation"></a>Explication  
 L'application a été indiquée comme utilisant le compte Administrateurs d'applications associées à SSO en tant que compte Administrateurs d'applications. Par conséquent, vous ne pouvez ni définir ni modifier le compte Administrateurs d'applications.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Contactez votre administrateur système pour obtenir de l'aide.
