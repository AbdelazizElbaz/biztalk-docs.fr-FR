# <a name="local-groups"></a>Groupes locaux
Le Gestionnaire de configuration crée le groupe Windows et les comptes d'utilisateur dont vous aurez besoin par défaut si vous installez BizTalk Server et tous les logiciels requis sur un ordinateur unique. BizTalk Server prend en charge les comptes d'utilisateur et de groupe locaux uniquement dans des configurations ne comprenant qu'un seul ordinateur.
