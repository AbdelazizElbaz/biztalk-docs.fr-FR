# <a name="manage-receive-ports"></a>Gérer les Ports de réception

## <a name="overview"></a>Vue d'ensemble
Cette rubrique explique comment créer, configurer et gérer des ports de réception d'une application BizTalk à l'aide de la console Administration de BizTalk Server. Un port de réception est un groupe logique d'emplacements de réception similaires. Pour plus d’informations sur les ports de réception, consultez [Ports de réception](../core/receive-ports.md).  
  
> [!NOTE]
>  Vous pouvez utiliser le Modèle objet de Microsoft Windows Management Instrumentation (WMI) pour créer et exécuter des scripts automatisant certaines tâches administratives. Pour plus d’informations sur l’utilisation de WMI, consultez le **référence de classe WMI** [!INCLUDE[ui-guidance-developers-reference](../includes/ui-guidance-developers-reference.md)].
