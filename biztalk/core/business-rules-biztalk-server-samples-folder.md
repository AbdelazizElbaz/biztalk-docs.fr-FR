# <a name="business-rules-biztalk-server-samples-folder"></a>Règles d’entreprise (dossier d’exemples BizTalk Server)
Le kit de développement logiciel (SDK) de Microsoft [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] inclut plusieurs exemples de règles d'entreprise. Cette section fournit des informations détaillées sur les fonctionnalités présentées dans les exemples de règles d'entreprise, des instructions sur la création et l'exécution des exemples et les résultats attendus.  
  
## <a name="in-this-section"></a>Dans cette section  
  
-   [Hello World1 de règles d’entreprise (exemple BizTalk Server)](../core/business-rules-hello-world1-biztalk-server-sample.md). Montre comment créer, enregistrer, charger et exécuter une règle BizTalk à l’aide d’un ensemble d’exemples de faits.  
  
-   [Exemple Hello World2 de règles d’entreprise (exemple BizTalk Server)](../core/business-rules-hello-world2-biztalk-server-sample.md). Montre comment vers la version, publier et déployer une règle Extensible Markup Language (XML) définie sur le magasin de règles SQL partagé, utilisez le **stratégie** de l’objet et effectuer des mises à jour dynamiques de stratégie.  
  
-   [À l’aide de règles d’entreprise (exemple BizTalk Server) de traitement de prêts](../core/loans-processing-using-business-rules-biztalk-server-sample.md). Montre comment utiliser un ensemble de règles BizTalk et de faits au sein d’une orchestration pour calculer les valeurs des champs d’un message d’instance.  
  
-   [Soins médicaux des revendications de traitement et de test des stratégies (exemple BizTalk Server)](../core/medical-claims-processing-and-testing-policies-biztalk-server-sample.md). Montre comment créer un jeu de règles contenant plusieurs règles qui examine les faits dérivés d’une table de base de données et du document entrant.
