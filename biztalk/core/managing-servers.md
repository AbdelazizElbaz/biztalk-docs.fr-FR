# <a name="managing-servers"></a>Gestion des serveurs
Le nœud Serveurs de la console Administration de BizTalk Server répertorie l'ensemble des serveurs qui appartiennent au groupe BizTalk Server. Les procédures décrites dans cette section vous expliquent comment ajouter des serveurs au groupe ou en supprimer, ou encore comment déplacer des serveurs d'un groupe vers un autre.
