# <a name="single-sign-on-event-10581"></a>Single Sign-On : Événement 10581
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10581|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|SSO_INFO_APP_DISABLED|  
|Texte du message|L'application a été désactivée.%r<br /><br /> ID de suivi : %1 %r<br /><br /> Ordinateur client : %2 %r<br /><br /> L’utilisateur client : %3 %r<br /><br /> Nom de l’application : %4|  
  
## <a name="explanation"></a>Explication  
 Ce message d'information peut s'avérer utile pour effectuer le suivi d'événements majeurs liés à la sécurité qui surviennent dans le système SSO. Ce message indique que l'application a été désactivée.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Aucune action de l'utilisateur n'est requise.
