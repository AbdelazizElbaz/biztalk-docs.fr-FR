# <a name="single-sign-on-event-10769"></a>Single Sign-On : Événement 10769
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10769|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|ENTSSO_E_APP_INCOMPLETE_FIELDS|  
|Texte du message|L'application ne peut pas être activée car les champs sont incomplets pour cette application.|  
  
## <a name="explanation"></a>Explication  
 L'application ne peut pas être activée car les champs sont incomplets pour cette application.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Lorsque vous avez spécifié le nombre de champs pour une application donnée, vous devez créer et renseigner ceux-ci avant d'activer l'application.
