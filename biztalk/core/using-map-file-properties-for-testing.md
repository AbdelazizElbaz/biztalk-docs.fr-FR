# <a name="using-map-file-properties-for-testing"></a>Utilisation des propriétés de fichier de mappage pour le test

## <a name="overview"></a>Vue d'ensemble
Les propriétés du mappage, accessibles en cliquant avec le bouton droit sur un mappage dans l'Explorateur de solutions, permettent de contrôler de nombreux comportements importants pendant le test :  
  
-   La validation des instances d’entrée et de sortie par rapport aux schémas source et de destination  
  
-   Le choix entre la création d’une instance d'entrée ou l’utilisation d’une instance existante  
  
-   Le format des instances test d'entrée et de sortie  
  
 Pour plus d’informations sur les propriétés de la carte, consultez la **référence de mappage de propriété** [!INCLUDE[ui-guidance-developers-reference](../includes/ui-guidance-developers-reference.md)].
