# <a name="success---validate-instance-generated-xml-output"></a>Réussite : valider a généré la sortie XML
**Explication**  
  
 La validation de l'instance a généré le fichier de sortie XML indiqué. Ce fichier contient l'équivalent XML de votre message d'instance entré sous un format non XML.  
  
 **Action de l’utilisateur**  
  
 Si nécessaire, examinez le fichier de sortie XML généré.
