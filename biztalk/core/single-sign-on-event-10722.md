# <a name="single-sign-on-event-10722"></a>Single Sign-On : Événement 10722
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10722|  
|Source de l'événement|ENTSSO|  
|Composant|N\A|  
|Nom symbolique|SSO_INFO_REPLAY_FILE_CLOSED_IDLE|  
|Texte du message|Le fichier de relecture a été fermé car il n'est pas actuellement utilisé.%r<br /><br /> Nom du fichier : %1|  
  
## <a name="explanation"></a>Explication  
 Cet événement d'informations indique que le fichier de relecture a été fermé car il n'est pas en cours d'utilisation.

-   Aucune action utilisateur n’est nécessaire
