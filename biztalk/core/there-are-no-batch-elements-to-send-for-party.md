# <a name="there-are-no-batch-elements-to-send-for-party"></a>Aucun élément par lot à envoyer au tiers

|||  
|-|-|  
|Nom du produit|[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]|  
|Version du produit|[!INCLUDE[btsEDIVersion](../includes/btsediversion-md.md)]|

|Composant|Moteur de traitement par lot|  
|Nom symbolique|EmptyMessageNotPossible|  
|Texte du message|Aucun élément par lot à envoyer au tiers {0}|  
  
## <a name="explanation"></a>Explication  
 Cet avertissement indique qu'aucun message de lot n'a été envoyé lorsque le déclenchement du lot a été exécuté, car aucun élément de lot n'a été reçu.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Aucune action n'est nécessaire.
