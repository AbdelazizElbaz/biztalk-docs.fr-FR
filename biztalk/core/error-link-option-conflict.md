# <a name="error---link-option-conflict"></a>Erreur : conflit d’options de lien
**Code d’erreur**  
  
 btm1022  
  
 **Explication**  
  
 Les nœuds indiqués dans le schéma de destination comportent des directives de compilateur contradictoires sur leurs liaisons entrantes.  
  
 **Action de l’utilisateur**  
  
 Sélectionnez une seule ou les deux liaisons associées aux nœuds indiqués dans le schéma de destination, puis, dans la fenêtre Propriétés de Microsoft [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)], modifiez les directives de compilateur de sorte qu'elles soient compatibles.
