# <a name="single-sign-on-event-10501"></a>Single Sign-On : Événement 10501
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10501|  
|Source de l'événement|ENTSSO|  
|Composant|N\A|  
|Nom symbolique|SSO_INFO_SERVICE_STARTING|  
|Texte du message|Le service d'authentification unique est en cours de démarrage.%r<br /><br /> Nom de l’ordinateur : %1 %r<br /><br /> Nom du serveur SQL : %2 %r<br /><br /> Nom de base de données SSO : %3 %r<br /><br /> Utilise SSL. Chiffrement de protocole appliqué aux connexions SQL Server.|  
  
## <a name="explanation"></a>Explication  
 Cet événement d'informations indique que le service d'authentification unique est en cours de démarrage avec les propriétés de configuration indiquées.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Pour résoudre cet événement d’erreur, procédez comme suit :  
  
-   Aucune action n'est nécessaire.
