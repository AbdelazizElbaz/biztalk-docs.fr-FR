[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]<span data-ttu-id="ceeda-103"> effectue les opérations suivantes pour générer une entrée de rapport d'état pour un échange sortant :</span><span class="sxs-lookup"><span data-stu-id="ceeda-103"> does the following to generate a status report entry for an outbound interchange:</span></span>  
  
1.  <span data-ttu-id="ceeda-104">Lorsqu'un message XML sortant est envoyé au pipeline d'envoi EDI, le pipeline d'envoi crée une entrée dans le magasin de données des rapports d'état, avec les valeurs suivantes :</span><span class="sxs-lookup"><span data-stu-id="ceeda-104">When outbound message XML is sent to the EDI send pipeline, the send pipeline creates an entry in the status reporting data store with the following values:</span></span>

    -   <span data-ttu-id="ceeda-106">L'entrée d'état de l'accusé de réception de l'échange (une par échange) est définie sur Attendu</span><span class="sxs-lookup"><span data-stu-id="ceeda-106">Interchange ACK status entry (one per interchange) is set to Expected</span></span>
