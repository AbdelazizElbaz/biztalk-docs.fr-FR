## <a name="explanation"></a>Explication  
 Cette erreur indique que l'URI (Uniform Resource Identifier) spécifié pour la publication du service est incorrect.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Vérifiez l'emplacement du service et déterminez s'il est valide ou non.
