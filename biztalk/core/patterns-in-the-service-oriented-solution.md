# <a name="patterns-in-the-service-oriented-solution"></a>Modèles dans la solution orientée services
Cette section présente les modèles de programmation basique constituant la solution orientée services. Elle commence par des considérations relatives aux modèles d'entreprise impliqués dans la solution, puis développe ceux-ci en modèles BizTalk. La dernière section répertorie les modèles de la solution, qu'il s'agisse de modèles d'entreprise, de BizTalk ou de programmation générale.
