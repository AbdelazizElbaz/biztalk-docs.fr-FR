# <a name="error---input-for-mass-copy-functoid-not-valid"></a>Erreur : entrée de fonctoid copie de masse non valide
**Code d’erreur**  
  
 btm1069  
  
 **Explication**  
  
 Le paramètre d’entrée pour le **copie de masse** fonctoid n’est pas valide ou trop de paramètres d’entrée pour le **copie de masse** ont été spécifiés. **Copie de masse** fonctoids doivent avoir une seule entrée : une liaison entre un **enregistrement** nœud dans le schéma source.  
  
 **Action de l’utilisateur**  
  
 Redéfinissez le **copie de masse** fonctoid telles qu’il possède un seul paramètre d’entrée une **enregistrement** nœud dans le schéma source.
