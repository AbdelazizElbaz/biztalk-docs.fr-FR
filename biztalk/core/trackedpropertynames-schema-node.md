# <a name="trackedpropertynames-schema-node"></a>TrackedPropertyNames (nœud de schéma)
Le nœud TrackedPropertyNames d'un fichier de liaison contient les éléments qui spécifient les propriétés du schéma parent à suivre.  
  
## <a name="nodes-in-the-trackedpropertyname-node"></a>Nœuds du nœud TrackedPropertyNames  
 Le tableau suivant répertorie les propriétés que vous pouvez définir pour ce nœud d'un fichier de liaison :  
  
|**Nom**|**Type de nœud**|**Type de données**|**Description**|**Restrictions**|**Commentaires**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|chaîne|Élément|xs:string|Indique un nom de propriété qui sera suivi.|Facultatif|Valeur par défaut : vide|
