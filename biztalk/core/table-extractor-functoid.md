# <a name="table-extractor-functoid"></a>Fonctoid Extracteur de table
Le **extracteur de Table** fonctoid détermine les données à extraire de chaque ligne de la grille de bouclage en tant que le **bouclage de Table** fonctoid présente les lignes. Vous devez installer un **extracteur de Table** pour chaque champ de sortie. Par exemple, supposez que votre message d'instance d'entrée a une adresse dans un seul format, mais que votre message d'instance de sortie a besoin de l'adresse dans deux formats dont chacun est marqué. Vous devez un **extracteur de Table** fonctoid de la balise et pour chaque élément de l’adresse. Pour plus d’informations sur la configuration de la **extracteur de Table** fonctoid, consultez [Configuration de bouclage Table-Driven](../core/table-driven-looping-configuration.md). Consultez également [exemple de bouclage Table-Driven](../core/table-driven-looping-example.md).
