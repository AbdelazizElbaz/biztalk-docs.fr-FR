# <a name="information-worker-technologies"></a>Technologies pour les travailleurs de l'information
Le moteur Microsoft [!INCLUDE[btsBizTalkServer2006r3](../includes/btsbiztalkserver2006r3-md.md)] prend en charge les processus d'entreprise automatisés qui couvrent plusieurs applications. Après avoir créé ces processus, les travailleurs de l'information qui les utilisent (les commerciaux et non pas les développeurs) doivent répondre à d'autres exigences. Par exemple, ils ont peut-être besoin de surveiller divers aspects commerciaux du processus ou de collaborer avec des partenaires de différentes façons.  
  
 [!INCLUDE[btsBizTalkServer2006r3](../includes/btsbiztalkserver2006r3-md.md)] offre plusieurs composants permettant de répondre à ces besoins. Ces composants sont regroupés au sein de l'analyse BAM (Business Activity Monitoring). Vous trouverez sa description dans cette section.
