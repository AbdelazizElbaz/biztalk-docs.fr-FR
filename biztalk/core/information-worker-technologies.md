# <a name="information-worker-technologies"></a>Technologies pour les travailleurs de l'information
Le moteur Microsoft BizTalk Server prend en charge les processus d’entreprise automatisés qui couvrent plusieurs applications. Après avoir créé ces processus, les travailleurs de l'information qui les utilisent (les commerciaux et non pas les développeurs) doivent répondre à d'autres exigences. Par exemple, ils ont peut-être besoin de surveiller divers aspects commerciaux du processus ou de collaborer avec des partenaires de différentes façons.  
  
 BizTalk Server fournit plusieurs composants permettant de traiter ces problèmes. Ces composants sont regroupés au sein de l'analyse BAM (Business Activity Monitoring). Vous trouverez sa description dans cette section.
