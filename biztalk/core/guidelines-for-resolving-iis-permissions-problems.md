## <a name="setting-user-access-rights-for-the-iis-server"></a><span data-ttu-id="27c21-121">Définition des droits d'accès utilisateur pour le serveur IIS</span><span class="sxs-lookup"><span data-stu-id="27c21-121">Setting User Access Rights for the IIS Server</span></span>
