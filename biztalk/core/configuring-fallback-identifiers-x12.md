### <a name="to-set-sender-receiver-and-security-properties"></a><span data-ttu-id="6072a-107">Pour définir les propriétés de l'expéditeur, du récepteur et de sécurité</span><span class="sxs-lookup"><span data-stu-id="6072a-107">To set sender, receiver, and security properties</span></span>

    > [!NOTE]
    >  <span data-ttu-id="6072a-120">La valeur **03 – mot de passe (pour la compatibilité descendante)**, est inclus pour la compatibilité descendante avec [!INCLUDE[btsbiztalkserver2006r2](../includes/btsbiztalkserver2006r2-md.md)] et sera supprimée dans les futures versions.</span><span class="sxs-lookup"><span data-stu-id="6072a-120">The value **03 – Password (for backward compatibility)**, is included for backward compatibility with [!INCLUDE[btsbiztalkserver2006r2](../includes/btsbiztalkserver2006r2-md.md)] and will be removed in future versions.</span></span>
