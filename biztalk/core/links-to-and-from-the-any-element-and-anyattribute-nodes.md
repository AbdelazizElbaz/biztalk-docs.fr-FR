# <a name="links-to-and-from-the-any-element-and-anyattribute-nodes"></a>Liens vers et à partir des nœuds Tout élément et Tout attribut
Vous devez utiliser le **copie de masse** fonctoid pour mapper les éléments contenant **tout** ou **anyAttribute** nœuds. Le Mappeur BizTalk ne prend pas en charge la liaison directe entre **tout** ou **anyAttribute** nœuds. Le **tout** et **anyAttribute** nœuds, en association avec le **copie de masse** fonctoid, sont couramment utilisés pour mapper les sections de documents HTML.
