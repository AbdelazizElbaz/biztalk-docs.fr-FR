# <a name="how-to-use-business-rules-in-orchestrations"></a>Utilisation des règles d'entreprise dans les orchestrations
Vous pouvez créer une instance d'une stratégie de règles d'entreprise et l'exécuter dans votre orchestration. Pour ce faire, ajoutez une forme Appeler règles à l'intérieur d'une étendue de transaction atomique et configurez-y une stratégie.
