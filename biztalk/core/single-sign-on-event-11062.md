# <a name="single-sign-on-event-11062"></a>Single Sign-On : Événement 11062
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|11062|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|SSO_WARN_PASSWORD_FILTER_FAILED|  
|Texte du message|Échec du filtrage de mot de passe. Aucun filtre de mot de passe ne sera utilisé.%r<br /><br /> Nom de l’application : %1 %r<br /><br /> Chaîne de filtre de mot de passe : %2 %r<br /><br /> Données supplémentaires : %3 %r<br /><br /> Code d’erreur : %4|  
  
## <a name="explanation"></a>Explication  
 Une erreur est survenue lors de la création d'un filtre de mot de passe. Le filtre n'a pas été créé.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Pour créer le filtre de mot de passe à l’aide de l’Assistant Création d’un filtre de mot de passe, consultez [comment utiliser des filtres de mot de passe](../core/how-to-use-password-filters.md).
