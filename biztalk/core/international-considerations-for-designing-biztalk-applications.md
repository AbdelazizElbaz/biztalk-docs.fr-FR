 <span data-ttu-id="c170c-107">**Caractères ne s’affichent pas correctement ou n’apparaissent pas du tout en raison d’une police incorrecte et les paramètres de secours de police**</span><span class="sxs-lookup"><span data-stu-id="c170c-107">**Characters do not appear correctly or do not appear at all due to incorrect font and font fallback settings**</span></span>  
  
 <span data-ttu-id="c170c-108">Vous pouvez rencontrer des problèmes d'affichage des caractères (avec les caractères tchèques par exemple) dans les outils [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] hébergés dans Microsoft [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)].</span><span class="sxs-lookup"><span data-stu-id="c170c-108">You may experience issues displaying characters (such as Czech characters) in the [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] tools hosted in Microsoft [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)].</span></span> <span data-ttu-id="c170c-109">Pour y remédier, il peut s'avérer nécessaire de modifier les paramètres de polices disponibles dans l'onglet Options de Visual Studio et de sélectionner une autre police prenant en charge ces caractères.</span><span class="sxs-lookup"><span data-stu-id="c170c-109">To address these issues, you may need to modify the font settings available in the Visual Studio Options tab and select another font that you know supports the characters.</span></span> <span data-ttu-id="c170c-110">Vous pouvez sélectionner Tahoma ou Microsoft Sans Serif comme police par défaut pour lesquelles des fonctionnalités de police de substitution sont fournies.</span><span class="sxs-lookup"><span data-stu-id="c170c-110">You can select Tahoma or Microsoft Sans Serif as the default font for which font fallback capabilities are provided.</span></span>

 <span data-ttu-id="c170c-118">**Restrictions de caractères de Services Web**</span><span class="sxs-lookup"><span data-stu-id="c170c-118">**Web Services character limitations**</span></span>

 <span data-ttu-id="c170c-138">**Données dépendant des paramètres régionaux dans les fichiers plats**</span><span class="sxs-lookup"><span data-stu-id="c170c-138">**Locale-dependent data in flat files**</span></span>

 <span data-ttu-id="c170c-145">Notez que vous pouvez également créer des mappages ou des composants de pipeline personnalisés afin de mettre à jour les valeurs de champ et générer un XML valide.</span><span class="sxs-lookup"><span data-stu-id="c170c-145">Note that you can also create custom pipeline components or maps to update field values to produce valid XML.</span></span>  
  
 <span data-ttu-id="c170c-146">**Prise en charge de langage de définition BAM**</span><span class="sxs-lookup"><span data-stu-id="c170c-146">**BAM definition language support**</span></span>  
  
 <span data-ttu-id="c170c-147">Avant de déployer un fichier XML de définition BAM, vous devez vous assurer que la langue utilisée pour le créer correspond aux paramètres régionaux de l'ordinateur sur lequel vous le déployez.</span><span class="sxs-lookup"><span data-stu-id="c170c-147">Before you can deploy a BAM definition XML file, you must ensure that the language used to create this file matches the locale settings of the computer on which it is being deployed.</span></span> <span data-ttu-id="c170c-148">Si les paramètres du fichier et de l'ordinateur ne correspondent pas, vous devez d'abord redémarrer l'ordinateur utilisé pour exécuter BM.exe.</span><span class="sxs-lookup"><span data-stu-id="c170c-148">If the file and the computer settings dont match, you must first reboot the computer used to run the BM.exe.</span></span>
