# <a name="single-sign-on-event-10786"></a>Single Sign-On : Événement 10786
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10786|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|ENTSSO_E_ACCOUNT_NOT_VALID|  
|Texte du message|Le nom du compte n'est pas valide ou n'existe pas. Pour plus d'informations, consultez le journal des événements (sur l'ordinateur « %1 »).|  
  
## <a name="explanation"></a>Explication  
 Le nom du compte n'est pas valide ou n'existe pas.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Vérifiez le nom et l'orthographe du compte, puis réessayez.
