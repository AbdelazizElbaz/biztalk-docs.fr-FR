# <a name="error---first-input-to-value-extractor-functoid-not-valid"></a>Erreur - première entrée du fonctoid Extracteur de valeur non valide
**Code d’erreur**  
  
 btm1002  
  
 **Explication**  
  
 Le premier paramètre d’entrée pour le fonctoid **extracteur de valeur** fonctoid doit correspondre à la sortie (jeu d’enregistrements ADO) à partir de la **recherche de base de données** fonctoid.  
  
 **Action de l’utilisateur**  
  
 Vérifiez que la première entrée à tous les **extracteur de valeur** fonctoids est un lien entre un **recherche de base de données** fonctoid à gauche de la page de grille de mappage.
