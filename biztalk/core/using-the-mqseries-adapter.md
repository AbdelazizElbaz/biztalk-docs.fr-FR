# <a name="using-the-mqseries-adapter"></a>À l’aide de l’adaptateur MQSeries
L'adaptateur MQSeries est utilisé par les organisations qui souhaitent bénéficier d'une intégration avec [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] et qui utilisent MQSeries Server comme système de messagerie principal.  
  
 Un administrateur réseau déploie et configure l'adaptateur. Un développeur de logiciels peut étendre les fonctionnalités en créant des composants d'adaptateur.  
  
> [!NOTE]
>  Vous pouvez utiliser l'adaptateur MQSeries avec [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] configuré pour plusieurs ordinateurs.
