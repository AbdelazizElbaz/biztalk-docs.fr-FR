# <a name="using-the-mqseries-adapter"></a>À l’aide de l’adaptateur MQSeries

## <a name="overvew"></a>Vue d’ensemble des
L’adaptateur MQSeries fait Office d’organisations nécessitant une intégration à BizTalk Server et à l’aide de MQSeries Server comme système de messagerie principal.  
  
 Un administrateur réseau déploie et configure l'adaptateur. Un développeur de logiciels peut étendre les fonctionnalités en créant des composants d'adaptateur.  
  
> [!NOTE]
>  Vous pouvez utiliser l’adaptateur avec BizTalk Server configuré de MQSeries pour plusieurs ordinateurs.
