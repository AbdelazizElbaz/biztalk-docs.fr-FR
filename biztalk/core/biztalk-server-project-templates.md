# <a name="biztalk-server-project-templates"></a>Modèles de projet BizTalk Server
Lorsque vous installez [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)], le processus d’installation ajoute le dossier de projets BizTalk à le **nouveau projet** boîte de dialogue. Ce dossier contient des modèles permettant de créer un projet BizTalk Server vide et un projet d'importation BPEL BizTalk Server.  
  
> [!NOTE]
>  Pour migrer un projet Biztalk pour une utilisation avec [!INCLUDE[btsBizTalkServer2006r3](../includes/btsbiztalkserver2006r3-md.md)], utilisez le [Assistant Conversion de Visual Studio](http://go.microsoft.com/fwlink/?LinkId=124086).  
  
 Le tableau suivant décrit les modèles de projet BizTalk.  
  
|Utiliser|Pour effectuer cette opération|  
|--------------|----------------|  
|**Projet BizTalk Server vide**|Créer un projet BizTalk Server vide.|  
|**Projet d’importation BPEL BizTalk Server**|Importer les langages BPEL (Business Process Execution Language), WSDL (Web Services Description Language) ou XSD (XML Schema Definition Language) dans un projet BizTalk.|
