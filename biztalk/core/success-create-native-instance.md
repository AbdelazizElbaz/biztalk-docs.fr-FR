# <a name="success---create-native-instance"></a>Réussite : créer une Instance Native
**Explication**  
  
 L'instance du schéma indiqué a été correctement générée, produisant ainsi le fichier de sortie natif qui contient le message de l'instance.  
  
 **Action de l’utilisateur**  
  
 Si nécessaire, examinez le fichier de sortie natif qui comporte le message de l'instance.
