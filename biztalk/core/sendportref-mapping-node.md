# <a name="sendportref-mapping-node"></a>SendPortRef (nœud Mappage)
Le nœud SendPortRef du nœud Mappage d'un fichier de liaison contient les noms des ports d'envoi référencés par le nœud Mappage parent.  
  
## <a name="nodes-in-the-sendportref-node"></a>Nœuds du nœud SendPortRef  
 Le tableau suivant répertorie les propriétés que vous pouvez définir pour ce nœud d'un fichier de liaison :  
  
|**Nom**|**Type de nœud**|**Type de données**|**Description**|**Restrictions**|**Commentaires**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|Nom|Attribut|xs:string|Spécifie le nom d'un port d'envoi référencé par le nœud Mappage parent.|Facultatif|Valeur par défaut : vide|
