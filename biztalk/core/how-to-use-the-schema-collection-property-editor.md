# <a name="how-to-use-the-schema-collection-property-editor"></a>L’utilisation de l’éditeur de propriétés de Collection de schémas
Vous utilisez l’Éditeur de propriétés des collections de schéma pour sélectionner des schémas pour votre composant de pipeline.  
  
### <a name="to-use-the-schema-collection-property-editor"></a>Pour utiliser l’Éditeur de propriétés des collections de schéma   
  
1.  Dans l’éditeur de propriétés de Collection de schémas, dans le **schémas disponibles** , sélectionnez les schémas qui doivent être disponibles pour le composant de pipeline. La touche CTRL vous permet de sélectionner plusieurs schémas.  
  
2.  Cliquez sur **Ajouter**.  
  
     Les schémas sélectionnés apparaissent maintenant dans la **schémas ajoutés** boîte.  
  
3.  Cliquez sur **OK**.
