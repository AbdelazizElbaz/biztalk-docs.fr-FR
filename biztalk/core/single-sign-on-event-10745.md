# <a name="single-sign-on-event-10745"></a>Single Sign-On : Événement 10745
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10745|  
|Source de l'événement|ENTSSO|  
|Composant|N\A|  
|Nom symbolique|SSO_ERROR_ADAPTER_OFFLINE|  
|Texte du message|L'adaptateur est hors ligne.%r<br /><br /> ID de suivi : %1 %r<br /><br /> Carte : %2 %r<br /><br /> Message d’erreur : %3 %r<br /><br /> Code d’erreur : %4|  
  
## <a name="explanation"></a>Explication  
 Cette erreur indique que l'adaptateur spécifié est hors ligne.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Pour résoudre cette erreur, procédez de l'une des manières suivantes :  
  
-   Consultez les journaux des événements de l'application et du système relatifs aux erreurs associées.  
  
-   Recherchez des erreurs dans l'adaptateur externe.
