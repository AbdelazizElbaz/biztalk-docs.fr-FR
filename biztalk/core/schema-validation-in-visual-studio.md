# <a name="schema-validation-in-visual-studio"></a>Validation de schémas dans Visual Studio
Une fois que vous avez terminé de créer un schéma ou durant les diverses étapes de sa création, vous pouvez le valider afin de déterminer s'il contient des incohérences internes ou s'il présente d'autres problèmes susceptibles de l'empêcher d'être utilisé efficacement avec les messages d'instance de processus.  
  
 Pour valider un schéma, utilisez la **valider le schéma** du menu contextuel associé au schéma dans l’Explorateur de solutions. Les résultats de la validation sont affichés dans la fenêtre Sortie de [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)].  
  
 Pour obtenir des instructions détaillées concernant la validation d’un schéma, consultez [validation de schémas](../core/how-to-validate-schemas-in-visual-studio.md).
