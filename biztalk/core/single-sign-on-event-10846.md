# <a name="single-sign-on-event-10846"></a>Single Sign-On : Événement 10846
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10846|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|ENTSSO_E_HOST_GROUP_ONE_ACCOUNT_ONLY|  
|Texte du message|Les applications de groupe d'hôtes ne peuvent pas avoir plusieurs comptes spécifiés pour le compte Windows.|  
  
## <a name="explanation"></a>Explication  
 Les applications de groupe d'hôtes ne peuvent pas avoir plusieurs comptes spécifiés pour le compte Windows.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Pour plus d’informations sur les applications de groupe hôte, consultez [Applications associées SSO](../core/sso-affiliate-applications.md).
