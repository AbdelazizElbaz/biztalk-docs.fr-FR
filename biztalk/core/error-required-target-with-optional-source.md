# <a name="error---required-target-with-optional-source"></a>Erreur : cible obligatoire avec Source facultative
**Code d’erreur**  
  
 btm1001  
  
 **Explication**  
  
 Les données du nœud requis indiqué dans le schéma de destination proviennent d'un nœud facultatif du schéma source. Par conséquent, il se peut que le mappage de certains messages de l'instance valides échoue.  
  
 **Action de l’utilisateur**  
  
 Confirmez les caractéristiques facultatives et obligatoires des nœuds source et de destination spécifiés et vérifiez que celles-ci soient compatibles.
