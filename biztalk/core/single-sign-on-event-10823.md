# <a name="single-sign-on-event-10823"></a>Single Sign-On : Événement 10823
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10823|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|ENTSSO_E_PSADMIN_INVALID_APP_TYPE|  
|Texte du message|Type d'application non valide pour l'application. Les types d'applications valides sont « Individuel », « Groupe » ou « Groupe d'hôtes ».|  
  
## <a name="explanation"></a>Explication  
 Type d'application non valide pour l'application. Les types d'applications valides sont « Individuel », « Groupe » ou « Groupe d'hôtes ».  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Utilisez l'un des types d'applications valides.
