# <a name="success---create-xml-instance"></a>Réussite : créer une Instance XML
**Explication**  
  
 L'instance du schéma indiqué a été correctement générée, produisant ainsi le fichier de sortie XML qui contient le message de l'instance.  
  
 **Action de l’utilisateur**  
  
 Si nécessaire, examinez le fichier de sortie XML qui comporte le message de l'instance.
