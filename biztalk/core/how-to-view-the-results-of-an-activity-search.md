# <a name="how-to-view-the-results-of-an-activity-search"></a>Affichage des résultats d'une recherche d'activité
Les utilisateurs peuvent afficher les résultats d’une requête en cliquant sur le **exécuter la requête** bouton dans la section supérieure du cadre contenu du portail BAM. Les résultats de la requête sont affichés dans le **résultats** section en bas de l’image.  
  
 Les résultats d'une recherche d'activité apparaissent à raison d'un enregistrement par ligne. Chaque enregistrement se compose des données et étapes majeures sélectionnées dans le sélecteur de colonne et sont affichent dans l’ordre dans lequel ils ont été organisées dans le **éléments à afficher** boîte.  
  
 Pour obtenir des détails supplémentaires à propos d'un enregistrement renvoyé, cliquez n'importe où dans la ligne.  
  
## <a name="prerequisites"></a>Conditions préalables  
 Pour suivre les procédures de cette rubrique, vous devez avoir déployé une activité. Pour plus d’informations sur la procédure à suivre, consultez [comment appliquer et supprimer un profil de suivi](../core/how-to-apply-and-remove-a-tracking-profile.md).  
  
## <a name="return-results"></a>Renvoi de résultats  
  
#### <a name="to-execute-a-query-to-return-results"></a>Pour exécuter une requête afin de renvoyer des résultats  
  
1.  Cliquez sur **Démarrer**, pointez sur **tous les programmes**, cliquez sur [!INCLUDE[btsBizTalkServerStartMenuItemui](../includes/btsbiztalkserverstartmenuitemui-md.md)], puis cliquez sur **Site Web du portail BAM**.  
  
2.  Dans le **Mes vues** volet, cliquez sur une vue pour laquelle vous créez la recherche d’activité.  
  
3.  Cliquez sur une activité associée à la vue pour charger cette activité dans le frame de contenu.  
  
4.  Créez une recherche d'activité ou ouvrez-en une existante. Pour plus d’informations sur la création ou l’ouverture des recherches d’activité, consultez [la création d’une requête de recherche d’activité](../core/how-to-create-a-query-in-activity-search.md).  
  
5.  Cliquez sur le **exécuter la requête** situé en haut du cadre contenu du portail.  
  
6.  Faites défiler vers le bas pour afficher la sortie de la requête dans le **résultats** zone du cadre contenu.  
  
## <a name="view-results-information"></a>Affichage des informations de résultat  
 Vous pouvez afficher l'état détaillé des éléments renvoyés par la requête. Parmi les détails supplémentaires qui sont affichés, on trouve :  
  
-   L'état de l'activité, composé d'étapes majeures et de données associées  
  
-   Documents associés  
  
-   Activités associées  
  
 En outre, vous pouvez demander une assistance à votre administrateur en cliquant sur le **Assistance** bouton ci-dessous le **des activités connexes** zone dans le cadre contenu.  
  
#### <a name="to-view-detailed-status-information-on-results"></a>Pour afficher des informations d'état détaillées concernant les résultats  
  
1.  Suivez la procédure précédente pour exécuter une requête et renvoyer des résultats.  
  
2.  Cliquez sur une ligne pour charger les informations d'état détaillées ayant trait à l'élément de résultat.  
  
3.  Si la zone contenant les informations détaillées qui vous intéressent est réduite, ouvrez-la en cliquant sur le signe plus (+) en regard de l'étiquette de la zone.
