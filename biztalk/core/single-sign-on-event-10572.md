# <a name="single-sign-on-event-10572"></a>Single Sign-On : Événement 10572
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10572|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|SSO_WARN_SSO_DISABLED_FOR_CHANGE_SSO_AFF_ADMIN|  
|Texte du message|Pour mettre à jour le nom du compte Administrateurs d'applications associées à authentification unique, vous devez commencer par désactiver l'authentification unique.%r|  
  
## <a name="explanation"></a>Explication  
 Pour mettre à jour le nom du compte Administrateurs d'applications associées à authentification unique, vous devez commencer par désactiver l'authentification unique.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Pour plus d’informations sur la désactivation de l’authentification unique, consultez [comment désactiver le SSO](../core/how-to-disable-sso.md).
