# <a name="configuring-ports-for-an-edi-solution"></a>Configuration des ports d'une solution EDI
Pour recevoir et envoyer des messages EDI et des accusés de réception, créez les ports de réception et d'envoi suivants :  
  
-   un port de réception FILE statique unidirectionnel qui reçoit les échanges EDI et les accusés de réception ou un port de réception FILE de requête-réponse bidirectionnel qui reçoit les échanges EDI et envoie les accusés de réception ;  
  
-   un port d'envoi FILE unidirectionnel qui envoie les échanges EDI ou les accusés de réception.
