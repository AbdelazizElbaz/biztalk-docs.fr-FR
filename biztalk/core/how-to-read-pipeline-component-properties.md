# <a name="how-to-read-pipeline-component-properties"></a>La lecture des propriétés du composant de Pipeline
La fenêtre Propriétés contient deux sections pour les composants : les propriétés générales et les propriétés du composant. Les propriétés générales sont communes à tous les composants, bien que leurs valeurs changent d’un composant à l’autre.  
  
### <a name="to-read-the-general-properties-for-a-pipeline-component"></a>Pour lire les propriétés générales d’un composant de pipeline  
  
1.  Déplacez le composant de pipeline dans une étape de pipeline ou sélectionnez un composant s’il existe déjà dans le pipeline.  
  
2.  Dans la fenêtre Propriétés, dans le **général** section, procédez comme suit.  
  
    |Utiliser|Pour effectuer cette opération|  
    |--------------|----------------|  
    |**Nom**|Indique le nom du composant.|  
    |**Assembly**|Indique l'assembly du composant et les informations .NET qui lui sont associées.|  
    |**Description**|Indique le nom convivial du composant de pipeline.|  
    |**Géré**|Indique si le composant de pipeline est géré. **Oui** si le composant est un composant .NET géré ; **Non** si le composant de pipeline est un composant COM.|  
    |**Chemin d'accès**|Indique le chemin d’accès complet au composant .NET.|  
    |**Type**|Indique le type, l'assembly et les informations .NET associées du composant.|
