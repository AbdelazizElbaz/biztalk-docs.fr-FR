# <a name="error---multiple-loop-paths"></a>Erreur : plusieurs chemins de bouclage
**Code d’erreur**  
  
 btm1030  
  
 **Explication**  
  
 Le nœud indiqué dans le schéma de destination possède plusieurs chemins de bouclage sources.  
  
 **Action de l’utilisateur**  
  
 Utilisez un **bouclage** fonctoid pour spécifier explicitement le nœud dans le schéma source sur laquelle le bouclage sera effectué.
