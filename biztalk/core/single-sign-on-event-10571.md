|-|-|  
|Nom du produit|Enterprise Single Sign-On|

## <a name="explanation"></a>Explication  
 Pour modifier le nom du compte Administrateurs de l'authentification unique, le compte du service d'authentification unique doit être membre du nouveau compte Administrateurs de l'authentification unique.
