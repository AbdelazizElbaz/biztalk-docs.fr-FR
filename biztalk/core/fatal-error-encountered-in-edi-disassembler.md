# <a name="fatal-error-encountered-in-edi-disassembler"></a>Erreur fatale rencontrée dans le Désassembleur EDI
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]|

|Composant|Moteur EDI|  
|Nom symbolique|EdiDasmFatalError|  
|Texte du message|Erreur fatale rencontrée dans le Désassembleur EDI, les informations sur l'erreur sont {0}|  
  
## <a name="explanation"></a>Explication  
 Cet événement d'erreur/d'avertissement/d'informations indique que le pipeline de réception n'a pas pu traiter l'échange entrant en raison de l'erreur mentionnée dans le Désassembleur EDI.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Pour résoudre cette erreur, identifiez la source de l'erreur indiquée, puis consultez l'aide du produit pour déterminer la manière de la corriger.
