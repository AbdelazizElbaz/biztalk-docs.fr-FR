# <a name="how-to-view-the-references-of-an-application"></a>Affichage des références d'une application
Cette rubrique explique comment afficher la liste des applications pour lesquelles l'application active possède des références à l'aide de la console Administration de BizTalk Server. Pour plus d’informations sur l’ajout de références, consultez [comment ajouter une référence à une autre Application](../core/how-to-add-a-reference-to-another-application.md). Vous pouvez également afficher la liste des applications qui possèdent des références à cette application active.  
  
## <a name="prerequisites"></a>Conditions préalables  
 Pour exécuter la procédure décrite dans cette rubrique, vous devez être connecté avec un compte membre du groupe d'administrateurs BizTalk Server. Pour plus d’informations sur les autorisations, consultez [autorisations requises pour déployer et gérer une Application BizTalk](../core/permissions-required-for-deploying-and-managing-a-biztalk-application.md).  
  
### <a name="to-view-the-references-for-an-application"></a>Pour afficher les références d'une application  
  
1.  Cliquez sur **Démarrer**, pointez sur **tous les programmes**, pointez sur [!INCLUDE[btsBizTalkServerStartMenuItemui](../includes/btsbiztalkserverstartmenuitemui-md.md)], puis cliquez sur **Administration de BizTalk Server**.  
  
2.  Dans l’arborescence de la console, développez [!INCLUDE[btsBizTalkServerAdminConsoleui](../includes/btsbiztalkserveradminconsoleui-md.md)], cliquez sur l’application dont vous souhaitez afficher, puis cliquez sur les références **propriétés**.  
  
3.  Dans le volet gauche de la page de propriétés, cliquez sur **références**.  
  
     Les applications auxquelles fait référence cette application sont répertoriées dans la partie supérieure du volet droit. Les applications faisant référence à cette application sont répertoriées dans la partie inférieure du volet droit.
