# <a name="enlisted-parties-role-node"></a>Tiers inscrits (nœud du rôle)
Le nœud Tiers inscrit d'un fichier de liaison est le nœud parent de tous les nœuds Tiers inscrit qui contiennent des informations spécifiques sur les tiers inscrits associés à un rôle exporté avec le fichier de liaison.  
  
## <a name="nodes-in-the-enlisted-parties-node"></a>Nœuds figurant dans le nœud Tiers inscrit  
 Le tableau suivant répertorie les propriétés que vous pouvez définir pour ce nœud d'un fichier de liaison :  
  
|**Nom**|**Type de nœud**|**Type de données**|**Description**|**Restrictions**|**Commentaires**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|[Tiers inscrit](../core/enlisted-party-enlisted-parties-node.md)|Record|EnlistedParty (ComplexType)|Spécifie les informations relatives à un tiers inscrit associé à un rôle.|Facultatif|Valeur par défaut : Aucun|
