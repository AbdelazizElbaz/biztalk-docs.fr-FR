# <a name="single-sign-on-event-10849"></a>Single Sign-On : Événement 10849
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10849|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|ENTSSO_E_DIRECT_SYNC_NOT_ALLOWED_CREATE|  
|Texte du message|Une application ne peut pas être créée avec l'indicateur « direct password sync » spécifié.|  
  
## <a name="explanation"></a>Explication  
 Une application ne peut pas être créée avec l'indicateur « direct password sync » spécifié.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Créez l'application sans l'indicateur « direct password sync ». Ensuite, ajoutez et créez les champs, activez l'application et spécifiez l'indicateur « direct password sync ».
