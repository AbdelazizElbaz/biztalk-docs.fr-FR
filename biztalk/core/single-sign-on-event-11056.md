## <a name="explanation"></a>Explication  
 Le mot de passe n'a pas été directement modifié pour le compte externe car les informations d'identification externes existantes n'ont pas pu être obtenues auprès de la base de données SSO.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Utilisez les informations fournies et contactez les services de support technique Microsoft.
