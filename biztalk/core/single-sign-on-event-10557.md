# <a name="single-sign-on-event-10557"></a>Single Sign-On : Événement 10557
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10557|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|SSO_WARN_USER_NOT_ALLOWED_FOR_GROUPS|  
|Texte du message|Les utilisateurs d'applications ne sont pas autorisés à contrôler les mappages relatifs aux applications de groupe.%r<br /><br /> Nom de domaine : %1 %r<br /><br /> Nom d’utilisateur : %2 %r<br /><br /> Nom de l’application : %3 %r<br /><br /> Utilisateur client : %4|  
  
## <a name="explanation"></a>Explication  
 Les utilisateurs d'applications ne disposent pas de privilèges suffisants pour créer ou contrôler les mappages relatifs aux applications de groupe.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Cette activité doit être effectuée par un administrateur d'application.
