# <a name="silent-configuration-of-the-mqseries-adapter"></a>Configuration silencieuse de l’adaptateur MQSeries
L'Assistant Configuration de l'application MQSAgent COM+ permet de réaliser une configuration silencieuse à partir de la ligne de commande. Vous pouvez également exécuter la version de ligne de commande de l'Assistant pour supprimer MQSAgent. L'Assistant lit les informations de configuration dans un fichier XML.
