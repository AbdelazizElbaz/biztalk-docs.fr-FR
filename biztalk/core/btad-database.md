# <a name="btaddatabase"></a>BTAD_Database
Lorsque les scripts de pré-traitement et de post-traitement sont exécutés lors de l'importation de l'application, la variable d'environnement BTAD_Database est définie. Elle spécifie le nom de la base de données de gestion BizTalk pour le groupe.
