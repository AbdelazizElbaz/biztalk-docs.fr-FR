# <a name="single-sign-on-event-10788"></a>Single Sign-On : Événement 10788
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10788|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|ENTSSO_E_DTC_IMPORT_FAILED1|  
|Texte du message|Impossible d'importer une transaction DTC. Vérifiez que MSDTC est correctement configuré pour l'opération distante. Pour plus d'informations, reportez-vous à la documentation.|  
  
## <a name="explanation"></a>Explication  
 Impossible d'importer une transaction DTC.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Vérifiez que MSDTC est correctement configuré pour l'opération distante et consultez le journal des événements sur l'ordinateur spécifié pour obtenir davantage d'informations.  
  
 Pour obtenir de l’aide sur les problèmes MSDTC, consultez [Troubleshooting Problems with MSDTC](../core/troubleshooting-problems-with-msdtc.md).
