# <a name="how-to-save-a-pipeline"></a>Comment enregistrer un Pipeline
L’enregistrement d’un pipeline stocke des informations à propos de la configuration du pipeline dans un fichier .btp, qui spécifie :  
  
-   le nom du pipeline ;  
  
-   le modèle et la stratégie sur lesquels il s’appuie (envoi ou réception) ;  
  
-   la séquence des composants faisant partie du pipeline, y compris l’ordre et l’ensemble des composants utilisés à chaque étape ;  
  
-   les informations de configuration de chaque composant de pipeline.  
  
### <a name="to-save-a-pipeline"></a>Pour enregistrer un pipeline  
  
1.  Dans le menu **Fichier** , cliquez sur **Enregistrer**.  
  
     — Ou :  
  
2.  Affichez le pipeline sur la surface de conception en cliquant sur son onglet en haut de l’écran.  
  
3.  Dans le menu **Fichier** , cliquez sur **Enregistrer sous**.  
  
4.  Dans le **enregistrer le fichier sous** boîte de dialogue le **fichier** nom de champ, tapez un nom pour le pipeline. Un nom par défaut vous est fourni.  
  
5.  Cliquez sur **OK**.
