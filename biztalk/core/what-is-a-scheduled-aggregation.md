# <a name="what-is-a-scheduled-aggregation"></a>Qu'est-ce qu'une agrégation planifiée ?
Une agrégation planifiée représente un instantané de l'activité à un moment donné, par exemple, un résumé des expéditions du matin. Demandez à votre administrateur de la base de données à quel moment vos agrégations sont traitées. Vous pouvez ensuite consulter les données historiques.
