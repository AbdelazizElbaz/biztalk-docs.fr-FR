# <a name="error--the-quot0quot-functoid-has-place-holders-replace-these-with-appropriate-links-or-constants"></a>Erreur : le &quot;{0}&quot; fonctoid a des espaces réservés. Remplacez-les par les constantes ou liens appropriés.
**Code d’erreur**  
  
 btm1090  
  
 **Action de l’utilisateur**  
  
 Le fonctoid comportant des liens d'espace réservé a des balises d'avertissement sur ces éléments dans l'interface utilisateur. Les liens d'espace réservé doivent être remplacés par les constantes ou liens appropriés.
