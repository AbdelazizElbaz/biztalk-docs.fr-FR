# <a name="how-to-invoke-the-orchestration-debugger-and-the-message-flow-views"></a>Appel des vues Flux message et Débogueur orchestration
Dans la page Hub du groupe de la console Administration de BizTalk Server, le volet Résultats de la requête affiche le résultat d'une requête. À partir du menu contextuel, vous pouvez afficher la vue Débogueur orchestration ou Flux message.  
  
## <a name="prerequisites"></a>Conditions préalables  
 Pour exécuter cette procédure, vous devez ouvrir une session en tant que membre du groupe Administrateurs de BizTalk Server.  
  
### <a name="to-display-the-orchestration-debugger-or-message-flow-views"></a>Pour afficher la vue Débogueur orchestration ou Flux message  
  
1.  Pour afficher la vue Flux message, cliquez sur une cellule associée à une instance d’orchestration dans le **résultats de la requête** et **le flux des messages**.  
  
2.  Pour afficher le débogueur d’orchestration, cliquez sur une cellule associée à une instance d’orchestration dans le **résultats de la requête** et **débogueur Orchestration**.
