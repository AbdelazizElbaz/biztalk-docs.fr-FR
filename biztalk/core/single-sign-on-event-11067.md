# <a name="single-sign-on-event-11067"></a>Single Sign-On : Événement 11067
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|11067|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|SSO_ERROR_PASSWORD_SYNC_MIIS_START_FAILED|  
|Texte du message|La synchronisation des mots de passe pour Windows (à partir de MIIS) n'a pas pu démarrer.%r<br /><br /> Code d’erreur : %1|  
  
## <a name="explanation"></a>Explication  
 Ce message d'information indique que la synchronisation des mots de passe pour Windows (à partir de MIIS) n'a pas pu démarrer.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Aucune action de l'utilisateur n'est requise.
