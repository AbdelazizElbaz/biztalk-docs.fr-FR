# <a name="using-expressions-in-orchestrations"></a>Utilisation d'expressions dans les orchestrations
Vous pouvez utiliser l'Éditeur d'expression BizTalk pour entrer des expressions XLANG/s et ajouter une logique afin de manipuler et de tester les valeurs des variables et messages de votre orchestration. Cependant, il est généralement déconseillé de l'utiliser pour exécuter une logique d'orchestration de niveau supérieur, qu'il serait préférable de voir dans le dessin de l'orchestration lui-même. Pour la transparence et la reconfiguration simple de votre processus d'entreprise, il est recommandé d'utiliser des expressions simples et modulables.
