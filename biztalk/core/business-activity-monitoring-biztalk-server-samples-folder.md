# <a name="business-activity-monitoring-biztalk-server-samples-folder"></a>Business Activity Monitoring (dossier d’exemples BizTalk Server)
Le Kit de développement logiciel (SDK) de Microsoft [!INCLUDE[btsBizTalkServer2006r3](../includes/btsbiztalkserver2006r3-md.md)] inclut plusieurs exemples BAM (Business Activity Monitoring). Cette section fournit des informations détaillées sur les fonctionnalités présentées dans les exemples BAM, des instructions sur la création et l'exécution des exemples et les résultats attendus.  
  
## <a name="in-this-section"></a>Dans cette section  
  
-   [L’API BAM (exemple BizTalk Server)](../core/bam-api-biztalk-server-sample.md). Montre comment :  
  
    -   un analyste d'entreprise définit des données intéressantes à l'aide d'un Assistant, examine les agrégations et recherche les données d'activité.  
  
    -   un administrateur TI crée une infrastructure SQL, DTS (Data Transformation Services) et OLAP (Online Analytical Processing).  
  
    -   un développeur instrumente l'implémentation d'entreprise.  
  
-   [BAM de bout en bout (exemple BizTalk Server)](../core/bam-end-to-end-biztalk-server-sample.md). Montre comment mettre en corrélation les événements à partir de plusieurs composants de l’analyse BAM.  
  
-   [API BAM à partir d’une Expression d’Orchestration (exemple BizTalk Server)](../core/bam-api-from-an-orchestration-expression-biztalk-server-sample.md). Montre comment :  
  
    -   utiliser l'API BAM à partir d'une orchestration BizTalk Server.  
  
    -   suivre des éléments répétés à l'intérieur d'un message comme des instances d'activité distinctes.  
  
    -   établir une relation entre les données BAM suivies à partir d'une orchestration à l'aide d'un modèle de suivi et des données suivies de l'API.
