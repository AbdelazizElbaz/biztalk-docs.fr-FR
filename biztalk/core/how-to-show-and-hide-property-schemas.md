# <a name="how-to-show-and-hide-property-schemas"></a>Affichage et masquage des schémas de propriété
La présente rubrique explique comment utiliser la console Administration de BizTalk Server pour afficher et masquer les schémas de propriété du dossier Schémas d'une application. Un schéma de propriété est la version simple d'un schéma BizTalk qui joue un rôle dans le processus de copie de propriétés promues dans un sens et dans l'autre entre le message d'instance et le contexte de message. Vous pouvez masquer les schémas de propriété afin de simplifier l'affichage des schémas et de ne visualiser que les schémas de document.  
  
## <a name="prerequisites"></a>Conditions préalables  
 Pour exécuter cette procédure, vous devez ouvrir une session en tant que membre du groupe Administrateurs BizTalk Server ou Opérateurs BizTalk Server. Pour plus d’informations sur les autorisations, consultez [autorisations requises pour déployer et gérer une Application BizTalk](../core/permissions-required-for-deploying-and-managing-a-biztalk-application.md).  
  
### <a name="to-show-or-hide-property-schemas-in-the-schemas-folder"></a>Pour afficher ou masquer les schémas de propriété du dossier Schémas  
  
1.  Cliquez sur **Démarrer**, cliquez sur **tous les programmes**, cliquez sur [!INCLUDE[btsBizTalkServerStartMenuItemui](../includes/btsbiztalkserverstartmenuitemui-md.md)], puis cliquez sur **Administration de BizTalk Server**.  
  
2.  Dans l’arborescence de la console, développez **Administration de BizTalk Server**, développez le groupe BizTalk contenant le dossier dans lequel vous souhaitez afficher ou masquer les schémas de propriété, puis développez l’application contenant le dossier.  
  
3.  Avec le bouton droit le **schémas** dossier, cliquez sur **masquer les schémas de propriété** ou **afficher les schémas de propriété**.
