# <a name="single-sign-on-event-10776"></a>Single Sign-On : Événement 10776
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10776|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|ENTSSO_E_INVALID_ACCOUNT_FORMAT|  
|Texte du message|Le format du nom du compte n'est pas valide. Les comptes de domaine doivent inclure le nom de domaine. Les comptes locaux ne doivent pas inclure un nom de domaine ou d'ordinateur.|  
  
## <a name="explanation"></a>Explication  
 Le format du nom du compte n'est pas valide. Les comptes de domaine doivent inclure le nom de domaine. Les comptes locaux ne doivent pas inclure un nom de domaine ou d'ordinateur.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Spécifiez un autre nom de compte.
