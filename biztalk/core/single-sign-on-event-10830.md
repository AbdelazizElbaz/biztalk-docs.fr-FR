# <a name="single-sign-on-event-10830"></a>Single Sign-On : Événement 10830
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10830|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|ENTSSO_E_PSADMIN_ADAPTER_SAME_COMPUTER|  
|Texte du message|L'adaptateur spécifié doit figurer sur le même ordinateur que l'adaptateur de groupe.|  
  
## <a name="explanation"></a>Explication  
 Chaque adaptateur d'un adaptateur de groupe doit être configuré avec le même nom d'ordinateur que ce dernier.
