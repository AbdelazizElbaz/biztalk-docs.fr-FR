# <a name="single-sign-on-event-11047"></a>Single Sign-On : Événement 11047
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|11047|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|SSO_ERROR_SSOSQL_FAILED|  
|Texte du message|Impossible de créer SSOSQL. Pour résoudre le problème, réinstallez l'authentification unique ou tapez « regasm SSOSQL.dll » dans une invite de commandes Visual Studio.%r<br /><br /> Code d’erreur : %1|  
  
## <a name="explanation"></a>Explication  
 Ce problème est probablement dû à une erreur d'installation.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Pour résoudre le problème, réinstallez l'authentification unique ou tapez « regasm SSOSQL.dll » dans une invite de commandes Visual Studio.
