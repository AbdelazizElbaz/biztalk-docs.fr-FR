# <a name="sample-biztalk-server-architectures"></a>Exemples d'architectures BizTalk Server
Pour que votre environnement Microsoft® BizTalk® Server bénéficie d'une sécurité optimale, il est recommandé d'utiliser une architecture dans laquelle les serveurs BizTalk Server sont distribués selon le niveau de gravité des données et services qu'ils contiennent et leur degré d'implication si une attaque les compromet.  
  
 Cette section fournit des informations pour vous aider à dériver votre déploiement sécurisé de BizTalk Server afin de répondre aux besoins et défis particuliers de votre entreprise et de votre environnement.
