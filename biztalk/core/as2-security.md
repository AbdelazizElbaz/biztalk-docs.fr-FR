# <a name="as2-security"></a>Sécurité AS2
La fonctionnalité AS2 native de Microsoft [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] permet de sécuriser les messages AS2 à l'aide de clés de chiffrement, de signatures numériques et de certificats. Elle requiert la configuration correcte de [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] et des certificats.  
  
 Pour plus d’informations sur la structure d’un message AS2 signé et/ou chiffré, consultez [les Messages AS2](../core/as2-messages.md).
