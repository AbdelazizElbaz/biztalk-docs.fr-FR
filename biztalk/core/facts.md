# <a name="facts"></a>Faits
Les faits sont des éléments d'information discrets sur le monde. Faits peuvent provenir de sources diverses (systèmes d’événements, objets dans les applications d’entreprise, les tables de base de données et ainsi de suite) et doivent être fournis au moteur de règle d’entreprise de [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] à l’aide d’un des éléments suivants :  
  
-   objets .NET (méthodes, propriétés et champs) ;  
  
-   documents XML (éléments, attributs et sous-sections de documents) ;  
  
-   ensembles de lignes de base de données (valeurs des colonnes de table).  
  
 Dans l'Éditeur des règles d'entreprise, vous pouvez utiliser l'Explorateur de faits pour parcourir les données de sources diverses et les insérer dans vos règles.
