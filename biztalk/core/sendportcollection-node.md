# <a name="sendportcollection-node"></a>Nœud SendPortCollection
Le nœud SendPortCollection d'un fichier de liaison est le nœud parent de tous les nœuds SendPort qui contiennent des informations spécifiques sur un port d'envoi exporté avec le fichier de liaison.  
  
## <a name="nodes-in-the-sendportcollection-node"></a>Nœuds du nœud SendPortCollection  
 Le tableau suivant répertorie les propriétés que vous pouvez définir pour ce nœud d'un fichier de liaison :  
  
|**Nom**|**Type de nœud**|**Type de données**|**Description**|**Restrictions**|**Commentaires**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|[Le port d’envoi](../core/sendport-sendportcollection-node.md)|Record|SendPort (ComplexType)|Spécifie les informations relatives à un port d'envoi exporté avec le fichier de liaison.|Facultatif|Valeur par défaut : Aucun|
