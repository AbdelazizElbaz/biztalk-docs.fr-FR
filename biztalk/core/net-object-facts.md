# <a name="net-object-facts"></a>Faits d'objets .NET
Dans l'Éditeur des règles d'entreprise, vous pouvez spécifier un assembly .NET comme source de données. Vous pouvez ensuite sélectionner une classe ou un membre de classe à partir de l’assembly et le déplacer dans une règle ou une définition.  
  
 Le traitement d'objets .NET déclarés est le type de traitement le plus simple. Le moteur traite chaque objet comme une instance unique. Pour plus d’informations sur la**Assert** , consultez [les fonctions de contrôle du moteur](../core/engine-control-functions.md).
