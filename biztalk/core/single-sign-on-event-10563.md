# <a name="single-sign-on-event-10563"></a>Single Sign-On : Événement 10563
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10563|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|SSO_WARN_PERFMON_FAILED|  
|Texte du message|L'analyse des performances n'a pas pu démarrer.%r<br /><br /> Code d’erreur : %1|  
  
## <a name="explanation"></a>Explication  
 L'analyse des performances n'a pas pu démarrer. Le système continuera à s'exécuter normalement mais l'analyse des performances ne sera pas disponible.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Il peut être nécessaire de désinstaller et réinstaller l'utilitaire perfmon.
