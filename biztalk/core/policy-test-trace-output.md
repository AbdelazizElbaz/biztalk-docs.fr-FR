# <a name="policy-test-trace-output"></a>Sortie de Trace de Test de stratégie
Lorsque vous testez votre stratégie, le testeur de stratégie affiche le résultat dans la fenêtre d’informations de version. Les activités incluses dans le suivi sont décrites dans cette section.  
  
> [!NOTE]
>  Si une classe dérivée est déclarée, mais que les règles sont écrites directement par rapport aux membres de la classe de base, une instance de la classe de base est déclarée et les conditions sont évaluées par rapport à cette instance.
