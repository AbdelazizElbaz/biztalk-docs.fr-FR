# <a name="single-sign-on-event-10819"></a>Single Sign-On : Événement 10819
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|Enterprise Single Sign-On|  
|Version du produit|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|ID d'événement|10819|  
|Source de l'événement|ENTSSO|  
|Composant|Néant|  
|Nom symbolique|ENTSSO_E_MAPPING_CONFLICT|  
|Texte du message|Le compte externe n'a pas été mis à jour en raison d'un conflit de mappage.|  
  
## <a name="explanation"></a>Explication  
 Le compte externe n'a pas été mis à jour en raison d'un conflit de mappage.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 S'il s'agit du comportement attendu, aucune action n'est requise, ce message n'est qu'informatif. Si les conflits de mappage doivent être autorisés, configurez l'application en ce sens.  
  
 Pour plus d’informations sur les conflits de mappage, consultez **propriétés de carte de synchronisation de mot de passe : Options** [!INCLUDE[ui-guidance-developers-reference](../includes/ui-guidance-developers-reference.md)].
