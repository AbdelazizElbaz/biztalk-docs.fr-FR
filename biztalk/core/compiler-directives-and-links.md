# <a name="compiler-directives-and-links"></a>Directives et liaisons du compilateur
Vous pouvez configurer les deux directives de compilateur suivantes lien par lien :  
  
-   Les données du message d’instance d’entrée qui sont récupérées et copiées comme valeur d’attribut ou d'élément pertinente dans le message d'instance de sortie  
  
    > [!NOTE]
    >  Cela comprend la possibilité de copier le nom de l’élément ou de l’attribut lui-même plutôt qu'une valeur associée à l’élément ou à l'attribut. Les noms d’élément et d’attribut ne sont normalement pas considérés comme faisant partie des données contenues dans un message d’instance XML.  
  
-   Le mode d'exécution d'une correspondance de nœuds entre les schémas source et de destination  
  
 Cette section propose une explication détaillée de ces options disponibles pour ces directives.
