# <a name="troubleshooting-biztalk-server-permissions"></a>Dépannage des autorisations BizTalk Server
Dans BizTalk Server, les problèmes liés aux autorisations se répartissent généralement dans les catégories suivantes :  
  
-   Autorisations SQL Server  
  
-   autorisations Active Directory sur des installations multiserveur BizTalk Server ;  
  
-   autorisations de services Web ;  
  
-   autorisations IIS.  
  
 Cette rubrique propose des recommandations d'ordre général pour éviter les problèmes d'autorisations, ainsi que des instructions de dépannage liées à des exemples précis.
