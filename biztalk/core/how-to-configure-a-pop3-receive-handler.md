# <a name="how-to-configure-a-pop3-receive-handler"></a>Pour configurer les Gestionnaire de réception POP3
La procédure suivante permet de modifier l'hôte associé au gestionnaire de réception POP3.  
  
### <a name="to-configure-the-general-properties-for-a-pop3-receive-handler"></a>Pour configurer les propriétés générales d'un gestionnaire de réception POP3  
  
1.  Dans la console Administration de BizTalk Server, développez **Administration de BizTalk Server**, développez **groupe BizTalk**, développez **paramètres de plateforme**, puis développez  **Adaptateurs**.  
  
2.  Dans la liste développée des adaptateurs, cliquez sur **POP3**, dans le volet de droite, avec le bouton du Gestionnaire de réception que vous souhaitez configurer, puis cliquez sur **propriétés**.  
  
3.  Dans le **propriétés du Gestionnaire d’adaptateur** boîte de dialogue le **général** sous l’onglet du **nom d’hôte** , sélectionnez l’hôte avec lequel associer le Gestionnaire de réception.  
  
4.  Cliquez sur **OK**.
