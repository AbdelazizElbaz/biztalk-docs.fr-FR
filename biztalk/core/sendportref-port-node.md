# <a name="sendportref-port-node"></a>SendPortRef (nœud Port)
Le nœud SendPortRef du nœud Port d'un fichier de liaison contient des informations sur un port d'envoi référencé par un service.  
  
## <a name="nodes-in-the-sendportref-node"></a>Nœuds du nœud SendPortRef  
 Le tableau suivant répertorie les propriétés que vous pouvez définir pour ce nœud d'un fichier de liaison :  
  
|**Nom**|**Type de nœud**|**Type de données**|**Description**|**Restrictions**|**Commentaires**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|Nom|Attribut|xs:string|Spécifie le nom d'un port d'envoi référencé par un service.|Facultatif|Valeur par défaut : vide|
