# <a name="optimizing-performance"></a>Optimisation des performances
Une installation par défaut du système d’exploitation Windows, SQL Server, [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)], et IIS peut être considérablement améliorée pour fournir des performances optimales pour la production [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] environnement. Paramètres de configuration WCF peuvent également être personnalisées à partir des paramètres par défaut pour fournir des performances significativement accrues. Cette section fournit des optimisations de performances spécifiques qui doivent être respectées lors du déploiement d’une production [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] solution.
