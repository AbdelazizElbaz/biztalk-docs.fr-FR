# <a name="how-to-view-management-pack-details"></a>Comment afficher les détails du Pack d’administration
Pour plus d’informations sur une analyse et les valeurs de remplacement associées, affichez la base de connaissances pour l’analyse.  
  
### <a name="to-view-knowledge-for-a-monitor"></a>Pour afficher les informations pour une analyse  
  
1.  Dans la Console opérateur, cliquez sur le **création** bouton.  
  
2.  Développez **objets du Pack d’administration**, puis cliquez sur **analyses**.  
  
3.  Dans le volet analyses, développez les cibles jusqu'à ce que vous atteigniez le niveau de l’analyseur. Vous pouvez également utiliser le **recherche** pour rechercher une analyse particulière.  
  
4.  Cliquez sur le moniteur, puis dans le volet analyses, **afficher la base de connaissances**.  
  
5.  Cliquez sur le **de connaissances** onglet.
