# <a name="performing-a-simple-update"></a>Effectuer une mise à jour Simple
Une simple mise à jour d’une application BizTalk ou d’artefact ne requiert aucun temps d’arrêt, ne nécessite aucune modification de code et peut être effectuée par un professionnel de l’informatique ou singlehandedly d’analyste d’entreprise. Ces types de modifications sont en général aussi simples que l’ajout d’un port d’envoi supplémentaires à un groupe de ports d’envoi, l’ajout d’un port d’envoi supplémentaires avec les filtres d’abonnement approprié, la modification de l’URI d’un port d’envoi ou mise à jour d’une règle dans le moteur BRE. Pour plus d’informations, consultez [création et à l’aide des règles d’entreprise](http://go.microsoft.com/fwlink/?LinkId=155136) (http://go.microsoft.com/fwlink/?LinkId=155136) dans [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] aide.
