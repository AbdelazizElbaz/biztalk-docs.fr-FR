# <a name="recovering-from-a-disaster"></a>Récupération après sinistre
Cette section détaille les étapes pour transférer les opérations de production vers le site de récupération d’urgence. Dans le cadre de cette section, le système de production en ligne est appelé le système « source ». Le site de récupération d’urgence est appelé le système « destination ». Cette section fournit une vue d’ensemble du processus de récupération d’urgence.
