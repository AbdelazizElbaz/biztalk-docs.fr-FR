# <a name="recovering-additional-applications"></a>La restauration des Applications supplémentaires
Il peut y avoir des applications supplémentaires qui doivent être restaurées dans le cadre de la solution d’application globale. Procédures de récupération des applications supplémentaires doivent être définis par les équipes d’application et inclus dans le plan de récupération d’urgence global.
