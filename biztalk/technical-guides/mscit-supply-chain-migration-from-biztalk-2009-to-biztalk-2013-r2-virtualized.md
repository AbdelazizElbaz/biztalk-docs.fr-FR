# <a name="mscit-supply-chain-migration-from-biztalk-server-2009-hub-to-biztalk-server-2013-r2-virtualized-hub"></a>MSCIT : Migration de la chaîne d’approvisionnement à partir du hub de BizTalk Server 2009 vers BizTalk Server 2013 R2 virtualisés hub
Migration d’application à partir de BizTalk Server 2010 vers BizTalk Server 2013 R2.  
  
 **Auteur**: Arvind Chaudhary, Microsoft &#124; Piyush Prasad, Microsoft  
  
 **Publié**: septembre 2015  
  
 **S’applique à**: BizTalk Server 2013 R2  
  
 **Résumé**: décrit les raisons et le processus de migration effectuée par Microsoft IT pour migrer des applications de BizTalk Server 2009 et 2010 vers BizTalk Server 2013 R2. MSIT identifié les flux de messages, les projets BizTalk dans Visual Studio, déplacé des contrats et des parties de gestion des partenaires commerciaux et mis à jour les liaisons d’application.  
  
 Vous pouvez également lire sur les défis MSIT et les meilleures pratiques utilisées.
