# <a name="how-to-schedule-a-backup-biztalk-server-job"></a>Comment planifier un travail de sauvegarde de BizTalk Server
La sauvegarde de BizTalk Server s’exécute comme prévu par le service de l’Agent SQL Server. Si vous souhaitez créer des sauvegardes plus fréquentes ou moins fréquentes, vous pouvez modifier la planification du travail de sauvegarde de BizTalk Server à l’aide de SQL Server Enterprise Manager. Pour plus d’informations sur la façon de planifier un travail de sauvegarde de BizTalk Server, consultez [comment planifier le travail de sauvegarde de BizTalk Server](http://go.microsoft.com/fwlink/?LinkId=154674) (http://go.microsoft.com/fwlink/?LinkId=154674) dans [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] aide.
