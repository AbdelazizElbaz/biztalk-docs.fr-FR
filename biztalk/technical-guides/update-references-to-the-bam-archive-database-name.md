# <a name="update-references-to-the-bam-archive-database-name"></a>Mettre à jour les références pour le nom de base de données d’archives BAM
Si vous avez sauvegardé vos bases de données des archives BAM, en cas de défaillance du système ou des données, vous pouvez restaurer cette sauvegarde vers un autre ordinateur, et vous pouvez renommer la sauvegarde.  
  
 Pour restaurer les bases de données des archives BAM, effectuez les étapes de [la restauration de bases de données dans le travail de sauvegarde de BizTalk Server](../technical-guides/how-to-restore-databases-in-the-backup-biztalk-server-job.md). En outre, vous devez mettre à jour les packages BAM SSIS avec le nouveau nom du serveur et le nom de la base de données.  
  
## <a name="how-to-update-references-to-bam-archive-database"></a>Comment mettre à jour les références à la base de données des archives BAM  
 Pour obtenir des instructions sur la façon de mettre à jour les références à la base de données des archives BAM, consultez [mise à jour des références à la base de données des archives BAM nouvelle](../technical-guides/how-to-move-the-bam-archive-database1.md#BKMK_UpdateArch).
