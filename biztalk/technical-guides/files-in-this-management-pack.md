# <a name="files-in-this-management-pack"></a>Fichiers dans ce pack d’administration
Pour surveiller [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)], vous devez d’abord télécharger le [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] Pack d’administration depuis [catalogue des packs d’administration System Center](http://www.microsoft.com/download/details.aspx?id=39617) (http://www.microsoft.com/download/details.aspx?id=39617). Le [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] Pack d’administration inclut les fichiers suivants :  
  
|Nom de fichier|Nom complet| Description|  
|---------------|------------------|-----------------|  
|Microsoft.BizTalkServer2013.Library.mp|Bibliothèque de BizTalk Server|Contient des classes génériques, les relations et les autres blocs de construction de pack de gestion qui sont utilisés par les autres packs d’administration pour fournir une analyse pour toutes les applications BizTalk.|  
|Microsoft.BizTalk.Server.2013.Monitoring.mp|Surveillance de BizTalk Server|Contient des moniteurs, des règles et des vues qui fournissent un moyen extensible pour contrôler toutes les applications BizTalk.|  
|Microsoft.BizTalk.Server.2013.Discovery.mp|Détection de BizTalk Server|Contient des détections qui sont utilisées pour rechercher les différents composants d’applications BizTalk.|
