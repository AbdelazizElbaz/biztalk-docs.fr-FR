# <a name="discoveries"></a>Découvertes
Le [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] Management Pack fournit une gestion centralisée et l’expérience de surveillance pour les artefacts liés à BizTalk Server.  
  
## <a name="discovery-of-artifacts"></a>Découverte d’artefacts  
 Avec [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] Pack d’administration, les agents de surveillance découvrent les artefacts à partir d’un ordinateur d’exécution unique. Par défaut, le Pack d’administration découvre les artefacts sur le premier ordinateur est joint au groupe BizTalk. Toutefois si vous souhaitez planifier découvrir les artefacts sur l’ordinateur de votre choix. vous devez utiliser la fonctionnalité de remplacement pour modifier les paramètres de détection automatique. Pour utiliser un remplacement pour modifier le paramètre de détection automatique, consultez [Configurations facultatives](../technical-guides/optional-configurations.md).  
  
 Les actions et les tâches qui sont effectuées à partir de la Console opérateur de SCOM sont uniques pour les artefacts qui sont détectées et analysées sur tous les ordinateurs de l’exécution. Pour afficher toutes les tâches du Pack d’administration de BizTalk Server fournit, cliquez sur **tâches** dans le volet analyse de la Console opérateur d’Operations Manager 2007 R2/2012.
