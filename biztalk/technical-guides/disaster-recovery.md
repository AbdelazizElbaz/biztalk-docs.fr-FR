# <a name="disaster-recovery"></a>Récupération d'urgence
Les procédures de récupération d’urgence améliorent la disponibilité du système en utilisant les étapes pour reprendre le fonctionnement d’un système ayant échoué. Récupération d’urgence est différente de la tolérance de panne dans la récupération d’urgence requiert généralement une intervention manuelle pour restaurer le système a échoué alors qu’une tolérance aux pannes peut continuer à fonctionner sans intervention manuelle si le système rencontre un échec condition.
