# <a name="installing-and-running-the-multiple-web-services-sample"></a>Installer et exécuter l’exemple de Services Web multiples
L’exemple de plusieurs Services Web montre comment vous pouvez développer un itinéraire qui peut envoyer un message à plusieurs services Web, un après l’autre, en utilisant le composant de pipeline du redirecteur. Cet exemple requiert que vous avez déjà installé l’exemple de feuille de route rampe d’entrée.
