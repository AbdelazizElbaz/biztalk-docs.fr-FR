# <a name="amberpoint-biztalk-nano-agent"></a>Agent de Nano AmberPoint BizTalk
Système de gestion AmberPoint SOA (SMS) propose une suite complète de visualisation et des outils d’analyse pour relever les défis de la gouvernance de l’exécution. Le [!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)] et Microsoft BizTalk Server facilitent les services et applications d’entreprise SOA intégrés, mais analyse des orchestrations et des applications composites qui en résulte est plus complexe. Intégration de BizTalk Server de AmberPoint via leurs BizTalk Nano Agent étend existant de l’exécution des fonctions de gouvernance pour inclure BizTalk Server emplacements de réception et ports, quel que soit leur liaison sous-jacente de l’adaptateur d’envoi.
