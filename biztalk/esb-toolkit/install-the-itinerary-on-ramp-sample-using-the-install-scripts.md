# <a name="install-the-itinerary-on-ramp-sample-using-the-install-scripts"></a>Installer l’exemple rampe d’entrée d’itinéraire à l’aide de Scripts d’installation
Cette section décrit comment vous pouvez installer l’exemple de feuille de route rampe d’entrée à partir de l’installation de scripts fournis avec le [!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)].  
  
### <a name="to-install-the-itinerary-on-ramp-sample-from-the-install-scripts"></a>Pour installer l’exemple de feuille de route rampe d’entrée à partir de scripts d’installation  
  
1.  Dans le menu **Démarrer** , cliquez sur **Exécuter**.  
  
2.  Dans le **exécuter** boîte de dialogue, tapez **cmd**, puis appuyez sur ENTRÉE pour ouvrir une invite de commandes.  
  
3.  Exécutez la commande suivante, en remplaçant le \<chemin d’accès > paramètre avec le chemin d’accès complet au fichier .cmd à installer (le chemin d’accès par défaut dans cette version est \Source\Samples\Itinerary\Install\Scripts\\) :  
  
    ```  
    <path>\Setup_bin.cmd  
    ```
