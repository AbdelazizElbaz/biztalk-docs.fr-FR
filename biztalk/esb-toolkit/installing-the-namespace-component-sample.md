# <a name="installing-the-namespace-component-sample"></a>Installation de l’exemple de composant de Namespace
L’exemple de composant de Namespace requiert le composant Namespace (dans l’assembly Microsoft.Practices.ESB.Namespace.PipelineComponents.dll) réside dans le dossier PipelineComponents de votre dossier d’installation de Microsoft BizTalk Server. L’installation de le [!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)] core copie automatiquement l’assembly à l’emplacement correct.  
  
 Cette section décrit le processus d’installation de l’exemple de composant de Namespace dans l’application BizTalk de GlobalBank.ESB. Vous pouvez installer l’exemple de composant de Namespace à partir du projet de la solution ou utilisez le fichier Windows Installer inclus avec le [!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)]. Cette section contient les rubriques suivantes :  
  
-   [Installer l’exemple de composant de Namespace à l’aide de Scripts d’installation](../esb-toolkit/install-the-namespace-component-sample-using-the-install-scripts.md)  
  
-   [Assemblys et artefacts installés par l’exemple de composant de Namespace](../esb-toolkit/assemblies-and-artifacts-installed-by-the-namespace-component-sample.md)
