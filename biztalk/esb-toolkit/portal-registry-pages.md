# <a name="portal-registry-pages"></a>Pages du portail de Registre
Lorsque vous déplacez la souris sur le **Registre** onglet, un menu contextuel fournit une liste des tâches de portail de Registre suivantes :  
  
-   [Nouvelle Page d’entrée de Registre](../esb-toolkit/new-registry-entry-page.md). Cet élément de menu ouvre une page où vous pouvez afficher une liste des points de terminaison existants dans une application Microsoft BizTalk Server ESB spécifiée. Un lien dans la ligne pour chaque point de terminaison disponible s’ouvre le [Page Détails du Registre](../esb-toolkit/registry-details-page.md) où vous pouvez modifier les détails du Registre ou créer une nouvelle publication de Universal Description, Discovery and Integration (UDDI) demande pour l’actuellement serveur UDDI.  
  
-   [Gérer les demandes en attente](../esb-toolkit/manage-pending-requests-page.md). Cet élément de menu ouvre une page qui répertorie toutes les demandes d’inscription de Registre UDDI en attente et vous permet de gérer ces demandes. Vous pouvez également afficher un historique des requêtes précédentes et leur état.
