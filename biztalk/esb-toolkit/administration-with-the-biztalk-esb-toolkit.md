# <a name="administration-with-the-biztalk-esb-toolkit"></a>Administration de BizTalk ESB Toolkit
Cette section présente une zone importante d’activité d’administration liée aux applications basées sur les [!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)] et Microsoft BizTalk Server ; plus précisément, il traite le portail de gestion ESB. Le portail de gestion ESB est un utilitaire basé sur le Web pour l’analyse des applications BizTalk Server génère des alertes lorsque des exceptions se produisent et gère certaines fonctionnalités de BizTalk ESB Toolkit, telles que l’inscription de Universal Description, Discovery, et Entrées de l’intégration (UDDI).
