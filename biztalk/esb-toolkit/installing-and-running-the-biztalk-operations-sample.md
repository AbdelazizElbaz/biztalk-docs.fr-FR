# <a name="installing-and-running-the-biztalk-operations-sample"></a>Installer et exécuter l’exemple d’opérations BizTalk
Opérations de Microsoft BizTalk montre comment vous pouvez rechercher les données opérationnelles sur les applications BizTalk à l’aide du Service d’opérations de BizTalk ESB. Le portail de gestion ESB utilise ce service pour retourner ou mettre à jour les informations de gestion et le suivi des bases de données BizTalk. Les informations disponibles sont les suivantes :  
  
-   L’état des applications BizTalk déployées et des hôtes  
  
-   Instances de service actifs ou suspendus  
  
-   Messages et leurs propriétés  
  
-   Applications BizTalk déployées  
  
-   Propriétés du port de réception et emplacements de réception  
  
-   Propriétés du port d’envoi
