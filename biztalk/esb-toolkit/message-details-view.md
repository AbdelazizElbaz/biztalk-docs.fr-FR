# <a name="message-details-view"></a>Vue Détails sur le message
Figure 1 montre la vue Détails du Message, qui affiche les propriétés ambiantes du message que vous avez sélectionné dans la vue Détails de l’erreur, le contenu du message et les valeurs des propriétés de contexte du message persistante. Cliquez sur le **télécharger** icône au-dessus de données de message pour le télécharger sous forme de document XML. Cliquez sur le **modifier** icône au-dessous des données de message à modifier et renvoyez le message.
