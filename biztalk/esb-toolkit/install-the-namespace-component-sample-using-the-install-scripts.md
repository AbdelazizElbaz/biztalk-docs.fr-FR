# <a name="install-the-namespace-component-sample-using-the-install-scripts"></a>Installer l’exemple de composant de Namespace à l’aide de Scripts d’installation
Cette section décrit comment vous pouvez installer l’exemple de composant de Namespace à partir du fichier Windows Installer avec le [!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)].  
  
 **Pour installer l’exemple de composant de Namespace dans les scripts d’installation**  
  
1.  Dans le menu **Démarrer** , cliquez sur **Exécuter**.  
  
2.  Dans le **exécuter** boîte de dialogue, tapez **cmd**, puis appuyez sur ENTRÉE pour ouvrir une invite de commandes.  
  
3.  Exécutez la commande suivante, en remplaçant le  *\<chemin d’accès >* paramètre avec le chemin d’accès complet au fichier .cmd à installer (le chemin d’accès par défaut dans cette version est \Source\Samples\Namepace\Install\Scripts\\) :  
  
    ```  
    <path>\Setup_bin.cmd  
    ```
