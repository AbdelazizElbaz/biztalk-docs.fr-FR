# <a name="installing-the-multiple-web-services-sample"></a>Installation de l’exemple de Services Web multiples
Cette section décrit le processus d’installation de l’exemple de plusieurs Services Web. Vous devez installer les exemples de [l’installation et l’exécution de l’exemple de rampe d’entrée d’itinéraire](../esb-toolkit/installing-and-running-the-itinerary-on-ramp-sample.md) et [l’installation et l’exécution de l’exemple de résolution dynamique](../esb-toolkit/installing-and-running-the-dynamic-resolution-sample.md) avant d’installer et utiliser cet exemple.  
  
 **Pour installer l’exemple de plusieurs Services Web**  
  
1.  Dans l’Explorateur Windows, ouvrez le dossier \Source\Samples\MultipleWebServices\Install\Scripts.  
  
2.  Double-cliquez sur le fichier de script nommé Setup_bin.cmd.
