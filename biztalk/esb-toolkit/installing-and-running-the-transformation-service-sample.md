# <a name="installing-and-running-the-transformation-service-sample"></a>Installer et exécuter l’exemple de Service de Transformation
Le service Web de Transformation expose directement les mappages déployés dans Microsoft BizTalk via un service Web, éliminant ainsi une consultation par le message de la base de données MessageBox de BizTalk. Le service Web de la Transformation permet à des applications externes envoyer un document à l’application Microsoft.Practices.ESB et transformé à l’aide d’une carte déployée au sein de BizTalk.
