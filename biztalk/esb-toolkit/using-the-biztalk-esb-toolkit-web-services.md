# <a name="using-the-biztalk-esb-toolkit-web-services"></a>L’utilisation des Services BizTalk ESB Toolkit Web
Le [!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)] expose sa principale de toutes les fonctionnalités en tant que services Web ; ainsi, les entités en dehors de l’architecture ESB, peut-être en cours d’exécution sur les plateformes non Microsoft, pour participer aux échanges d’ESB et tirer parti des fonctionnalités ESB. Tous les services Web sont situés dans des sous-dossiers du dossier du répertoire d’installation ESB \Web distincts. La [!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)] inclut les services Web d’ESB suivants :  
  
-   [Les Services Web de rampe d’entrée d’itinéraire](../esb-toolkit/the-itinerary-on-ramp-web-services.md)  
  
-   [Le Service Web de résolution](../esb-toolkit/the-resolver-web-service.md)  
  
-   [Le Service Web de Transformation](../esb-toolkit/the-transformation-web-service.md)  
  
-   [Le Service Web de gestion des exceptions](../esb-toolkit/the-exception-handling-web-service.md)  
  
-   [Le Service Web d’opérations BizTalk](../esb-toolkit/the-biztalk-operations-web-service.md)
