# <a name="audit-log-page"></a>Page du journal d’audit
Figure 1 montre la page journal d’Audit. Cette page affiche une liste de tous les événements audités.  
  
 ![La Page auditlog](../esb-toolkit/media/ch8-auditlogpagelargeview.gif "Ch8-AuditLogPageLargeView")  
  
 **Figure 1**  
  
 **La page journal d’Audit du portail de gestion ESB**  
  
 La liste suivante explique comment vous pouvez utiliser les fonctionnalités de la page journal d’Audit de ESB gestion portail :  
  
-   Cliquez sur le lien d’identificateur de message pour afficher le corps du message soumise à nouveau. Cette opération ouvre le journal d’Audit - page de l’Afficheur des messages (voir Figure 2), qui affiche l’utilisateur, une date et une URL pour le message soumise à nouveau et le contenu du message soumise à nouveau.
