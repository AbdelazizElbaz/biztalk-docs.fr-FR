# <a name="manage-pending-requests-page"></a>Page Gérer les requêtes en attente
Figure 1 montre la page Gérer les demandes en attente, qui affiche une liste d’enregistrement en attente les demandes et fournit un lien pour afficher l’historique des demandes.  
  
 ![Gérer les demandes en attente](../esb-toolkit/media/ch8-managependingrequestspage.gif "Ch8-ManagePendingRequestsPage")  
  
 **Figure 1**  
  
 **La page ESB Management Portal gérer les demandes en attente**  
  
 La liste suivante explique comment vous pouvez utiliser les fonctionnalités de la page ESB Management Portal gérer les demandes en attente :  
  
-   Cliquez sur le **ViewDetails** icône (la Loupe) pour ouvrir la [Page Détails du Registre](../esb-toolkit/registry-details-page.md), où vous pouvez afficher les détails de la demande en attente, publier, mettre à jour ou le supprimer.  
  
-   Cliquez sur le **approuver** icône (coche) pour approuver la demande en attente.  
  
-   Cliquez sur le **rejeter** icône (croix) à rejeter la demande en attente.  
  
-   Cliquez sur le **afficher l’historique de demande** lien pour ouvrir une page qui affiche des informations pour les demandes d’inscription Universal Description, Discovery and Integration (UDDI) précédentes.
