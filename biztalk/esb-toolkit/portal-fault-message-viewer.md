# <a name="portal-fault-message-viewer"></a>Afficheur des messages d’erreur de portail
L’Afficheur des messages d’erreur ESB affiche tous les messages générés par le composant de Pipeline encodeur de Exception personnalisé dans le portail de gestion ESB. Les vues suivantes sont disponibles :  
  
-   [Affichage des détails de l’erreur](../esb-toolkit/fault-details-view.md)  
  
-   [Affichage des détails de message](../esb-toolkit/message-details-view.md)
