# <a name="implementing-design-patterns-in-itineraries"></a>Implémenter des modèles de conception dans les itinéraires
Cette rubrique décrit les modèles d’architecture orientée services (SOA) courants et les catégories qui peuvent être implémentées à l’aide de la [!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)]. Chaque catégorie de modèle comprend une description de chaque modèle associée et des liens vers des informations plus détaillées. La description de chaque modèle inclut des références aux éléments connexes [!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)] exemples d’applications qui implémentent des modèles et les liens vers les activités de développement conçues pour montrer les développeurs de créer des solutions d’ESB pour implémenter ces modèles. Cette section comprend les sections suivantes de la rubrique :  
  
-   [Modèles de routage de messages](../esb-toolkit/message-routing-patterns.md)  
  
-   [Modèles de Transformation de messages](../esb-toolkit/message-transformation-patterns.md)  
  
-   [Modèles de médiation de service](../esb-toolkit/service-mediation-patterns.md)  
  
-   [Modèles de service de gestion](../esb-toolkit/service-management-patterns.md)
