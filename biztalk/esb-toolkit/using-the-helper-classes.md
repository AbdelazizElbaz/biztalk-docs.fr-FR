# <a name="using-the-helper-classes"></a>L’utilisation des Classes d’assistance
[!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)]comprend un ensemble de classes d’assistance utilisées par les autres processus, services et composants dans la boîte à outils. Étant donné que ces classes d’assistance sont également publics, vous pouvez les utiliser dans votre propre code en fonction des besoins.  
  
 Dans certains cas, vous devez les utiliser pour fournir des paramètres du type approprié pour les méthodes d’autres classes dans l’architecture ESB. Par exemple, les méthodes de service Web du programme de résolution retournent une collection générique d’instances de la **résolveur** classe.  
  
 Les sections suivantes décrivent l’assistance de classes sont incluses dans [!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)]:  
  
-   [La classe MapHelper](../esb-toolkit/the-maphelper-class.md)  
  
-   [La classe de programme de résolution](../esb-toolkit/the-resolver-class.md)  
  
-   [La classe ResolverDictionary](../esb-toolkit/the-resolverdictionary-class.md)  
  
-   [La classe de gestionnaire de programme de résolution (ResolverMgr)](../esb-toolkit/the-resolver-manager-resolvermgr-class.md)
